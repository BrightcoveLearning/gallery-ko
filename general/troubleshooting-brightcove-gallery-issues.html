    브라이트코브 갤러리 문제 해결

    이 항목에서는 갤러리 환경을 만들 때 고객이 겪는 일반적인 문제를 해결하는 방법에 대해 알아봅니다.

    SSL을 사용하도록 사이트를 구성할 때의 문제

    갤러리 포털 환경을 구성하기 위한 요구 사항은SSL을 사용하여 포털 환경 보안문서에 설명되어 있습니다. 일반적인 문제는 여기에 나열되어 있습니다.

    키가 잘못 복사되었습니다.

    • 값 주위에 공백이 없는지 확인하십시오.
    • 전체 키가 복사되었는지 확인하십시오. '—BEGIN 인증서—'및 '—최종 인증서'가 포함되어 있는지 확인합니다.
    • 공개 키는 각각 '- 인증서 시작 -'와 '- 최종 인증서-'로 시작하고 끝나야합니다. 개인 키는 '—RSA 개인 키 시작—'및 '—RSA 개인 키 끝—'로 시작하고 끝나야 합니다.
    • 인증서 체인은 여러 개의 연결된 키를 가질 수 있습니다.

    키가 PEM 형식이 아닙니다.

    인증서 발급자에게 문의하여 키가 PEM 형식인지 확인합니다.

    키가 자체 서명되었습니다.

    인증서 발급자에게 확인하십시오.

    개인 키가 암호로 보호됨

    암호로 보호된 개인 키는 지원하지 않습니다. 암호로 보호되지 않은 키는 다음과 같습니다.

    잘못된 줄 끝

    키가 Linux와 Windows 시스템간에 전송 된 경우 줄 끝이 손상되었을 가능성이 매우 낮습니다. 자세한 내용은이 링크를참조하십시오.

    공개 키와 개인 키가 일치하지 않습니다.

    드문 경우입니다. 키가 일치하는지 확인하는 한 가지방법은이 링크에설명되어 있습니다.

    브라이트코브 지원에 문의

    SSL을 사용하도록 갤러리 사이트를 구성하는 데 여전히 문제가 있는 경우브라이트코브 지원에 문의하십시오 . 계정 ID, 인증서 발급자 및 인증서 업로드를 시도한 기간을 입력하십시오.

    인터넷 익스플로러 9와 10에서 페이지가 빈 상태로 표시됩니다.

    Internet Explorer에서 호환성 보기 모드를 사용하여 사이트를 볼 수 없습니다. 기본적으로 갤러리 페이지는 “비호환성” 모드로 로드되며 IE 9 및 10을 사용하여 올바르게 표시되어야 합니다. 자세한 내용은브라이트코브 갤러리 시스템 요구 사항을참조하십시오.

    내 사이트를 변경했지만 표시되지 않습니다.

    갤러리 환경을 변경한 후에는 변경 내용이 나타나기 전에 다시 게시해야 합니다. 다시 게시해야 하는 환경은 환경 세부 정보의 일부로변경 내용 게시링크를 표시합니다. 자세한 내용은사이트에 변경 사항 게시를참조하십시오.

    참고: 변경 사항이 나타나지 않으면 브라우저 캐시와 쿠키를 지워야 할 수도 있습니다.

    사이트에 대한 소셜 공유 옵션을 활성화했지만 표시되지 않습니다.

    기본적으로 소셜 공유 아이콘은 밝은 색 구성표를 사용하며 사이트에서 밝은 색 구성표를 사용하는 경우 보기가 불가능하거나 어려울 수 있습니다. 아이콘 색 구성표를 변경하는 방법에 대한 자세한 내용은포털 환경에 대한 소셜 설정 사용자 지정을참조하십시오.

    내 도메인에서 호스팅하는 옵션을 선택했지만 내 사이트가 표시되지 않습니다.

    자신의 도메인에서 사이트를 호스팅하려면 갤러리 설정의 일부로 도메인을 추가하고 도메인을 사이트 속성의 일부로 할당하고 호스팅 공급자와 CNAME 레코드를 추가해야 합니다. 자세한 내용은포털 환경에 사용자 지정 도메인 할당을참조하십시오.

    Video Cloud에서 동영상 제목 및/또는 설명을 업데이트했지만 내 사이트에 업데이트가 표시되지 않습니다.

    갤러리는 비디오 클라우드를CMS API사용하여 비디오 정보를 가져옵니다. CMS API호출은 성능을 향상시키기 위해 캐시됩니다. 비디오 메타데이터에 대한 변경 사항이 갤러리 모듈에 반영되기까지 5-15분 정도 걸릴 수 있습니다.

    비디오에 태그를 지정하거나 사용자 정의 필드, 재생 목록 등을 업데이트했는데 갤러리에서 컬렉션을 만들 때 표시되지 않습니다.

    갤러리는 비디오 클라우드를CMS API사용하여 비디오 정보를 가져옵니다. CMS API호출은 성능을 향상시키기 위해 캐시됩니다. 비디오 메타데이터에 대한 변경 사항이 갤러리 모듈에 반영되기까지 5-15분 정도 걸릴 수 있습니다.

    모음에 비디오를 추가할 때 잘못된 비디오가 나타나거나 사이트에 비디오가 추가되지 않습니다.

    Video Cloud 계정이 HTTP 전송을 사용하도록 설정되어 있는지 확인합니다. HTTP 전송이 활성화되지 않은 경우 컬렉션에 비디오를 추가하려고 할 때 이상한 문제가 발생했습니다. 또한 HTTP 전송 설정은 비디오 다운로드 기능과 사이트 맵을 생성하는 데 필요합니다.
