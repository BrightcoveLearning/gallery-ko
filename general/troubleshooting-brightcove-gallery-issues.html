SSL을 사용하도록 사이트를 구성 할 때 문제

갤러리 포털 환경을 구성하기 위한 요구 사항은 SSL을 사용하여 포털 환경보안문서에 요약되어 있습니다. 일반적인 문제가 여기에 나열되어 있습니다.

키가 잘못 복사되었습니다.

  • 값 주위에 공백이 없는지 확인하십시오.
  • 전체 키가 복사되었는지 확인하십시오. ' ----- BEGIN CERTIFICATE --- --'및 ' ----- END CERTIFICATE --- --'가 포함되어 있는지 확인합니다.
  • 공개 키는 각각 ' ----- BEGIN CERTIFICATE --- --'및 ' ----- END CERTIFICATE --- --'로 시작하고 끝나야합니다. 개인 키는 ' ----- BEGIN RSA PRIVATE KEY --- --'및 ' ----- END RSA PRIVATE KEY --- --'로 시작하고 끝나야합니다.
  • 인증서 체인에는 여러 개의 연결된 키가있을 수 있습니다.

키가 PEM 형식이 아닙니다.

인증서 발급자에게 문의하여 키가 PEM 형식인지 확인하십시오.

키는 자체 서명되었습니다.

인증서 발급자에게 확인하십시오.

개인 키는 암호로 보호됩니다.

잘못된 줄 끝

키가 Linux와 Windows 시스템간에 전송 된 경우 줄 끝이 손상되었을 가능성이 매우 낮습니다. 자세한 내용은이 링크를참조하십시오.

공개 키와 개인 키가 일치하지 않습니다.

이것은 드문 경우입니다. 키가 일치하는지 확인하는 한 가지 방법은이 링크에 설명되어있습니다.

Brightcove 지원에 문의

SSL을 사용하도록 갤러리 사이트를 구성하는 데 문제가 계속 발생하면 브라이트코브 지원에문의하십시오 . 계정 ID, 인증서 발급자 및 인증서 업로드를 시도한 기간을 제공하십시오.

Internet Explorer 9 및 10에서 페이지가 공백으로 표시됩니다.

Internet Explorer에서 호환성보기 모드를 사용하여 사이트를 볼 수 없습니다. 기본적으로 갤러리 페이지는 '비 호환성'모드로로드되며 IE 9 및 10을 사용하여 올바르게 표시되어야합니다. 자세한 내용은브라이트코브 갤러리 시스템 요구 사항을참조하십시오.

내 사이트를 변경했지만 표시되지 않습니다.

갤러리 환경을 변경 한 후 변경 사항이 표시되기 전에 다시 게시해야합니다. 다시 게시해야 하는 경험에는 경험 세부 정보의 일부로 변경 내용게시링크가 표시됩니다. 자세한 내용은 사이트에 변경 사항게시를 참조하십시오 .

참고: 변경 사항이 나타나지 않는 경우 브라우저 캐시와 쿠키를 지워야 할 수도 있습니다.

사이트의 소셜 공유 옵션을 활성화했지만 표시되지 않습니다.

기본적으로 소셜 공유 아이콘은 밝은 색 구성표를 사용하며 사이트에서 밝은 색 구성표를 사용하는 경우 표시가 불가능하거나 어려울 수 있습니다. 아이콘 색 구성표를 변경하는 방법에 대한 자세한 내용은 포털 환경에 대한 소셜 설정사용자 지정을 참조하십시오 .

내 도메인에서 호스팅하는 옵션을 선택했지만 내 사이트가 표시되지 않습니다.

자체 도메인에서 사이트를 호스팅하려면 갤러리 설정의 일부로 도메인을 추가하고 사이트 속성의 일부로 도메인을 할당하고 호스팅 제공 업체에 CNAME 레코드를 추가해야합니다. 자세한 내용은 포털 경험에 사용자 지정 도메인할당을 참조하십시오 .

Video Cloud에서 비디오 제목 및 / 또는 설명을 업데이트했지만 내 사이트에 업데이트가 표시되지 않습니다.

갤러리는 비디오 클라우드 CMS API를사용하여 비디오 정보를 가져옵니다. CMS API호출은 성능을 향상시키기 위해 캐시됩니다. 동영상 메타 데이터의 변경 사항이 갤러리 모듈에 반영되기까지 5-15 분 정도 걸릴 수 있습니다.

비디오 (또는 업데이트 된 사용자 정의 필드, 재생 목록 등)에 태그를 지정했는데 갤러리에서 컬렉션을 만들 때 표시되지 않습니다.

갤러리는 비디오 클라우드 CMS API를사용하여 비디오 정보를 가져옵니다. CMS API호출은 성능을 향상시키기 위해 캐시됩니다. 동영상 메타 데이터의 변경 사항이 갤러리 모듈에 반영되기까지 5-15 분 정도 걸릴 수 있습니다.

컬렉션에 비디오를 추가 할 때 잘못된 비디오가 나타나거나 사이트에 추가 된 비디오가 없습니다.

HTTP 전달을 위해 Video Cloud 계정이 활성화되어 있는지 확인하십시오. HTTP 전송이 활성화되지 않은 경우 컬렉션에 비디오를 추가하려고 할 때 이상한 문제가 발생했습니다. HTTP 전송 설정은 비디오 다운로드 기능과 사이트 맵 생성에도 필요합니다.
