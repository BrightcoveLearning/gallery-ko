갤러리 경험 생성 및 관리

{{ page.description }}

갤러리 환경을 만들고 게시하는 것을 빠르게 시작하려면 다음 문서 중 하나를 확인하십시오.

  • 단계별: 포털 경험 생성 및 게시
  • 단계별: 인 페이지 경험 생성 및 게시

갤러리 모듈을 열면경험페이지에 생성된 모든 경험이 나열됩니다.

경험이 생성되지 않은 경우 경험을생성하라는 메시지가 표시됩니다 . 경험페이지는 다음 기능을 제공합니다.

  • 경험 삭제
  • 오프라인으로 경험하기
  • 중복 된 경험
  • 경험 편집
  • 경험 만들기
  • 경험 필터링
  • 검색 경험

데이터를 정렬하려면 열 머리글을 클릭하십시오. 열 선택기()를 사용하여 페이지에 표시할 열을 선택할 수 있습니다.

다음 열이 표시 될 수 있습니다.

  • 표제 -체험 제목
  • 템플릿 - 경험에 사용되는 템플릿입니다. 경험 유형은 다음과 같습니다.
    • 문 -선택적인 클릭 유도 문안 및 광고 지원을 통해 모든 크기의 콘텐츠 라이브러리를 지원하는 사이트
    • 인 페이지 -하나 이상의 비디오를 게시하고 상호 작용을 추가 한 다음이 레이아웃을 사이트에 추가합니다.
    • 방문 페이지 -리드 양식과 선택적인 클릭 유도 문안 및 광고 지원이 포함 된 단일 페이지 사이트
    • 이벤트 - 라이브 이벤트 홍보 및 방송에 최적화된 사이트
  • 템플릿 상태 -템플릿 상태
    • 최신 정보 -경험이 최신 버전의 프로젝트 템플릿을 사용하고 있습니다.
    • 업데이트 필요 -경험에 대한 템플릿 업데이트를 사용할 수 있습니다.
    • 업데이트 재개 -템플릿 업데이트가 진행 중이지만 게시되지 않았습니다.
  • 업데이트 됨 -마지막 경험 업데이트 날짜 및 시간
  • 체험 현황 -체험 현황
    • 온라인 -경험은 온라인이며 액세스 할 수 있습니다.
    • 오프라인 -경험이 오프라인 상태이며 액세스 할 수 없습니다.
    • 게시되지 않은 변경 사항 -경험이 온라인이고 액세스 할 수 있지만 경험이 변경되었으며 게시되지 않았습니다.

행을 클릭하면 페이지 오른쪽에 경험 세부 정보가 표시됩니다.

경험 만들기

새 경험을 만들려면 [새 경험] 을 클릭합니다 . 새로운 경험을 만들 때 템플릿을 선택하여 시작합니다. 템플릿은 전체적인 모양과 동작을 제어합니다. 템플릿을 선택한 후 경험을 사용자 지정할 수 있습니다. 자세한 내용은 다음 항목 중 하나를 참조하십시오.

  • 개요: 포털 경험 생성, 편집 및 게시
  • 개요: 인 페이지 환경 생성, 편집 및 게시

경험 복제

경험 복제는 여러 언어로 경험을 만들어야하거나 기존 경험의 모양과 느낌에서 마이크로 사이트를 만드는 경우에 유용 할 수 있습니다. 경험을 복제하려면 행을 클릭한 다음복제를 클릭합니다 .

입력경험 제목그리고설명클릭구하다 .

편집 경험

경험을 편집하려면 경험제목을 클릭합니다 . 경험을 편집하면 갤러리 사이트 편집기가 열립니다. 편집 경험에 대한 정보는 다음 주제 중 하나를 참조하십시오.

  • 사이트 편집기를 사용하여 포털 경험 사용자 지정
  • 인 페이지 경험 사용자 지정

오프라인으로 경험하기

경험을 오프라인으로 전환하면 더 이상 액세스 할 수 없습니다. 경험을 오프라인으로 전환하려면 행을 클릭한 다음오프라인으로 전환을 클릭합니다 . 경험을 오프라인으로 전환해야 하는지 확인하려면 오프라인으로 전환을 클릭합니다 . 환경을 다시 사용할 수 있도록하려면 환경을 다시 게시해야합니다.

경험 삭제

경험을 삭제하려면 행을 클릭 한 다음 삭제를 클릭하십시오. 삭제를 클릭하여삭제를 확인합니다 .

참고: 경험이 사용자 지정 도메인을 사용하도록 구성된 경우 경험에 할당된 사용자 지정 도메인을 제거한 다음 경험을 삭제하기 전에 경험을 다시 게시해야 합니다.

경험 필터링

경험 페이지에 필터를 적용하여 표시되는 경험 유형을 제한 할 수 있습니다. 필터를 적용하려면 표시 할 경험 유형을 선택하십시오.

  • 모두 -모든 경험 표시
  • 게시 됨 -모든 게시 된 (온라인) 경험 표시
  • 널리 알려지지 않은 -오프라인으로 전환 된 모든 경험 표시
  • 템플릿 업데이트 필요 -사용 가능한 템플릿 업데이트가있는 모든 경험 표시
  • 방문 페이지 -모든 랜딩 페이지 경험 표시
  • 행사 -모든 이벤트 경험 표시
  • 문 -디스플레이 포털 경험
  • 인 페이지 -모든 인 페이지 경험 표시

경험 검색

검색상자는 경험 목록을 검색하는 데 사용할 수 있습니다. 검색 할 때 다음 속성이 검색됩니다.

  • 표제
  • 기술
  • 신분증
  • 주형
  • URL
