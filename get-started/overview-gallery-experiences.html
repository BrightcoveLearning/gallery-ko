갤러리 경험 개요

{{ page.description }}

Brightcove Gallery를 사용하면 사용자 지정 개발 없이도 강력하고 매력적인 비디오 경험을 쉽게 만들 수 있습니다. 갤러리를 사용하면 다음 유형의 경험을 만들 수 있습니다.

  • 문
  • 인 페이지
  • 방문 페이지
  • 행사
참고: Gallery 환경 만들기 및 게시를 빠르게 시작하려면포털또는인페이지환경을 만드는 단계별 주제 중 하나를 확인하십시오.

만들 경험 유형을 선택하면 Gallery는포털 (랜딩 페이지 및 이벤트 환경 포함) 과 다음을 제어하는페이지 내경험에 대한 템플릿 세트를 제공합니다.경험의 전체적인 모양과 느낌.

문

포털 경험은 브라이트코브가 호스팅하는 지정된 URL에서 액세스하는 독립 실행형 비디오 사이트 (예: 비디오 포털) 입니다. 다음 예에서는 Video Cloud 및 Gallery 교육 비디오가 포털 경험으로 구성되었습니다. 왼쪽에서 카테고리를 선택하면 페이지 본문에 해당 카테고리의 관련 비디오가 표시됩니다.

인페이지

인 페이지 경험은 기존 웹 페이지 내부에 포함됩니다. 인 페이지 경험을 사용하면 비디오 레이아웃을 만들고 기존 웹 사이트 내에 표시 할 수 있습니다. 인 페이지 환경은 동영상이 재생 될 때 표시 될 수있는 링크 및 카드와 같은 플레이어 내부 및 외부 상호 작용을 모두 지원합니다. 다음 예에서는 인 페이지 경험 (비디오 축소판)이 기존 웹 페이지 내부에 포함되었습니다. 동영상 미리보기 이미지 중 하나를 클릭하면 라이트 박스에서 동영상 플레이어가 열립니다.

방문 페이지

랜딩 페이지 경험은 Brightcove에서 호스팅하는 단일 비디오 페이지이며 일반적으로 이메일 캠페인의 일부로 사용됩니다. 사용자가 이메일 메시지의 링크를 클릭하면 단일 비디오를 볼 수있는 랜딩 페이지 환경으로 리디렉션됩니다. 아래 예에는 뷰어 정보를 캡처하는 데 사용할 수있는 리드 양식도 포함되어 있습니다.

행사

이벤트 경험을 사용하면 브랜드가있는 사전 이벤트 대상, 라이브 스트림 이벤트 페이지 및 이벤트 후 비디오 아카이브를 만들어 라이브 이벤트 전, 도중 및 후에 청중을 참여시킬 수 있습니다. 이벤트 경험은 독립 실행 형, 포털 사이트로 만들거나 기존 페이지에 인 페이지 경험으로 포함 할 수 있습니다. 다음 예에서는 라이브 이벤트 포털이 생성되었으며 사전 이벤트 상태를 표시하도록 구성되었습니다 (홈 페이지의 카운트 다운 타이머에 유의).
