    단계별: 포털 환경 만들기 및 게시

    이 주제의 목적은 브라이트코브 갤러리를 사용하여 매력적인 포털 경험을 만드는 방법을 보여주기 위한 것입니다.
     

    이 항목을 완료한 후 다음을 수행할 수 있습니다.

    • Gallery 설정 구성
    • 갤러리를 사용하여 새 포털 환경 만들기
    • 포털 환경에 비디오 추가
    • 포털 환경 사용자 지정
    • 포털 환경 미리 보기 및 게시
    참고: 페이지 내 경험을 만드는 단계별 가이드는단계별 가이드를 참조하십시오. 페이지 내 환경 만들기 및 게시

    고객

    모든 유형의 맞춤형 개발 없이 매력적인 포털 경험을 제작해야 하는 모든 Video Cloud 퍼블리셔입니다.

    전제 조건

    • 갤러리 모듈에 액세스할 수 있는 비디오 클라우드 계정
    • 비디오 클라우드의 실무 지식
    • 필요한 경우 이 항목에 사용된 샘플 비디오를다운로드할수 있습니다.
    • 갤러리 교육을 사용하여 건물 포털 경험을보거나 참석하는 데 도움이 됨

    기본 플레이어로 갤러리 구성

    브라이트코브 갤러리는 갤러리 사이트에서 사용할 기본 비디오 플레이어로 구성할 수 있습니다. 기본 플레이어가 지정되지 않은 경우, 브라이트코브 기본 플레이어가기본값이 됩니다. 기본 플레이어로 갤러리를 구성하려면 다음 단계를 따르십시오.

    1. 비디오 클라우드 스튜디오에로그인합니다.
    2. 현재 비디오 클라우드 계정 이름이 페이지 우측 상단 모서리에 나타납니다. 계정이 여러 개인 경우 계정 선택기 드롭다운을 클릭하고 환경을 만들려는 계정을 선택합니다. 이 계정도 사용할 동영상이 있는 계정이어야 합니다.
    3. 탐색 메뉴에서갤러리를 클릭합니다.
    4. 페이지 상단에 있는설정링크를 클릭합니다.
    5. 갤러리 사이트에서 사용할 플레이어를 선택합니다. 단계를 수행하는 경우 앞에서 만든갤러리 플레이어를선택합니다.
      참고: 가장 좋은 방법은 갤러리 경험을 위한 전용 플레이어를 만들고 스타일을 지정하는 것이 좋습니다.
    6. 저장을 클릭합니다.

    사용자 지정 도메인 이름 만들기

    새 포털 경험을 생성하면 갤러리에서 임의의 URL (예: site-12345.bcvp0rtal.com ) 을 할당합니다. 포털 경험에는 브라이트코브가 제공하는 도메인 (이 주제에서 수행할 작업) 을 할당하거나 고유한 사용자 지정 도메인을 사용할 수 있습니다. 브라이트코브가 제공하는 새로운 도메인을 생성하려면 다음 단계를 따르십시오.

    1. 설정페이지의 왼쪽 탐색에서사용자 지정 도메인 및 SSL을 클릭합니다.
    2. 사용자 지정 도메인 만들기를클릭합니다.
    3. 브라이트코브갤러리.com또는갤러리.비디오도메인을 선택합니다.
    4. 사용자 정의 도메인을 입력합니다. 자신의 이름 또는 회사 이름을 사용합니다. 도메인을 사용할 수 없는 경우 메시지가 표시됩니다. 다른 이름을 입력합니다.
    5. 유효한 도메인을 입력했으면저장을클릭합니다.
    참고: 사용자 지정 도메인은 이후 섹션에서 포털에 할당됩니다.

    새 포털 환경 만들기

    새 갤러리 포털 환경을 만들려면 다음과 같이 하십시오.

    1. 페이지 상단에 있는경험링크를 클릭합니다.
    2. 새 경험을클릭합니다.
    3. 페이지 상단에 있는포털탭을 클릭합니다. 포털 템플릿 목록이 표시됩니다. 선택윤곽템플릿을 찾아 [ 선택 ] 을 클릭합니다.
      참고: 템플릿을 선택하면 변경할 수 없습니다. 갤러리 템플릿에 대한 자세한 내용은갤러리 포털 템플릿 개요항목을 참조하십시오.
    4. 체험 타이틀을위해동물의 세계로들어갑니다. 사전 채우기 경험을 원하십니까?설정에서아니오 감사를선택합니다.
    5. 경험 만들기를클릭합니다.

    갤러리 사이트 편집기가 열립니다. 갤러리 사이트 편집기는 환경을 사용자 지정하는 데 사용되며 페이지 레이아웃을 관리하고, 비디오를 추가하고, 스타일을 선택하고, 포털에 대한 기타 설정을 구성하는 기능을 제공합니다. 모든 사용자화 설정은 왼쪽 탐색 메뉴를 사용하여 액세스합니다.

    사이트 세부 정보 편집

    경험 제목, 설명 및 언어 설정은 갤러리 사이트 편집기를 사용하여 편집할 수 있습니다.

    1. 왼쪽 탐색에서개요를 클릭합니다.
    2. 체험 설명을보려면동물 세계 비디오 포털을추가합니다.
    3. 저장을 클릭합니다.

    페이지 템플릿에 개체 추가

    각 포털 템플릿은 사용자 정의할 수 있는 일련의 페이지 템플릿을 제공합니다. 제공되는 페이지 템플릿은 경험을 처음 만들 때 선택한 템플릿에 따라 다릅니다. 선택 윤곽 템플릿은홈 , 재생및범주페이지 템플릿을 제공합니다. 이 페이지는 페이지 메뉴의 일부로 왼쪽 여백에서 볼 수 있습니다.

    페이지 템플릿에 개체를 추가하는 방법에 대한 자세한 내용은포털 경험 페이지의 컨텐츠 사용자 정의를참조하십시오.

    페이지 템플릿에 개체를 추가하려면 다음과 같이 하십시오.

    1. 왼쪽 탐색에서PAGE를클릭합니다.
    2. 왼쪽 여백에서홈페이지를 클릭합니다. 더하기 아이콘 ( ) 은 페이지에서 객체를 추가할 수 있는 영역을 나타냅니다. 페이지의 정적 텍스트 객체는 클릭하여 편집할 수 있습니다.
    3. 페이지상단에서+이미지 또는 텍스트 추가를클릭합니다.
    4. 이미지탭을 클릭합니다.
    5. 찾아보기를클릭한 다음동물 세계 Logo.png를 찾아 선택합니다. 이 파일은 샘플 에셋의 일부로 제공됩니다. 저장을 클릭합니다.
    6. 이미지가 페이지 머리글에 나타나는지 확인합니다.
    7. 페이지 하단으로 스크롤합니다. [ 내 브랜드]를 클릭합니다. 제목에동물 세계를입력한 다음저장을클릭합니다.
    8. 왼쪽 여백에 있는재생및범주페이지를 클릭합니다. 머리글 이미지와 제목 텍스트도 해당 페이지에 나타납니다. 홈페이지로 돌아갑니다.

    경험 미리 보기

    환경을 미리 보면 포털이 모바일, 태블릿 및 데스크톱 장치에서 어떻게 표시되는지 확인할 수 있습니다. 포털을 사용자 지정하는 단계를 진행하면서 포털을 미리 볼 수 있습니다. 포털을 새 브라우저 탭에서 미리 보거나 공유 가능한 링크를 생성하여 다른 사용자가 포털을 미리 볼 수 있습니다. 포털 환경 미리 보기 및 게시에 대한 자세한 내용은 포털 환경미리 보기 및 게시를참조하십시오.

    포털을 미리 보려면 다음 단계를 수행합니다.

    1. 갤러리 사이트 편집기를 사용하여 사이트를 편집하는 동안미리 보기를클릭한 다음반응형 미리 보기를클릭합니다.
    2. 미리보기가 새 브라우저 탭에서 열립니다.
    3. 사이트 미리 보기 탭을 닫습니다.

    컬렉션 만들기 및 컬렉션에 비디오 추가

    갤러리 사이트 편집기를 사용하여 포털에 표시할 비디오를 선택할 수 있습니다. 포털 경험을 만들 때 비디오는 컬렉션으로 구성됩니다. 컬렉션은 Video Cloud 비디오 메타데이터를 기반으로 생성되며 다음을 포함할 수 있습니다.

    • Video Cloud 재생 목록
    • 키워드
    • 사용자 정의 메타데이터
    • 태그

    YouTube 계정의 동영상도 컬렉션에 추가할 수 있습니다. 모음 만들기 및 사이트에 비디오 추가에 대한 자세한 내용은포털 경험에 비디오 추가를참조하십시오.

    포털 경험에 비디오를 추가하려면 다음 단계를 따르십시오.

    1. 왼쪽 탐색에서비디오를클릭합니다.
    2. 첫 번째 컬렉션 이름을새다음 Enter 키를 누릅니다.
    3. 컬렉션 이름을 클릭하여 컬렉션 편집기를 엽니다.
    4. [ 컬렉션 소스 ] 로 [ 비디오 클라우드 ] 를 선택하고 기준에 대해재생 목록을선택합니다.
    5. 새재생 목록을 클릭합니다. 새 재생목록의 모든 동영상이 동영상 목록에 표시됩니다. 저장을 클릭합니다.
      참고: 재생 목록 비디오 할당을 사용하는 경우 비디오 순서는 미디어 모듈에서 설정한 재생 목록 설정에 따라 결정됩니다.
    6. 동물 재생 목록을 기준으로 사용하여동물이라는다른 컬렉션을 만듭니다.
    7. 바다 생물재생 목록을 기준으로 사용하여 바다 생물라는 이름의 다른 컬렉션을 만듭니다.
    8. 비행 중 조류라는이름의 다른 컬렉션을 만듭니다. 기준으로 [ 사용자 정의 ] 를 선택합니다.
    9. 동영상을 포함하려면태그를선택하고 ' bird in flight ' 태그를 추가합니다. Enter 키를 누릅니다.
    10. 가장 최근동영상을 기준으로 정렬합니다.
    11. Chickadee 비디오가 비디오 컬렉션 목록에 나타나는지 확인합니다. 저장을 클릭합니다.
    12. 4개의 모음이 생성되었는지 확인합니다.
      참고: 컬렉션을 클릭하고 새 위치로 끌어 컬렉션의 순서를 변경할 수 있습니다.

    부지 스타일 변경

    스타일 설정은 환경에 사용되는 사이트 색상 및 글꼴을 사용자 지정하는 데 사용할 수 있습니다. 사이트 스타일 변경에 대한 자세한 내용은포털 경험의 스타일 사용자 지정을참조하십시오.

    대지 스타일을 변경하려면 다음과 같이 하십시오.

    1. 왼쪽 탐색에서스타일을 클릭합니다.
    2. 테마는템플릿에 적용할 수 있는 색상 및 글꼴 집합입니다. 테마를선택합니다. 이 포털은트리톤테마를 사용합니다.
    3. 사용할기본및보조글꼴을 선택합니다. 이 포털은Open Sans글꼴을 사용합니다.
    4. 스타일 메뉴 상단에 있는경험으로 < 돌아가기링크를 클릭합니다.

    모양 및 동작설정의 일부로 추가 스타일 관련 설정을 사용할 수도 있습니다.

    1. 왼쪽 탐색에서모양 및 동작 > 템플릿 설정을 클릭합니다.
    2. 추천 비디오를 선택한 범주의 첫 번째 비디오로 변경을선택합니다.
      참고: 사용 가능한템플릿 설정은환경을 만들 때 선택한 템플릿을 기반으로 합니다. 모든 템플릿에 템플릿 설정이 있는 것은 아닙니다.
    3. 저장을 클릭합니다.

    모양 및 동작 설정에 대한 자세한 내용은포털 환경의 모양 및 동작 사용자 정의를참조하십시오.

    사이트 검색 활성화

    사이트 검색 설정을 사용하여 사이트 검색 기능을 활성화하고 사용자 지정할 수 있습니다. 사이트 검색을 활성화하면 사이트 페이지에 검색 기능이 추가되어 시청자가 콘텐츠를 검색할 수 있습니다. 사이트 검색 사용자 지정에 대한 자세한 내용은포털 경험의 사이트 기능 사용자 지정을참조하십시오.

    사이트 검색을 사용 하려면 다음과이 같이 하십시오.

    1. 왼쪽 탐색에서> 사이트 기능 사이트 검색을클릭합니다.
    2. 사이트 검색 사용옵션을 선택합니다.
    3. 저장을클릭하여 설정을 저장합니다.

    소셜 설정 사용자 지정

    소셜 설정을 사용하면 시청자가 콘텐츠를 인기 있는 소셜 미디어 사이트에 쉽게 공유할 수 있습니다. 시청자가 소셜 미디어 사이트에 빠르게 액세스할 수 있는 방법도 제공할 수 있습니다. 소셜 설정 사용자 지정에 대한 자세한 내용은포털 경험에 대한 소셜 설정 사용자 정의를참조하십시오.

    소셜 설정을 구성하려면 다음과 같이 하십시오.

    1. 왼쪽 탐색에서사이트 기능 > 소셜을클릭합니다.
    2. 모든공유 옵션을선택합니다. 그러면 플레이어 옆에 소셜 공유 아이콘이 추가됩니다.
    3. 조명 시 어둡게 아이콘옵션을 선택합니다.
    4. 저장을 클릭합니다.
    5. 페이지 상단의연결 상태 유지탭을 클릭합니다. 갤러리 사이트는 소셜 미디어 계정 정보를 제공하여 페이지 바닥글에 아이콘을 표시하여 시청자가 소셜 미디어 사이트에 직접 액세스할 수 있도록 합니다.
    6. 소셜 미디어 정보를 입력하거나 Brightcove에 대한페이스 북 이름 , 트위터 핸들과 LinkedIn 사용자 이름 .
    7. 어두운 곳에서 조명 아이콘옵션을 선택합니다.
    8. 저장을클릭하여 설정을 저장합니다.

    비디오 및 재생 설정 사용자 정의

    비디오 및 재생 설정을 사용하여 사용자 지정 플레이어를 선택하고, 리드 양식을 추가하고, 기타 비디오 관련 설정을 사용자 지정할 수 있습니다. 소셜 설정 사용자 지정에 대한 자세한 내용은포털 경험의 비디오 및 재생 설정 사용자 지정을참조하십시오.

    비디오 및 재생 설정을 사용자 지정하려면 다음과 같이 하십시오.

    1. 갤러리 사이트 편집기에서비디오 및 재생 > 플레이어 및 리드 양식을클릭합니다.
    2. 플레이어 비헤이비어설정이 모두 선택되어 있는지 확인합니다. 필요한 경우저장을클릭합니다.
    3. 왼쪽 탐색에서> 비디오 및 재생 비디오를클릭합니다.
    4. 각 비디오에 대한 관련 링크 표시옵션을 선택합니다.
    5. 저장을 클릭합니다.

    사용자 지정 도메인 할당

    포털 환경이 생성되면 임의의 정적 URL이 할당됩니다. 대부분의 경우 보다 친숙한 URL로 사이트에 액세스해야 합니다. 포털 경험에 사용자 지정 도메인을 할당하는 방법에 대한 자세한 내용은 포털 환경에사용자 지정 도메인 할당을참조하십시오.

    포털에 할당된 URL을 보고 사용자 지정 URL을 할당하려면 다음 단계를 수행합니다.

    1. 갤러리 사이트 편집기에서사이트 구성 > URL을클릭합니다. 사이트에 할당 된 URL이 표시됩니다.
    2. 사용자 지정 도메인 추가를클릭합니다.
      참고: 사용자 지정 도메인 추가 단추가없으면 사용자 지정 도메인이만들어지지 않은것입니다.
    3. 이전에 만든 도메인을 선택합니다.
    4. 저장을 클릭합니다.

    경험 게시

    포털 환경을 게시하면 공용 URL에서 환경을 사용할 수 있습니다. 포털 환경 미리 보기 및 게시에 대한 자세한 내용은 포털 환경미리 보기 및 게시를참조하십시오.

    경험을 게시하려면 다음과 같이 하십시오.

    1. 갤러리 사이트 편집기에서게시를클릭합니다.
      참고: 사이트에 할당한 사용자 지정 도메인이 대화상자에 나열되어야 합니다.
    2. 사이트 게시 대화상자에서지금 게시를클릭하여 사이트 게시를 확인합니다.
