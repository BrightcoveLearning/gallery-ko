갤러리 설정 구성

{{ page.description }}
 

갤러리 설정을 구성하려면 갤러리 모듈을 열고 페이지 상단의설정링크를 클릭합니다. 다음 설정을 구성 할 수 있습니다.

  • 플레이어
  • 유튜브
  • 맞춤 도메인 및 SSL
  • 액세스 제어 프로필
  • 맞춤 HTML

기본 Video Cloud 플레이어 선택

갤러리는 갤러리 사이트에서 사용될 계정 전체 기본 비디오 플레이어로 구성되어야합니다. 개별 갤러리 사이트의 플레이어는 계정 전체 기본 플레이어를 재정의 할 수있는 기능을 제공하는 동영상 및 재생 설정의 일부로 구성 할 수도 있습니다. 자세한 내용은 포털 경험에 대한 플레이어 및 리드 양식 설정사용자 지정을 참조하십시오 .

모범 사례로 갤러리 사이트 전용 플레이어를 만들고 스타일을 지정하는 것이 좋습니다. 갤러리 사이트 전용 플레이어를 만들지 않은 경우 먼저 만들어야합니다. 플레이어 생성에 대한 자세한 내용은 플레이어 모듈시작하기 단원을 참조하십시오 .

기본 비디오 플레이어로 갤러리를 구성하려면 다음 단계를 따르십시오.

  1. 딸깍 하는 소리선수왼쪽 탐색에서
  2. 플레이어 목록에서 계정 전체의 기본 플레이어를 선택한 다음저장을 클릭합니다 .
참고: 기본 플레이어를 선택해야 합니다. 그렇지 않으면 포털 경험이 게시되지 않습니다.

관중이 활성화된 플레이어는 관중 아이콘(). 이러한 플레이어는 Audience 플러그인으로 구성되며 비디오 참여 데이터를 추적 할 수 있습니다. 아이콘 위로 마우스를 가져 가면 Audience 연결 정보가 표시됩니다.

대상 지원 플레이어 생성에 대한 자세한 내용은 대상 지원 플레이어만들기 단원을 참조하십시오 .

YouTube 설정 구성

갤러리 사이트에는 Video Cloud 계정 또는 YouTube 계정의 비디오가 포함될 수 있습니다. YouTube 비디오를 갤러리 사이트에 추가하려면 먼저 YouTube 설정을 구성해야합니다. 이러한 설정에 대한 자세한 내용은 YouTube 설정구성을 참조하십시오 .

사용자 정의 도메인 및 SSL 구성

Gallery 사이트를 처음 만들면 형식의 임의 도메인이 할당됩니다<site-slug>.bcovp0rtal.com . 갤러리는 Brightcove에서 제공하는 도메인 또는 사용자 지정 도메인을 사용하여 갤러리 사이트를 게시하는 기능을 제공합니다. SSL을 사용하여 사이트를 보호 할 수도 있습니다. 사용자 지정 도메인 및 SSL 구성에 대한 자세한 내용은 사용자 지정 도메인 및 SSL 구성을 참조하십시오 .

액세스 제어 프로필 구성

갤러리 사이트에 대한 액세스를 제어하기 위해 액세스 제어 프로필을 만들 수 있습니다. 사이트에 대한 액세스는 다음에 의해 제한 될 수 있습니다.

  • IP 주소 -시청자의 IP 주소에 따라 접근이 제한됩니다.
  • SSO 자격 증명 -접속을 위해서는 사용자 이름과 비밀번호가 필요합니다.
  • 액세스 코드 - 사용자가 사이트가 표시되기 전에 액세스 코드를 입력해야 합니다 (사이트를 만들고 테스트할 때 유용함)

액세스 제어 프로파일 생성에 대한 자세한 내용은 포털 경험에 대한 액세스제어를 참조하십시오 .

사용자 정의 머리글 / 바닥 글 HTML 사용

갤러리는 모든 사이트에 대해 사용자 지정 머리글 및 / 또는 바닥 글을 사용하는 옵션을 제공합니다. 이렇게하면 갤러리 사이트를 사용자 지정하여 회사 사이트의 모양과 느낌에 더 가깝게 일치시킬 수 있습니다. 개별 사이트에 대해 사용자 지정 머리글 / 바닥 글을 구성 할 수도 있습니다. 사이트의 사용자 지정 머리글/바닥글 구성에 대한 자세한 내용은 포털 경험의 모양 및 동작사용자 지정을 참조하십시오 .

사용자 지정 머리글 또는 바닥 글을 사용하도록 갤러리를 구성하려면 먼저 사이트에 사용할 머리글 / 바닥 글을 디자인합니다. 머리글 / 바닥 글에는 유효한 HTML, JavaScript 또는 CSS 코드가 포함될 수 있습니다. 완료되면 다음 단계를 따르십시오.

  1. 딸깍 하는 소리사용자 정의 HTML왼쪽 탐색에서
  2. 헤더 HTML을사용자 정의 헤더 HTML필드에 붙여넣습니다.
  3. 바닥글 HTML을사용자 정의 바닥글 HTML필드에 붙여넣습니다.
  4. 저장을클릭하여 변경 사항을 저장합니다.

사용자 지정 머리글 및 바닥 글 사용에 대한 참고 사항

  • 갤러리는 머리글 / 바닥 글에 제공된 HTML의 유효성을 검사하지 않습니다.
  • 머리글 및 바닥 글 HTML에는 명시적인 문자 제한이 없습니다.
  • 외부 CSS 및 JavaScript 파일이 지원됩니다.
  • 사용자 정의 HTML은 반응 형 디자인 원칙을 따라야하므로 장치간에 적절하게 확장됩니다.
  • Brightcove는 사이트가 Bootstrap을 사용할 때 CSS 충돌을 보았습니다 (Gallery도 Bootstrap을 사용함). 충돌을 확인하고 해결하십시오.
