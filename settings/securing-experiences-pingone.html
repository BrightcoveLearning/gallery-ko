페이지 내용

    PingOne을 사용하여 포털 환경 보호

    이 항목에서는 PingOne SSO를 사용하여 포털 환경을 보호하는 방법에 대해 알아봅니다.

    SSO를 사용하여 환경이 보호되는 경우 사용자가 환경에 액세스하기 전에 인증해야 합니다.

    참고: 이 항목의 단계와 이미지는 PingOne SSO의평가판을사용하고 있습니다.

    이 항목의 단계에서는 PingOne 평가판 계정을 만들었으며 전자 메일을 통해 계정을 확인했다고 가정합니다.

    ID 저장소 만들기

    ID 저장소를 만드는 것부터 시작합니다.

    1. PingOne 계정에 로그인합니다. 대시보드페이지가 나타납니다.
    2. 헤더에서설정을 클릭합니다. 설정페이지가 열립니다.
    3. ID 저장소에 연결을누릅니다.
    4. PINGONE 디렉토리를클릭하고다음을클릭합니다.
    5. 디렉토리가 구성되었는지 확인합니다.
    6. 다음을 클릭합니다.
    7. MAP 속성섹션의 경우 기본값을 유지하고완료를누릅니다.
    [PingOne 디렉토리] 페이지에 디렉토리 통계가 표시됩니다.

    응용 프로그램 만들기

    다음으로 SSO를 사용하여 보안을 설정할 응용 프로그램을 만듭니다.

    1. 헤더에서응용 프로그램을클릭합니다. 내 응용 프로그램페이지가 열립니다.
    2. 응용 프로그램 추가를클릭합니다.
    3. 새 SAML 응용 프로그램을선택합니다.
    4. 응용프로그램 이름 및 응용프로그램 설명을입력합니다.
    5. 범주를선택합니다 ( 커뮤니케이션권장).
    6. ( 선택 사항) [ 변경 ] 을 클릭하고 응용 프로그램의 아이콘을 찾아 선택합니다.
    7. 다음 단계로 계속을클릭합니다.
    8. [ 응용 프로그램 구성 ] 페이지에서SAML 메타데이터필드를 찾습니다.
    9. 다운로드링크를 클릭합니다.
    다운로드한 파일은 브라이트코브 갤러리에서 액세스 제어 프로필을 만들 때 필요합니다.

    액세스 제어 프로파일 생성

    SSO를 사용하여 보호하려는 환경을 보호하는 데 사용할 액세스 제어 프로필을 만들어야 합니다.

    1. 브라이트코브 갤러리 계정에 로그인합니다.
    2. 헤더에서설정을클릭합니다.
    3. 왼쪽 탐색에서액세스 제어 프로파일을클릭합니다.
    4. 프로파일 작성을 클릭합니다.
    5. 프로파일에이름을 지정합니다 .
    6. SSO옵션을 선택합니다.
    7. [ SSO 메타데이터 가져오기 ] 섹션을 찾은 다음 [ 파일 선택]을 클릭합니다.
    8. PingOne에서 다운로드한 SAML 메타데이터 파일을 찾아 선택합니다 (이전 섹션 참조).
    9. SAML 2.0 끝점및X.509 인증서의값이 입력되었는지 확인합니다.
    10. 저장을클릭하고 프로파일이 생성되었는지 확인합니다.
    11. 메타데이터 URL필드를 클릭하고 XML 파일을 저장합니다.

    PingOne에서 응용 프로그램 구성을 완료하려면 XML 파일이 필요합니다.

    응용 프로그램 구성 완료

    1. PingOne 응용 프로그램으로 돌아갑니다.
    2. [ 응용 프로그램 구성 ] 섹션에서 [ 메타데이터 업로드 ] 필드를 찾습니다.
    3. 파일 선택을클릭하고 브라이트코브 갤러리에서 액세스 제어 프로필을 만든 후 다운로드한 파일을 찾아 선택합니다.
    4. 어설션 소비자 서비스 (ACS)및엔터티 ID필드가 가져오기 후 값으로 채워졌는지 확인합니다.
    5. 다음 단계로 계속을클릭합니다.
    6. SSO 속성 매핑페이지에서다음 단계로 계속을누릅니다.
    7. 그룹 액세스페이지에서Users @directory그룹을 찾은 다음추가를클릭합니다.
    8. 다음 단계로 계속을클릭합니다.
    9. 마침을클릭합니다.
    10. 응용 프로그램이활성및사용으로 설정되었는지확인합니다.

    환경에 액세스 제어 프로파일 할당

    마지막 단계는 보안하려는 환경에 생성된 액세스 제어 프로필을 할당하는 것입니다.

    1. 브라이트코브 갤러리로 돌아갑니다.
    2. 보호하려는 환경을 편집합니다.
    3. 왼쪽 탐색에서사이트 구성 > 액세스 제어를클릭합니다.
    4. PingOne SSO 구성과 연결된액세스 제어 프로필을선택합니다.
    5. 저장을 클릭합니다.
    6. 경험을게시합니다 .

    환경에 액세스하면 PingOne에서 로그인하라는 메시지가 표시됩니다.
