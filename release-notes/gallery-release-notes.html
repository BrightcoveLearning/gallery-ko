SSO 인증 컨텍스트 요청 비활성화

15년 4월 2021일

SSO 인증 컨텍스트 요청을 비활성화하는 새로운 인터페이스 옵션이 액세스 제어 프로필에 추가되었습니다. 자세한 내용은 액세스 제어 프로파일구성을 참조하십시오 .

자동 재생 비디오 설정

  2019년 9월 4일

새로운자동 재생 비디오설정이 추가되었으며 세부 정보 페이지에서 비디오의 재생 동작을 제어하는 데 사용할 수 있습니다. 자세한 내용은 포털 경험에 대한 플레이어 및 리드 양식 설정사용자 지정을 참조하십시오 .

새롭게 디자인 된 갤러리 탐색

  2018년 9월 27일

포털 경험을 편집 할 때의 탐색 메뉴가 더 직관적이고 인 페이지 경험 편집 환경과 일관되도록 재 설계되었습니다. 자세한 내용은 사이트 편집기를사용하여 포털 환경 사용자 지정을 참조하십시오 .

갤러리 홈페이지에 검색 옵션 추가

2018년 8월 1일

갤러리 홈페이지에검색상자가 추가되어 경험 목록을 검색할 수 있습니다. 검색 할 때 다음 속성이 검색됩니다.

  • 표제
  • 기술
  • 신분증
  • 주형
  • URL

인 페이지 경험에 비디오 추가

2018년 7월 18일

인 페이지 경험에 동영상을 추가 할 때 이제 재생 목록 참조 ID를 사용하는 옵션이 있습니다. 재생 목록 참조 ID를 사용하면 미디어 모듈을 사용하여 참조 ID를 다른 재생 목록에 다시 할당하여 경험의 비디오를 쉽게 변경할 수 있습니다. 경험에 표시되는 동영상을 변경해야 하는 경우 경험을 편집하고 다시 게시할 필요가 없습니다. 자세한 내용은 재생 목록 참조 ID 사용을 참조하십시오 .

리드 양식 변경

2018년 5월 15일

이제 갤러리는 잠재 고객 모듈을 사용하여 리드 양식을 만들고 관리합니다. 잠재 고객은 리드 데이터를 수집하고 데이터 내보내기 옵션을 제공합니다. 자세한 내용은 포털 환경에서 가망 고객 양식사용을 참조하십시오 .

라이브 이벤트 인 페이지 템플릿

2018년 5월 14일

갤러리는 이제 인 페이지 경험의 일부로 라이브 이벤트를 제공하는 데 사용할 수있는 라이브 이벤트 인 페이지 템플릿을 제공합니다. 자세한 내용은 라이브 이벤트 인페이지 환경만들기를 참조하십시오 .

포털 경험 사이트 설정 UI 업데이트

2018년 4월 13일

사이트 설정 UI가 업데이트되어사용자 지정대화 상자에서 새 비디오 설정 대화 상자로비디오 관련설정을 제거합니다. 자세한 내용은 포털 경험에 대한 비디오 설정사용자 지정을 참조하십시오 .

마키 템플릿 설정

2018년 3월 30일

새로운 추천 비디오 동작 설정이 Marquee 템플릿에 추가되었습니다. 자세한 내용은 선택 윤곽 템플릿 설정사용자 정의를 참조하십시오 .

사용자 지정 머리글 및 바닥 글 옵션

2018년 3월 30일

이제 사용자 지정 머리글 및 바닥 글을 사용할 때 기본 머리글 및 바닥 글을 포함 할 수 있습니다. 자세한 내용은 사용자 지정 머리글/바닥글 HTML 구성을 참조하십시오 .

인 페이지 경험 3Play 구성 요소

2018년 2월 1일

이제 3Play Media 대화 형 대본 구성 요소를 인 페이지 환경에 추가 할 수 있습니다. 이를 통해 시청자는 비디오가 재생 될 때 비디오 대본을 볼 수 있습니다. 자세한 내용은 인페이지 환경에 구성 요소추가를 참조하십시오 .

인 페이지 경험 채팅 구성 요소

2018년 1월 17일

이제 채팅 구성 요소를 인 페이지 환경에 추가 할 수 있습니다. 자세한 내용은 인페이지 환경에 구성 요소추가를 참조하십시오 .

사용자 정의 된 액세스 제어 페이지

28일 2017년 11월

  2017년 8월 9일

이벤트 경험에 대한 라이브 이벤트 상태를 구성 할 때 이제 참조 ID를 사용하여 라이브 비디오 자산을 지정할 수 있습니다. 참조 ID를 사용하면 경험을 변경하지 않고도 라이브 비디오 자산에 참조 ID를 할당 할 수 있으므로 더 많은 유연성을 제공합니다. 자세한 내용은 이벤트 경험에 대한 라이브 이벤트 상태만들기를 참조하십시오 .
