갤러리 템플릿 출시 노트

{{ page.description }}

시간이 지남에 따라 Brightcove에서 템플릿에 새로운 기능과 개선 사항을 적용 할 수 있습니다. Brightcove는 주요 템플릿 업데이트를 기존 사이트로 푸시하지 않습니다. 게시자는 새 템플릿을 사용하도록 사이트를 업그레이드 한 다음 게시하기 전에 사이트를 테스트해야합니다. 새 템플릿 버전을 사용하도록 Gallery 사이트를 업데이트하는 방법에 대한 자세한 내용은 갤러리 템플릿업데이트를 참조하십시오 .

템플릿에 대한 변경 사항은 템플릿 버전 릴리스 정보의 일부로 문서화됩니다. 릴리스 정보는 다음 템플릿에 사용할 수 있습니다.

  • 목록
  • 연대기
  • 권위 있는
  • 라이브 이벤트
  • 큰 천막
  • 모자이크
  • 발행자
  • 유리 진열장

카탈로그 릴리스 정보

v5.0 - 2018년 2월 14일

  • 홈페이지에 선택적 제목 추가

v4.0 - 2018년 1월 24일

  • 컬렉션에 비디오가 하나만 포함 된 경우 모바일 장치에서 비디오를 중앙에 배치하도록 캐 러셀이 업데이트되었습니다.
  • 캐 러셀 전환이 더 깔끔해졌습니다.

Chronicle 릴리스 정보

3.0 - 2016년 12월 22일

  • 새로운 이미지 로더를 사용하여 축소판 이미지 개선

버그2.0 - 2016년 10월 31일

  • 영웅 비디오에 대한 두 가지 레이아웃 옵션 추가
    • 새 레이아웃에는 고품질 미리보기 이미지가없는 게시자를 위해 더 작은 히어로 이미지가 있습니다.
  • 스크롤에서 지연 로딩을 위해 페이지 매김을 제거했습니다.
  • 지연로드 된 동영상에 대한 CTA 지원 추가
    • 쇼케이스와 유사하게 Chronicle은 5, 10, 15 개의 동영상 이후에 CTA를 반복하는 옵션이 있습니다.
  • 컬렉션 탐색 / 메뉴 레이아웃 업데이트
    • 전체 화면 탐색을 화면 왼쪽의 탐색 표시 줄로 대체했습니다.
