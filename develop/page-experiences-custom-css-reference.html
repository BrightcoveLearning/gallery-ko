스타일이 지정된 구성 요소

이러한 CSS 클래스는 UI를 통해 스타일이 지정된 항목에 적용됩니다.  이러한 스타일은 이러한 클래스를 사용하여 사용자 정의 CSS로 재정의 할 수 있습니다.  이러한 클래스를 사용하여 UI 스타일을 사용자 정의 HTML에 적용 할 수도 있습니다.  이러한 클래스는 안정적이며 템플릿 버전간에 변경되지 않아야합니다.

ee 구성 요소 스타일 앵커 일반 링크 스타일링
ee 구성 요소 스타일 탐색 탐색 링크 스타일링 (뒤로 버튼, 공유 버튼 등)
ee- 컴포넌트 스타일 -videoInfo 비디오 정보 상자 스타일링 (오버레이 없음)
ee- 컴포넌트 스타일 -videoTitle 비디오 타이틀 스타일링 (오버레이 아님)
ee 구성 요소 스타일 비디오 설명 동영상 설명 스타일 (비 오버레이)
