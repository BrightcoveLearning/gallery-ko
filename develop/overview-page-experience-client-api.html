개요: 인 페이지 경험 클라이언트 API

{{ page.description }}

소개

In-Page Experiences Client API는 Brightcove Player API가 플레이어의 동작을 제어하는 데 사용되는 방식과 유사하게 런타임시 In-Page Experience의 동작을 관리하는 데 도움이되는 JavaScript 라이브러리입니다. API는 하나 이상의 Brightcove 임베디드 환경을 실행하는 모든 페이지에서 사용할 수 있습니다. 타사 브라이트코브 경험 통합을 지원하기 위해 고안되었습니다. 재생 관련 이벤트에 대한 리스너뿐 아니라 현재 플레이어, 비디오 및 상태 정보에 대한 액세스를 제공합니다.

<div data-experience="5bb2134180b4990011750f06"></div>
<script src="https://players.brightcove.net/1752604059001/experience_5bb2134180b4990011750f06/live.js"></script>

경험에 대한 참조를 얻는 가장 쉬운 방법은id ( div태그에) 을 제공하는 것입니다. 페이지 내에서 고유 한 한 원하는 것이 될id수 있지만 경험 ID 자체가 스크립트의 URL에 있기 때문에 (위의 코드의 강조 표시된 부분 참조) 다음과 같이 사용할 수 있습니다.

<div data-experience="5bb2134180b4990011750f06" id="experience_5bb2134180b4990011750f06"></div>
<script src="https://players.brightcove.net/1752604059001/experience_5bb2134180b4990011750f06/live.js"></script>

이제 JavaScript의 경험에 대한 참조를 얻을 준비가되었습니다. 경험은 에서iframe구현되므로 상위 페이지의 스크립트에서 쉽게 통신할 수 없습니다. 대신 경험 자체의사용자 지정 HTML블록에 스크립트를 배치해야 합니다.

var experience = window.top.bcov.gal.getEmbed('experience_5bb2134180b4990011750f06');

경험에 대한 참조가 있으면 API에서 메서드를 호출 할 수 있습니다. 모든 메서드는 자식clientApi객체에 있습니다.

var experience = window.top.bcov.gal.getEmbed('experience_5bb2134180b4990011750f06'),
experienceApi,
video,
current_video = document.getElementById('current_video'),
video_paused = document.getElementById('video_paused');

experienceApi = experience.clientApi;

// get initial video
video = experienceApi.getCurrentVideo();
current_video.textContent = video.name;

// event listeners
experienceApi.on('videoChanged', function() {
  video = experienceApi.getCurrentVideo();
  current_video.textContent = video.name;
});

experienceApi.on('videoStarted', function() {
  video_paused.textContent = 'false';
});

experienceApi.on('videoPaused', function() {
  video_paused.textContent = 'true';
});
