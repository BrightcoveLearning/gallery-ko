인 페이지 경험 API 오류 참조

이 항목에서는 인 페이지 경험 API (플랫폼 API)에 대한 오류 참조를 제공합니다.

인 페이지 경험 API 오류

인페이지 경험 API 오류 참조
오류 예 기술
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 일반적으로template또는 다른 필수 필드를 생략하거나 존재하지 않는 템플릿 지정과 같은 요청 본문에 문제가 있습니다.
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 요청과 함께 전송 된 액세스 토큰이 없거나 액세스 토큰을 만드는 데 사용 된 클라이언트 자격 증명에 요청에 대한 올바른 권한이 없습니다.
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience 5bc2624e0f6f340011b not found"
}
 업데이트하려는 환경이 존재하지 않습니다. 삭제되었거나 요청의 경험 ID가 올바르지 않을 수 있습니다.
