인 페이지 경험 클라이언트 API 샘플

{{ page.description }}

소개

인 페이지 경험 클라이언트 API는 Brightcove Player API와 유사한 JavaScript 라이브러리를 제공합니다. API는 페이지에서 인 페이지 경험의 상호 작용 및 제어를 단순화합니다.

이 기본 샘플은 다음 방법을 보여줍니다.

  1. 경험에 대한 참조를 가져온 다음 메서드가있는 clientApi 개체에 대한 참조를 가져옵니다.
  2. API 메서드를 호출하여 다양한 이벤트에 대한 리스너를 설정하고 현재 플레이어에로드 된 동영상에 대한 정보를 가져옵니다.
  3. 경험 UI에 추가하는 HTML 요소에 정보를 삽입합니다.

인 페이지 경험 예

 

샘플을 만드는 단계

  1. 재생 목록을 사용하여 인 페이지 환경을 만듭니다 (재생 목록이 표시되는 방식은 중요하지 않음).
  2. 경험을 게시하십시오.
  3. 경험 임베드 코드를 복사하여 HTML 페이지에 붙여 넣으십시오.
  4. 추가id속성<div>값이 포함된 태그: custom_in_page_experience .
  5. Studio로 돌아가서 다음 코드를 사용하여 사용자 정의 HTML 구성 요소를 경험에 추가하십시오. 
    <div style="padding: .5em;border: 1px #64AAB2 solid; border-radius:.5em;">
<p>Current Video: <span id="current_video" style="color:RGB(155, 37, 86);font-weight:bold;"></span></p>
<p>Video Paused: <span id="video_paused" style="color:RGB(155, 37, 86);font-weight:bold;"></span></p>
</div>
<script>
  var BCLS = ( function (window, document) {
    var experience = window.top.bcov.gal.getEmbed('customized_in_page_experience'),
    experienceApi,
    video,
    current_video = document.getElementById('current_video'),
    video_paused = document.getElementById('video_paused');

    experienceApi = experience.clientApi;

    // get initial video
    video = experienceApi.getCurrentVideo();
    current_video.textContent = video.name;

    // event listeners
    experienceApi.on('videoChanged', function() {
      video = experienceApi.getCurrentVideo();
      current_video.textContent = video.name;
    });

    experienceApi.on('videoStarted', function() {
      video_paused.textContent = 'false';
    });

    experienceApi.on('videoPaused', function() {
      video_paused.textContent = 'true';
    });

  })(window, document);
</script>
  6. 재생 전, 재생, 재생 후를 표시하도록 구성 요소를 설정합니다.
  7. 변경 사항을 저장하고 경험을 다시 게시하십시오.
  8. 페이지를 탐색하면 "현재 비디오"및 "플레이어 일시 중지됨"메시지가있는 상자가 표시됩니다. 게시는 비동기식이므로 잠시 기다렸다가 브라우저 캐시를 지우고 페이지를 새로 고쳐 변경 사항을 확인해야 할 수 있습니다.
