랜딩 페이지 경험 만들기

{{ page.description }}

랜딩 페이지 템플릿을 사용하면 단일 비디오를 표시하고 선택적으로 뷰어 정보를 캡처 할 수있는 경험을 만들 수 있습니다. 예를 들어 랜딩 페이지 사이트에 대한 링크가있는 새 제품에 대한 이메일 광고를 만들 수 있습니다. 이 사이트는 제품 비디오를 표시하고 시청자가 정보를 입력하여 더 많은 정보를받을 수 있도록합니다.

랜딩 페이지 환경을 만들고 게시하는 프로세스는 다른 템플릿을 사용하는 환경과 유사합니다. 그러나 랜딩 페이지 환경을 만들 때 활용할 수있는 레이아웃 옵션과 사이트 설정이 여기에 설명되어 있습니다.

랜딩 페이지 경험 만들기

랜딩 페이지 환경을 만들려면 다음 단계를 따르십시오.

  1. 갤러리 모듈을 열고 페이지 상단의경험링크를 클릭합니다.
  2. 새 경험 을 클릭합니다 . 템플릿 목록이 표시됩니다.
  3. 랜딩 페이지템플릿을 찾아선택을 클릭합니다 .

갤러리 사이트 편집기가 열립니다.

랜딩 페이지 레이아웃 변경

방문 페이지의 레이아웃은 선택적으로 클릭 유도 문안 또는 리드 양식을 표시하도록 구성 할 수 있습니다. 랜딩 페이지의 레이아웃을 변경하려면 다음 단계를 따르십시오.

  1. 갤러리 사이트 편집기에서페이지왼쪽 탐색에서
  2. 레이아웃변경드롭다운을 클릭합니다.
  3. 레이아웃 옵션 링크를 클릭하십시오.
    • 사이드 바 CTA / 광고 및 리드 양식 -플레이어 오른쪽의 오른쪽 사이드 바에 리드 양식과 클릭 유도 문안이 표시됩니다.
    • 사이드 바 CTA / 광고 -플레이어 오른쪽의 오른쪽 사이드 바에 클릭 유도 문안 만 표시
    • 사이드 바 CTA / 광고 없음 -플레이어 오른쪽의 오른쪽 사이드 바에 아무것도 표시되지 않습니다.

갤러리 사이트 편집기의 미리보기가 업데이트되어 레이아웃을 표시합니다. 필요에 따라 단일 비디오, 이미지, 텍스트, 링크 및 클릭 유도 문안을 추가 할 수 있습니다. 갤러리 사이트 편집기를 사용하여 사이트 사용자 지정에 대한 자세한 내용은 사이트 편집기를사용하여 포털 환경 사용자 지정을 참조하십시오 .

단일 비디오 선택기 사용

랜딩 페이지 템플릿을 사용하는 경우 하나의 비디오 만 표시 할 수 있습니다. 랜딩 페이지 템플릿은 단일 비디오 선택기 대화 상자와 다른 템플릿에서 사용하는 컬렉션 편집기를 사용합니다. 비디오를 선택하려면 다음 단계를 따르십시오.

  1. 딸깍 하는 소리비디오왼쪽 탐색에서
  2. 비디오 소스를선택하면비디오클라우드와 YouTube가 모두 지원됩니다.
  3. 비디오 기준을 선택합니다.
    • 재생 목록 -동영상이 재생 목록에 있습니다. 재생 목록을 선택한 다음 비디오를 선택하십시오.
    • 검색 -동영상 검색 기준을 입력하세요. 비디오 ID, 이름 및 짧고 긴 설명 필드가 검색됩니다.
  4. 하나의 비디오를 선택하십시오.
  5. 저장을클릭하여 변경 사항을 저장합니다.

리드 양식 구성

리드 양식을 사용하려면 리드 양식으로 구성된 Audience 지원 플레이어를 선택해야합니다.

랜딩 페이지에서 사용할 대상 지원 플레이어를 선택하려면비디오 및 재생 > 플레이어 및 리드 양식을 클릭합니다 .

청중이 활성화된 플레이어는 청중 아이콘(). Audience 지원 플레이어를 선택하면 대화 상자가 업데이트되어 리드 양식 및 Audience 연결 정보를 표시합니다.

리드 양식 생성 및 대상 지원 플레이어 작업에 대한 자세한 내용은 포털 환경에서 리드 양식사용을 참조하십시오 .

하위 도메인 구성

사용자 지정 도메인 및 하위 도메인을 사용하면 고유 한 URL에 사이트를 쉽게 게시 할 수 있습니다. 예를 들어 랜딩 페이지를 여러 개 만들어mycustomurl.com/bike_campaign및 에 게시하도록 할 수mycustomurl.com/shoes_campaign있습니다. 사용자 지정 도메인 생성에 대한 자세한 내용은 사용자 지정 도메인 및 SSL 구성및 포털 경험에 사용자 지정 도메인할당을 참조하십시오 .

리드 데이터 내보내기

잠재 고객 모듈을 사용하여 리드 양식에 캡처 된 모든 데이터를 내보낼 수 있습니다. 자세한 내용은 갤러리 리드 데이터내보내기를 참조하십시오 .

참고: 관리자 권한이 있는 사용자만 리드 데이터를 내보낼 수 있습니다.
