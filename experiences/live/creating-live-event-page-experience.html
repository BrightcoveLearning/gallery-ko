라이브 이벤트 인 페이지 경험 만들기

{{ page.description }}

갤러리는 인 페이지 경험의 일부로 라이브 이벤트를 제공하는 데 사용할 수있는 라이브 이벤트 인 페이지 템플릿을 제공합니다. Brightcove는 라이브 스트리밍 비디오를 제공하기위한 다양한 옵션을 제공합니다. 모든 옵션 목록은 라이브 스트림제공을 참조하십시오. 라이브 모듈을 사용하여 라이브 이벤트를 브로드 캐스트하는 자세한 단계는 다음을 참조하십시오.

  • 단계별: Telestream Wirecast와 함께 라이브 모듈 사용
  • 단계별: 오픈 브로드 캐스터 소프트웨어 (OBS)와 함께 라이브 모듈 사용
참고: 라이브 이벤트 인 페이지 경험은 플레이어가 자동 재생되지 않도록 구성된 경우에도 데스크톱 브라우저에서 자동 재생됩니다. 이것은 의도적으로 설계된 것입니다. 갤러리는 카운트 다운이 완료되면 플레이어를 강제로 재생하여 이벤트가 시작되기를 기다리는 페이지에 앉아있는 사용자가 재생을 다시 클릭하지 않고도 시청을 시작할 수 있도록합니다. 라이브 이벤트 인페이지 환경은 iOS 또는 Android 기기에서는 자동 재생되지 않습니다.

새 라이브 이벤트 인페이지 환경을 만들려면 새 인페이지 환경을 만들고라이브 이벤트 인페이지템플릿을 선택합니다.

라이브 이벤트 인 페이지 템플릿에는 다른 인 페이지 경험 템플릿에없는 일부 설정 및 콘텐츠 항목이 있습니다.

  1. 이제 시청자에게 표시
  2. 라이브 이벤트 상태
  3. 소셜 공유
  4. 이미지 자리 표시 자
  5. 이벤트 카운트 다운
  6. 텍스트 자리 표시 자
  7. 캘린더 알림
  8. 동영상 자리 표시 자

이제 시청자에게 표시

뷰어에게지금 표시설정은 뷰어에게 표시되는라이브 이벤트 상태를구성하는 데 사용됩니다. 게시자 인 귀하는 라이브 이벤트 상태를 변경할 책임이 있습니다. 갤러리는 라이브 이벤트 상태를 자동으로 전환하지 않습니다.

변경을클릭하고 표시할 상태를 선택합니다. 이 설정을 변경 한 후에는 환경을 다시 게시 할 필요가 없습니다.

참고: 프로덕션 사이트에 상태 변경이 나타나기까지 몇 분 정도 걸릴 수 있습니다.

라이브 이벤트 상태

라이브 이벤트 인 페이지 경험의 경험 상태는 다른 인 페이지 경험 유형과 다릅니다. 이러한 각 상태의 모양은 페이지 상단의 링크를 클릭하여 사용자 정의 할 수 있습니다.

라이브 이벤트 인 페이지 경험에는 다음과 같은 상태가 있습니다.

  • 사전 이벤트 -라이브 이벤트가 시작되기 전에 라이브 경험이 표시되는 방식
  • 라이브 -라이브 이벤트가 진행되는 동안 경험이 표시되는 방식
  • 이벤트 후 -라이브 이벤트가 종료되었을 때 경험이 표시되는 방식

다양한 경험 상태를 편집하려면 상태를 클릭한 다음경험에 구성 요소 추가필요에 따라. 레이아웃메뉴는 각 상태에 대한 추가 사용자 지정 옵션을 제공합니다.

소셜 공유

소셜 공유 옵션은 이벤트 및 비디오를 소셜 미디어 계정에 공유 할 수있는 링크를 제공합니다.

소셜 공유 설정을 구성하려면공유 구성링크 또는 클릭설정> 사회의왼쪽 탐색에서 소셜 설정은 다음으로 구성됩니다.

  • 이벤트 공유
  • 비디오 공유

이벤트 공유

이벤트 공유 설정을 통해 시청자는 이벤트 페이지를 소셜 미디어 계정에 공유 할 수 있습니다. 이벤트 세부 정보를 구성하려면 다음 단계를 따르십시오.

  1. 추가하려면이벤트 이미지 , 클릭이미지 추가그리고:
    • 파일 시스템에서 이미지를 끌어서 놓기 영역에 놓으십시오.
    • 찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다.
    • 원격 이미지 파일의 URL을 입력하십시오.
  2. 이벤트 제목및이벤트 설명을 입력합니다 .
  3. 이벤트 공유 옵션이 표시되는 방식을 선택하고, 공유하다문자를 보내거나 소셜 아이콘을 표시합니다.
  4. 사용 가능한 공유 옵션을 선택하십시오.
  5. 경험이 포함될페이지 URL을입력합니다.
  6. 생성을 클릭합니다 . 이렇게 하면 페이지<head>섹션에 추가할 수 있는 메타 태그가 생성됩니다. 이 기능을 사용하면 시청자가 경험을 공유 할 때 소셜 사이트에서 이벤트 이미지, 제목 및 설명을 표시 할 수 있습니다.

비디오 공유

동영상 공유 설정을 통해 시청자는 동영상을 소셜 미디어 계정으로 공유 할 수 있습니다. 비디오 공유를 구성하려면 다음 단계를 따르십시오.

  1. 비디오 공유 옵션이 표시되는 방식을 선택하고, 공유하다문자를 보내거나 소셜 아이콘을 표시합니다.
  2. 사용 가능한 공유 옵션을 선택하십시오.
  3. 저장을클릭하여 변경 사항을 저장합니다.

이미지 자리 표시 자

PRE-EVENT 및 POST-EVENT 상태는 히어로 이미지를 표시하는 기능을 제공합니다.

영웅 이미지를 추가하려면

  1. 이미지추가링크를 클릭합니다.
  2. 이미지의이름을입력합니다 (ASCII 문자만 허용됨).
  3. 이미지를 지정하려면 다음을 수행 할 수 있습니다.
    • 파일 시스템에서 이미지를 끌어서 놓기 영역에 놓으십시오.
    • 찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다.
    • 원격 이미지 파일의 URL을 입력하십시오.
  4. 이미지 크기를 설정합니다 .
    • 이미지 자르기 - 이미지의 가로 세로 비율에 따라 전체 16:9 공간을 채우지 만 위/아래 또는 왼쪽/오른쪽 중 일부가 손실되도록 가운데에서 이미지를 자릅니다.
    • 구성 요소 경계 내에 이미지 포함 - 16:9 공간의 경계 내에 이미지를 포함하지만 이미지의 위/아래 또는 왼쪽/오른쪽 공백은 종횡비에 따라 나타납니다.
    • 이미지 늘이기 - 이미지의 종횡비를 무시하고 16:9 공간을 완전히 채우도록 강제합니다. 그러면 이미지가 왜곡됩니다.
    • 원본 이미지 크기 - 원본 이미지 크기를 사용합니다. 크기 조정이 수행되지 않습니다. 응답하지 않음
  5. 이미지가 표시되어야하는 상태를 선택하십시오.
  6. 저장을클릭하여 이미지를 경험에 추가합니다.

이벤트 카운트 다운

카운트 다운 타이머는 라이브 이벤트가 시작되기 전의 월, 일, 시간, 분 및 초 수를 표시합니다. 샘플 카운트 다운 타이머가 아래에 나와 있습니다.

참고: 카운트다운 타이머의 스타일은스타일왼쪽 탐색에서

카운트 다운 타이머를 추가하려면 다음 단계를 따르세요.

  1. 페이지에서 이벤트 카운트다운구성링크를 클릭합니다. 이벤트 설정 페이지가 열립니다.
  2. 이벤트 카운트다운표시를 선택합니다 .
  3. 시작 날짜 , 시작 시간및이벤트 시간대를 설정합니다 .
  4. 전환 메시지를 입력합니다 . 이 메시지는 카운트다운 타이머가 0에 도달했을 때 실시간 스트림이 시작되기 전에 표시됩니다.
  5. 저장을 클릭합니다 .

텍스트 자리 표시 자

PRE-EVENT 및 POST-EVENT 상태는 정적 텍스트를 표시하는 기능을 제공합니다.

정적 텍스트를 추가하려면 다음을 수행합니다.

  1. 텍스트추가링크를 클릭합니다.
  2. 경험에 추가 할 텍스트를 입력하십시오. 도구 모음을 사용하여 텍스트 서식을 지정할 수 있습니다.
  3. 저장을클릭하여 텍스트를 경험에 추가합니다.

캘린더 알림

사용자가 링크를 클릭하여 캘린더 미리 알림을 다운로드 할 수 있도록 캘린더 미리 알림을 구성 할 수 있습니다. 샘플 캘린더 미리 알림은 다음과 같습니다.

알림을 클릭하면 캘린더 옵션이 표시됩니다.

참고: 캘린더 미리 알림의 스타일은 다음을 클릭하여 스타일을 지정할 수 있습니다. 스타일왼쪽 탐색에서

캘린더 미리 알림을 추가하려면 다음 단계를 따르세요.

  1. 페이지에서 일정 미리 알림구성링크를 클릭합니다. 이벤트 설정 페이지가 열립니다.
  2. “캘린더에 추가” 미리 알림표시를선택합니다.
  3. 종료 날짜 , 종료 시간및이벤트 제목을 설정합니다 . 이벤트 설명및이벤트 위치는 선택사항입니다.
    참고: 이벤트 카운트다운표시옵션을 선택하면 인터페이스가 약간 다르게 나타납니다.
  4. 저장을 클릭합니다 .

동영상 자리 표시 자

PRE-EVENT, LIVE 및 POST-EVENT 상태에서 다른 비디오를 표시 할 수 있습니다. Video Cloud 비디오 및 YouTube 비디오를 경험에 추가 할 수 있습니다.

참고: YouTube 동영상을 경험에 추가하려면 먼저 YouTube 설정을구성해야 합니다 .

표시할 비디오를 선택하려면 다음과 같이 하십시오.

  1. 클릭동영상 선택링크 또는 클릭비디오왼쪽 탐색에서 비디오선택대화 상자가 나타납니다.
  2. 비디오를 선택할 상태를 선택하십시오.
    • 이벤트 전
    • 라이브
    • 이벤트 후
  3. 비디오소스 선택 :
    • 비디오 클라우드 -Video Cloud 비디오 콘텐츠가 사용됩니다.
    • 유튜브 -YouTube 동영상 콘텐츠가 사용됩니다.
  4. 비디오를검색하고 클릭하여 선택한 다음 선택을 클릭합니다 . 동영상이 동영상 목록에 추가됩니다.
  5. 라이브또는포스트 이벤트섹션을 클릭하거나 비디오선택을 클릭하여 추가 비디오를 선택합니다.

상호 작용 및 컴패니언 구성 요소를 이벤트 전 및 이벤트 후 비디오에 추가하고 비디오가 재생 될 때 표시 될 수 있습니다. 상호 작용에는 다음이 포함됩니다. 연결그리고카드비디오가 재생될 때 플레이어의 일부로 표시됩니다. 컴패니언 구성 요소에는 HTML , 이미지및텍스트객체가 포함되며 플레이어 외부에 표시됩니다.

참고: VOD 에셋이 아니라면 라이브 비디오에서 상호 작용 및 컴패니언 컴포넌트를 사용할 수 없습니다.

상호 작용으로 구성된 비디오는 상호 작용 유형을 나타내는상호 작용열에 아이콘이 표시됩니다. 상호 작용 및 컴패니언 구성 요소 추가 및 구성에 대한 자세한 내용은 다음을 참조하십시오.

  • 인 페이지 경험에 상호 작용 추가
  • 인 페이지 경험을위한 컴패니언 구성 요소 구성

라이브 비디오 선택

라이브상태를 선택하면 라이브 비디오 에셋에 대한 Video Cloud 비디오 ID 또는 참조 ID를 입력해야 합니다. 라이브 비디오의 소스는 항상 Video Cloud입니다.

참조 ID옵션을 선택하면 환경에서 비디오 ID를 변경하지 않고도 미디어 모듈에서 라이브 비디오의 참조 ID를 변경할 수 있습니다. 예를 들어, 라이브이벤트의 참조 ID로 현재 라이브 비디오를 표시하도록 선택할 수있습니다. 라이브 경험을 만들 때참조 ID를선택하고라이브 이벤트를 입력합니다 .

이제 새로운 라이브 이벤트가 있을 때마다 미디어 모듈을 사용하여 해당 동영상의 참조 ID를라이브 이벤트로 설정합니다 .

참조 ID를 사용하면 새 이벤트가 시작될 때 라이브 경험을 편집하고 다시 게시 할 필요가 없습니다. 새 이벤트가 시작되면 미디어 모듈을 사용하여 라이브 비디오 자산의 비디오 참조 ID를 편집합니다.

비디오 검색

검색필드는 비디오를 검색하는 데 사용할 수 있습니다. Video Cloud 비디오를 검색 할 때 Gallery는 비디오 이름, 짧고 긴 설명, 태그 및 사용자 정의 필드를 검색합니다. YouTube 동영상을 검색 할 때 Gallery는 YouTube 검색 API를 사용합니다.

검색되는 필드를더 세밀하게 제어하고 텍스트를 포함 및 제외하려면 추가 검색 옵션을클릭합니다. 이 옵션은 Video Cloud 소스가 사용되는 경우에만 사용할 수 있습니다.
