포털 이벤트 경험을위한 라이브 이벤트 상태 생성

{{ page.description }}

라이브 이벤트 포털 템플릿은사전 이벤트 , 라이브 이벤트, 사후 이벤트 , 비디오 아카이브 및 비디오 세부 정보 페이지 템플릿을 제공합니다. 이러한 페이지 템플릿은 사용자 정의 할 수 있으므로 라이브 이벤트 전, 도중 및 후에 사이트의 모양이 달라집니다. 라이브 이벤트 상태는 라이브 이벤트 중에 사이트에 표시되는 것입니다.

사이트의 라이브 이벤트 홈 페이지 상태를 만들려면 라이브 이벤트 포털 템플릿을 사용하여 새 포털 환경을 만듭니다. 갤러리 사이트 편집기에서페이지왼쪽 탐색에서홈페이지: 라이브 .

라이브 이벤트 홈 페이지 상태는 다음 컨텐츠 영역을 제공합니다.

  • A-홈 로고 및 이름 -뷰어를 홈 페이지로 되 돌리는 데 사용됩니다. 여기에는 두 개의 자리 표시자가 있으므로 예를 들어 이미지 뒤에 텍스트를 표시하도록 선택할 수 있습니다.
  • B-비디오 아카이브 링크 -사이트에 대해 생성 된 모든 비디오 컬렉션을 보는 데 사용됩니다.
  • C-트위터 설정 -트위터 피드 위젯을 사용할 수있는 기능을 제공합니다. 트위터 피드구성하기를 참조하십시오 .
  • D-배경 이미지 -배경 이미지 추가에 사용
  • E-라이브 비디오 설정 -라이브 영상 ID를 설정합니다. 라이브 비디오구성을 참조하십시오 .
  • F-라이브 이벤트 이름 -정적 문자 메시지
  • G- "지금 일어나고 있습니다"표시기 - "지금 일어나고 있습니다"표시기 숨기기 / 표시
  • H - 채팅 설정 - 타사 채팅 위젯 (예: Chatroll) 을 포함시키는 선택적 영역입니다. 채팅 위젯구성하기를 참조하십시오 .
  • I-비디오 그리드 박스 -사용자 정의 HTML 또는 이미지를 포함한 클릭 유도 문안을 추가하는 데 사용됩니다. 보다포털 경험에서 클릭 유도문안 구성자세한 내용은.
  • J-바닥 글 개체 -페이지 바닥 글에 텍스트 및 링크를 추가하는 데 사용됩니다.

포털 페이지 템플릿에 이미지, 링크 및 텍스트를 추가하는 방법에 대한 자세한 내용은 포털 경험 페이지의 콘텐츠사용자 지정을 참조하십시오 .

참고: 사이트 색상은 선택한 테마에 따라 달라집니다.

Twitter 피드 구성

사이트는 페이지에 Twitter 임베드로 구성 할 수 있습니다. 사용 가능한 임베드 유형에 대해서는 Twitter 웹 사이트를 확인하십시오 . 타임 라인을 좋아하는 샘플은 아래와 같습니다.

Twitter 피드를 추가하려면 다음 단계를 따르세요.

  1. 트위터 사이트에서 위젯을 만듭니다 . 트위터 위젯 생성에 대한 도움말은 Twitter 개발자 문서를 참조하십시오 .
  2. 페이지에서 Twitter 설정링크를 클릭합니다.
  3. 트위터 위젯의 HTML을 복사하여트위터 피드설정에 붙여 넣습니다.
  4. 저장을 클릭합니다 .

라이브 비디오 구성

라이브 비디오구성링크를 클릭하여 라이브 비디오 자산의비디오 ID또는참조 ID로 라이브 비디오 설정을 구성합니다. 동영상 ID를 사용하는 경우 새 라이브 이벤트가 생성 될 때마다 새 동영상 ID로 환경을 업데이트해야합니다. 참조 ID를 사용하면 더 많은 유연성이 제공됩니다. 새 라이브 이벤트가 생성되면 지정된 참조 ID를 미디어 모듈의 새 비디오에 할당하기 만하면 경험을 변경할 필요가 없습니다.

타사 채팅 위젯을 추가하려면 다음 단계를 따르세요.

  1. 페이지에서채팅 설정링크를 클릭합니다.
