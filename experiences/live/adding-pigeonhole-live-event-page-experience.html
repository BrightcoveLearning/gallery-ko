라이브 이벤트 인페이지 경험에 비둘기 구멍 Q&A 추가하기

{{ page.description }}

Pigeonhole을 사용하여 라이브 이벤트 인페이지 경험에 이벤트 기반 상호작용(라이브 Q&A, 설문조사, 설문조사 등)을 쉽게 추가할 수 있습니다. 일반적으로 이벤트에 대해 새 Pigeonhole이 생성 된 다음 라이브 이벤트에 대한 세션이 생성됩니다. 세션이 생성되면 Pigeonhole 소스 코드가 라이브 이벤트 인 페이지 템플릿에 붙여 넣어집니다. 임베드 코드로 인해 Pigeonhole Q & A 창이 라이브 이벤트 플레이어 옆에 표시됩니다.

Pigeonhole 이벤트 및 세션 생성

라이브 이벤트를위한 새 Pigeonhole을 만든 다음 Q & A가 활성화 될 각 비디오에 대한 세션을 추가해야합니다.

  1. Pigeonhole 계정에 로그인하십시오.
  2. + Pigeonhole 추가를클릭하고 사용할 계정을 클릭합니다.
  3. 이벤트 이름및암호를 입력합니다 .
  4. 이벤트의시작및종료날짜를 입력합니다. 이 날짜는 Pigeonhole이 활성화되는 기간을 제어합니다.
  5. 시간대및사용자 지정 브랜딩옵션 (해당되는 경우) 을 선택합니다.
  6. 계속을 클릭합니다 .
  7. 이벤트가 생성되었으므로 왼쪽 탐색에서 Agenda항목이 선택됩니다. 세션추가 > 새로만들기 를 클릭합니다 .
  8. 세션 유형Q&A를 선택합니다 .
  9. 세션 이름을 입력합니다 .
  10. 시작및종료날짜 및 시간을 입력합니다.
  11. Q&A 설정을 적절히구성합니다.
  12. Q&A 추가를 클릭합니다 .세션이 생성되었는지 확인합니다.

Pigeonhole Embed 코드 생성

Pigeonhole이 경험에 표시되도록하기 위해 경험에 사용자 지정 구성 요소를 사용하여 다음 코드가 추가됩니다. 코드 형식은 다음과 같습니다.

<iframe src="https://pigeonhole.at/이벤트 암호/NS/세션 ID“스타일 ="테두리: 1px #333333 고체; 테두리 반경: 3px” 이름 ="비둘기 홀 라이브” 스크롤 = “아니오” 프레임 테두리 ="1" 마진 높이 = “0px” 여백 너비 = “0px”높이 = “600"너비 =”100% "화면 허용>  <  /아이프레임 >

이벤트 암호및세션 ID 값을가져오려면 다음 단계를 수행합니다.

  1. 비둘기집에서의제왼쪽 탐색에서
  2. 세션탭에서 이벤트 옆에 있는 드롭다운 화살표를 클릭합니다. 세션 ID가 표시됩니다.
  3. 이벤트 암호를 얻으려면 헤더에서 이벤트 이름을 찾으십시오. 이벤트 암호는 이벤트 이름 아래에 표시됩니다.

이벤트 패스 코드와 세션 ID가 포함 된 위의 코드가 경험에 사용됩니다.

경험에 Pigeonhole 추가

Brightcove는 라이브 이벤트 인 페이지 경험의 오른쪽에 Pigeonhole을 추가 할 것을 권장합니다. 현재 Pigeonhole 플러그인은 특히 오른쪽을 대상으로합니다.

참고: 경험 왼쪽에 Pigeonhole 을 추가해야 하는 경우 계정 관리자에게 문의하십시오.

라이브 이벤트 인 페이지 환경에 Pigeonhole을 추가하는 작업은 사용자 지정 구성 요소를 사용하여 수행됩니다. Pigeonhole을 추가하려면 다음 단계를 따르십시오.

  1. 갤러리 모듈을 엽니 다.
  2. 편집하려면 라이브 이벤트 인 페이지 경험을 클릭하십시오.
  3. 더하기 아이콘() 경험의 오른쪽에 있습니다.
  4. 딸깍 하는 소리관습구성 요소 선택기에서.
  5. HTML 구성 요소의이름을입력합니다.
  6. 이전 섹션의 비둘기 HTML에사용자 정의 HTML 코드필드에 붙여넣습니다.
  7. HTML이 표시되어야하는시기를 선택하십시오.
  8. 저장을 클릭합니다 . HTML은 경험에서 렌더링됩니다.
  9. 왼쪽 탐색에서설정 > 고급을 클릭합니다 .
  10. 다음 코드를 JavaScript URL 필드에 붙여 넣습니다. 
    https://static.pigeonhole.at/brightcove-plugin/phl-brightcove-gallery-inpage.js
  11. 저장을 클릭합니다 .
  12. 변경사항을 경험에 게시합니다.

경험을 미리 볼 때 Pigeonhole이 표시되어야합니다.
