포털 이벤트 경험에 대한 이벤트 후 상태 만들기

{{ page.description }}

라이브 이벤트 포털 템플릿은사전 이벤트 , 라이브 이벤트, 사후 이벤트 , 비디오 아카이브 및 비디오 세부 정보 페이지 템플릿을 제공합니다. 이러한 페이지 템플릿은 사용자 정의 할 수 있으므로 라이브 이벤트 전, 도중 및 후에 사이트의 모양이 달라집니다. 이벤트 후 상태는 라이브 이벤트가 완료되었을 때 사이트에 표시되는 상태입니다.

사이트에 대한 이벤트 후 홈페이지 상태를 만들려면 새 라이브 이벤트 포털 환경을 만듭니다. 갤러리 사이트 편집기에서페이지왼쪽 탐색에서홈페이지: 포스트 이벤트 .

이벤트 후 홈 페이지 상태는 다음과 같은 컨텐츠 영역을 제공합니다.

  • A-홈 로고 및 이름 -뷰어를 홈 페이지로 되 돌리는 데 사용됩니다. 여기에는 두 개의 자리 표시자가 있으므로 예를 들어 이미지 뒤에 텍스트를 표시하도록 선택할 수 있습니다.
  • B-비디오 아카이브 링크 -사이트에 대해 생성 된 모든 비디오 컬렉션을 보는 데 사용됩니다.
  • C-트위터 설정 -트위터 피드 위젯을 사용할 수있는 기능을 제공합니다. 트위터 피드구성하기를 참조하십시오 .
  • D-배경 이미지, 텍스트 및 링크 -배경 이미지, 자막 텍스트 및 링크 추가에 사용
  • E-라이브 이벤트 이름 -정적 문자 메시지
  • F-실시간 스트림 종료 메시지 -정적 문자 메시지
  • G-이메일 가입 설정 -이메일 등록을 구성하는 데 사용됩니다. 이메일 가입구성을 참조하십시오 .
  • H-동영상 추가 -사이트에 비디오 컬렉션을 추가하는 데 사용됩니다. 보다동영상 추가자세한 내용은.
  • 나는-비디오 그리드 박스 -사용자 정의 HTML 또는 이미지를 포함한 클릭 유도 문안을 추가하는 데 사용됩니다. 보다포털 경험에서 클릭 유도문안 구성자세한 내용은.
  • J-바닥 글 개체 -페이지 바닥 글에 텍스트 및 링크를 추가하는 데 사용됩니다.

갤러리 페이지 템플릿에 이미지, 링크 및 텍스트를 추가하는 방법에 대한 자세한 내용은 포털 경험 페이지의 콘텐츠사용자 지정을 참조하십시오 .

참고: 사이트 색상은 선택한 테마에 따라 달라집니다.

Twitter 피드 구성

사이트는 페이지에 Twitter 임베드로 구성 할 수 있습니다. 사용 가능한 임베드 유형에 대해서는 Twitter 웹 사이트를 확인하십시오 . 타임 라인을 좋아하는 샘플은 아래와 같습니다.

Twitter 피드를 추가하려면 다음 단계를 따르세요.

  1. 트위터 사이트에서 위젯을 만듭니다 . 트위터 위젯 생성에 대한 도움말은 Twitter 개발자 문서를 참조하십시오 .
  2. 페이지에서 Twitter 설정링크를 클릭합니다.
  3. Twitter 위젯의 HTML을 복사하여트위터 피드설정.
  4. 저장을 클릭합니다 .

이메일 등록 구성

사이트 방문자가 이메일 주소를 제공 할 수 있도록 사이트를 구성 할 수 있습니다.

이메일 등록을 추가하려면 다음 단계를 따르십시오.

  1. 페이지에서이메일 가입 설정링크를 클릭합니다.
  2. 필요에 따라 이메일 등록 속성을 구성합니다.
    • 이메일 리드 양식 표시 안함 -이메일 양식을 표시하지 마십시오
    • 템플릿과 함께 제공된 이메일 양식 사용 -템플릿과 함께 제공된 기본 이메일 양식을 사용하려면 선택하십시오.
    • 마케팅 자동화 소프트웨어에서 제공하는 사용자 정의 HTML 사용 -이메일 캡처에 사용자 정의 양식 HTML을 사용하려면 선택하십시오.
  3. 저장을 클릭합니다 .
  4. 이메일 설정이 구성되면 페이지로 돌아가서이메일 가입 메시지링크를 클릭하고 이메일 가입 필드 위에 나타나는 텍스트를 설정합니다.

이메일 양식에서 캡처한 모든 데이터는 Audience 모듈의 갤러리 리드내보내기옵션을 사용하여 내보낼 수 있습니다. 사용자 정의 HTML 옵션을 사용하여 캡처 한 리드는 마케팅 자동화 소프트웨어 내부에서 볼 수 있습니다.

비디오 추가

+비디오 추가링크를 클릭하여 사이트에 비디오 컬렉션을 추가합니다. 컬렉션 생성에 대한 자세한 내용은 포털 경험에 비디오추가를 참조하십시오 .

컬렉션 목록의 첫 번째 컬렉션은 이벤트 후 홈페이지에 표시됩니다. 모든 컬렉션을 보려면 홈 페이지에서 비디오 중 하나를 클릭하거나 시청자가 모든 컬렉션을 볼 수 있도록비디오 아카이브 링크객체를 구성합니다.
