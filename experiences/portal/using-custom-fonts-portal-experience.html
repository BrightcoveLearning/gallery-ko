포털 경험에서 사용자 정의 글꼴 사용

{{ page.description }}

갤러리 사이트 편집기는 다음 기능을 제공합니다. 스타일을 사용자 정의테마, 색상 및 글꼴을 변경하여 Portal Experience의 포털 경험에서 사용자 지정 글꼴을 사용할 수도 있습니다. 이 주제에서는절정글꼴구글 폰트 디렉토리포털 경험을 사용자 정의하는 데 사용됩니다.

사용자 정의 글꼴 파일에 연결

Google Font 디렉토리는 웹 페이지에서 사용할 수 있는 오픈 소스 글꼴을제공합니다. 사용할 글꼴을 찾은 다음 이 스타일선택을 클릭합니다 . 이 항목에서는 Acme글꼴이 사용됩니다.

Google 도움말에 따라 다음 줄이 문서에 포함되어야합니다.

<link rel="preconnect" href="https://fonts.gstatic.com"> 
<link href="https://fonts.googleapis.com/css2?family=Acme&display=swap" rel="stylesheet">

포털 경험에 이러한 라인을 포함하려면 다음 단계를 따르십시오.

  1. 경험을 편집하십시오.
  2. 딸깍 하는 소리외모와 행동> 머리글과 바닥 글왼쪽 탐색에서
  3. <link>태그를사용자 정의 헤더 HTML필드에 붙여넣습니다.
  4. 저장을 클릭합니다 .

사용자 지정 글꼴 용 CSS 만들기

사용자 정의 글꼴을 사용하려면style태그가 포털에 추가됩니다. 사용자 정의 헤더 HTML필드를 사용하여 적절한 텍스트 개체의 스타일을 지정합니다.

브라우저와 함께 제공되는 개발 도구는 스타일을 지정하려는 페이지의 개체 (글꼴을 변경하려는 개체)의 클래스 이름을 가져 오는 데 유용합니다. 요소 이름은 사용중인 템플릿에 따라 변경됩니다. 아래 예에서는 추천 비디오의 글꼴과 페이지의 비디오 그리드에 나타나는 비디오 이름에 사용되는 글꼴을 변경합니다. Google 크롬을 사용하여 스크린 샷을 찍습니다.

  1. 브라우저 개발 도구를 사용하여 글꼴을 변경하려는 경험의 요소를 검사하십시오. 이 경우 추천 비디오 타이틀이 검사 중입니다. 개체의 클래스 이름을 찾습니다.
  2. 사용자 정의 헤더 HTML필드에 적절한 CSS를 추가하여 새 글꼴로 추천 비디오 제목의 스타일을 지정합니다. 
      
      <style>
        .video-name {
         font-family:Acme;
       }
      </style>
  3. 그리드에서 비디오 타이틀의 클래스 이름을 찾습니다.
  4. 기존<style>태그 안에 적절한 CSS를 추가하여 새 글꼴로 그리드의 비디오 제목에 스타일을 지정합니다. 
    
      .bc-video-browser .bc-video-grid-items .bc-video-grid-item .caption {
       font-family:Acme;
    }
  5. 사용자 지정 헤더 HTML필드에 다음이 포함되어 있는지 확인합니다. 
    <link rel="preconnect" href="https://fonts.gstatic.com"> 
    <link href="https://fonts.googleapis.com/css2?family=Acme&display=swap" rel="stylesheet">
  <style>
      .video-name {
      font-family:Acme;
    }

    .bc-video-browser .bc-video-grid-items .bc-video-grid-item .caption {
    font-family:Acme;
  }
</style>
    참고: 또한 CSS를 별도의 파일에 저장하고사용자 정의 CSS URL필드를 사용하여 경험과 연결할 수도 있습니다.
  6. 저장을 클릭합니다 .
  7. 미리 보기 > 반응형 미리 보기를클릭하여 업데이트된 글꼴이 표시되는지 확인합니다.
  8. 완료되면 경험을게시합니다.

템플릿에 대한 전체 CSS 파일은모양 및 동작 > 사용자 지정설정의 일부로 CSS 보기링크를 클릭하여 볼 수 있습니다. 글꼴의 모든 인스턴스를 변경하려면 템플릿font-family CSS에서 의 모든 항목을 검색하고 해당 항목을<style>태그.
