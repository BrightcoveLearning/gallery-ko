포털 환경에서 리드 양식 사용

{{ page.description }}

리드 양식은 시청자가 동영상을 볼 때 시청자 정보를 캡처하는 데 사용할 수 있으며 다음 유형의 경험과 함께 사용할 수 있습니다.

  • 문
  • 라이브 이벤트 포털
  • 방문 페이지

포털, 라이브 이벤트 포털 및 일부 랜딩 페이지 환경에서는 리드 양식이 플레이어 내부에 표시됩니다. 리드 양식은 지정된 수의 비디오를 본 후 또는 미리 결정된 시간 간격이 지난 후에 표시되도록 구성 할 수 있습니다. 다음은 플레이어 내부에 표시되는 리드 양식의 예입니다.

랜딩 페이지 환경은 페이지 본문에 리드 양식을 표시하도록 구성 할 수도 있습니다. 예를 들면 다음과 같습니다.

리드 데이터는 쉼표로 구분 된 또는 Excel 파일 형식으로 다운로드 할 수 있습니다.

리드 양식을 사용하는 단계는 다음과 같습니다.

  • Audience 모듈을 사용하여 리드 양식 만들기
  • 잠재 고객 지원 플레이어에 리드 양식 할당
  • 리드 양식을 사용하도록 환경 구성
  • 리드 데이터 내보내기

리드 양식 만들기

잠재 고객 모듈을 사용하여 리드 양식을 만듭니다. 사용 가능한 모든 옵션을 포함하여 가망 고객 양식을 만드는 전체 단계는 대상 가망 고객 양식만들기를 참조하십시오 . 리드 양식을 만들려면 다음 단계를 따르세요.

  1. Audience 모듈을 엽니 다.
  2. 딸깍 하는 소리리드 양식왼쪽 탐색에서
  3. 가망 고객 양식생성을 클릭합니다 .
  4. 가망 고객 양식의이름을입력합니다.
  5. 리드 양식 설정을 구성한 다음저장을 클릭합니다 .
참고: 입력된 리드 양식 데이터에는 유효성 검사가 수행되지 않습니다.
참고: 리드 양식이 랜딩 페이지 환경에서 사용되고 경험이 페이지 본문에 양식을 표시하도록 구성된 경우 리드 양식 타이밍 및 표시 설정은 무시됩니다.

잠재 고객 지원 플레이어에 리드 양식 할당

리드 양식은 Audience 지원 플레이어에서만 표시 할 수 있습니다. Audience 지원 플레이어는 Audience 플러그인으로 구성된 Brightcove 플레이어입니다. 청중이 가능한 플레이어는 청중 모듈 내부에서 볼 수있는 비디오 참여 데이터를 기록합니다. Audience 지원 플레이어를 만들려면 다음 단계를 따르십시오.

  1. Audience 모듈을 엽니 다.
  2. 오디언스 플레이어를 클릭합니다 .
  3. 플레이어생성을 클릭하여 대상 지원 플레이어를 새로 만듭니다.
    참고: 기존 플레이어에 Audience 플러그인을추가할 수도있습니다.
  4. 플레이어를 찾아 Lead Form드롭다운 목록을 사용하여 표시할 리드 양식을 선택합니다.
  5. (선택 사항) Audience를 사용하여 비디오 참여 데이터를 추적하는 경우 플레이어는 다음을 사용하여 데이터 연결로 구성되어야 합니다. 연결 대상드롭 다운 목록.

리드 양식을 사용하도록 환경 구성

리드 양식이 생성되어 Audience 지원 플레이어에 할당되면 포털 경험에서 사용할 수 있습니다. 리드 양식을 사용하도록 환경을 구성하려면 다음 단계를 따르세요.

  1. 갤러리 모듈을 엽니 다.
  2. 리드 양식을 추가 할 환경을 편집합니다.
  3. 딸깍 하는 소리비디오 및 재생> 플레이어 및 리드 양식왼쪽 탐색에서
  4. 표시 할 리드 양식으로 구성된 청중 지원 플레이어를 선택합니다.
  5. 플레이어를 선택하면리드 양식및연결정보가 표시됩니다. 이것이 올바른지 확인하십시오.
    참고: 랜딩 페이지 환경이 페이지 본문에 리드 양식을 표시하도록 구성된 경우 리드 양식 타이밍 및 표시 설정은 무시됩니다.
  6. 저장을 클릭합니다 .

확인경험을 다시 게시변경 후.

리드 데이터 내보내기

갤러리 리드 양식에서 수집 된 모든 리드 데이터는 대상 모듈을 사용하여 내보내집니다. 갤러리 리드 데이터를내보내는 방법에대한 자세한 내용은 갤러리 리드 데이터 내보내기를 참조하십시오.
