포털 경험에 Video Cloud 비디오 추가

{{ page.description }}

갤러리 컬렉션 편집기를 사용하여 포털 경험에 표시 할 비디오를 선택할 수 있습니다. Portal Experiences를 사용하면 비디오가 컬렉션으로 구성됩니다. 컬렉션에는 비디오 클라우드 계정 또는 YouTube 채널의 동영상이 포함될 수있습니다. 이 항목은 Video Cloud 비디오를 컬렉션에 추가하는 데 중점을 둡니다. 컬렉션에 YouTube 동영상을 추가하는 방법에 대한 자세한 내용은 포털 경험에 YouTube 동영상추가를 참조하십시오 .

컬렉션 만들기

포털 경험의 비디오는 컬렉션으로 구성됩니다. 컬렉션을 만들려면 다음 단계를 따르세요.

  1. 딸깍 하는 소리비디오왼쪽 탐색에서 컬렉션 관리 페이지가 열립니다. 페이지 오른쪽에는 선택한 템플릿 유형에 대한 홈 페이지 비디오 레이아웃의 미리보기가 표시됩니다.
  2. 컬렉션의 이름을 입력하고 Enter 키를 누릅니다 . 시청자에게 이름이 표시되므로 의미있는 이름을 사용하세요. 수정() 아이콘을 클릭하여 컬렉션 이름을 편집합니다.

컬렉션 편집

  1. 컬렉션을 편집하려면 컬렉션 이름을 클릭합니다.
  2. 컬렉션 편집 페이지가 열립니다. 컬렉션 편집 페이지는 컬렉션에 비디오를 추가하고 추가 컬렉션 속성을 구성하는 데 사용됩니다. 컬렉션변경드롭다운에는 생성된 모든 컬렉션이 나열됩니다. 편집 할 컬렉션을 선택합니다. 오른쪽의 미리보기는 홈 페이지에서 컬렉션이 나타나는 위치를 보여줍니다.
  3. 컬렉션 소스는 컬렉션에 대한 비디오 콘텐츠의 소스를지정합니다.
    • 비디오 클라우드 -Video Cloud 비디오 콘텐츠가 사용됩니다.
    • 유튜브 -YouTube 동영상 콘텐츠가 사용됩니다. 자세한 내용은 포털 경험에 YouTube 동영상추가를 참조하십시오 .
  4. 비디오 클라우드 컬렉션 소스를 사용하면재생 목록에서 동영상을만들거나사용자 지정과제를 사용할 수 있습니다.
    • 재생 목록 -컬렉션의 비디오는 Video Cloud 재생 목록을 기반으로합니다.
    • 커스텀 -컬렉션의 비디오는 비디오 태그, 사용자 지정 검색 또는 사용자 지정 메타 데이터 필드를 기반으로합니다.
참고: 에 대해 입력된 값태그 포함 및 태그 제외일부로사이트 기능> 사이트 검색설정은 생성된 모든 컬렉션에도 적용됩니다. 이렇게 하면 컬렉션을 만들 때 표시되는 동영상이 제한될 수 있습니다.
참고: 최근에 비디오 메타데이터 또는 재생 목록을 변경한 경우 이러한 변경 사항은 컬렉션 편집기 및 게시된 경험에 표시되는 데 최대 6분이 소요될 수 있습니다.

재생 목록 비디오 할당 사용

재생 목록비디오 할당을 사용하도록 선택하면 컬렉션의 동영상은 Video Cloud 재생 목록을 기반으로 합니다.

  1. 재생 목록옵션을 선택합니다. Video Cloud 계정의 재생 목록 목록이 나타납니다.
  2. 탐색 컨트롤() 또는 검색 기능을 사용하여 원하는 재생 목록을 찾습니다. 재생 목록을 선택하면 해당 재생 목록의 비디오 목록이 표시됩니다.
    참고: 컬렉션의 비디오 순서는 재생 목록 설정에 따라 결정됩니다.
  3. (선택 사항) 선택한 템플릿에 따라 추가 컬렉션 세부 정보를 구성 할 수 있습니다. 자세한 내용은 이 항목의 컬렉션 정보추가섹션을 참조하십시오.
  4. 저장을클릭하여 변경 사항을 저장합니다.

사용자 지정 비디오 할당 사용

사용자 지정비디오 할당을 사용하도록 선택하면 컬렉션의 동영상은 비디오 태그, 사용자 지정 검색 또는 사용자 지정 메타데이터 필드를 기반으로 합니다.

  1. 사용자 지정옵션을 선택합니다.
  2. 검색 매개 변수는태그 , 검색을 기반으로비디오포함또는제외로 설정할 수 있습니다.또는사용자 정의 필드 .
    참고: 사용할 때사용자 정의 필드옵션, 당신은 입력해야내부의사용자 정의 필드의 이름, 아니다 NS 표시하다이름.
  3. 원하는 비디오를 반환하려면 적절한 검색 매개 변수를 구성하십시오. 다음 예에서는 비디오 클라우드 필수 태그가 지정된비디오가반환됩니다.
  4. (선택 사항) 추가 아이콘() 추가 매개변수를 포함하거나 제외하려면 해당 섹션에 입력합니다.
  5. 비디오 정렬 순서 설정:
    • 가장 최근 -활성화 된시기를 기준으로 정렬 된 동영상, 가장 최근 동영상이 먼저 나열 됨
    • 오래된 것 먼저 -활성화 된시기를 기준으로 정렬 된 동영상, 가장 오래된 동영상이 먼저 나열 됨
    • 가장 많이 본 -대부분의 모든 시간보기
    • AZ -오름차순으로 정렬
    • ZA -내림차순으로 정렬
  6. (선택 사항) 선택한 템플릿에 따라 추가 컬렉션 세부 정보를 구성 할 수 있습니다. 자세한 내용은 이 항목의 컬렉션 정보추가섹션을 참조하십시오.
  7. 저장을클릭하여 변경 사항을 저장합니다.

컬렉션 정보 추가

선택한 템플릿에 따라 컬렉션 세부 정보를 구성 할 수 있습니다. 컬렉션 세부 정보를 추가하려면컬렉션 정보섹션을 클릭합니다. 컬렉션 정보 섹션은 다음을 추가하는 기능을 제공합니다. 컬렉션 이미지그리고컬렉션 설명 . 일부 소셜 네트워크는 경험에 대한 링크가 소셜 미디어 사이트에 게시 될 때 이미지와 설명을 사용합니다.

컬렉션이미지는 갤러리 사이트에서 컬렉션을 나타내는 데 사용됩니다. 컬렉션 이미지가 지정되지 않은 경우 갤러리는 컬렉션의 첫 번째 비디오에서 비디오 썸네일을 사용하여 하나를 할당합니다. 컬렉션 이미지를 추가하려면찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다. 이미지를 지정된 영역으로 끌어다 놓을 수도 있습니다.

컬렉션 설명은 컬렉션을 설명하는 데사용됩니다.
