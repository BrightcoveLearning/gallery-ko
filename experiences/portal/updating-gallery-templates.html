갤러리 템플릿 업데이트

{{ page.description }}

모든 갤러리 경험은 템플릿을 기반으로합니다. 새 환경을 만들 때 첫 번째 단계는 사용할 템플릿을 선택하는 것입니다. 템플릿은 경험의 전체적인 레이아웃과 스타일을 제어합니다. 시간이 지남에 따라 Brightcove에서 템플릿에 새로운 기능과 개선 사항을 적용 할 수 있습니다. Brightcove는 주요 템플릿 업데이트를 기존 환경으로 푸시하지 않습니다. 게시자는 새 템플릿을 사용하도록 환경을 업그레이드 한 다음 게시하기 전에 테스트해야합니다.

참고: 이 항목은 포털, 랜딩 페이지 및 이벤트 경험에만 적용됩니다.

템플릿 업데이트를 사용할 수있는시기 결정

템플릿 업데이트가 필요한 경험이있는 경우 갤러리 모듈이 열릴 때 페이지 상단에 배너가 표시됩니다. 템플릿 업데이트가 필요한 경험만 표시하려면링크를 클릭합니다.

페이지 상단의 필터를 사용하여 템플릿 업데이트가 필요한 경험을 표시 할 수도 있습니다.

템플릿 업데이트가 필요한 모든 경험에는 업데이트필요의 템플릿 상태가 표시됩니다 .

참고: 동일한 템플릿의 다른 버전을 사용하여 경험을 가질 수 있습니다. 예를 들어, 선택 윤곽 템플릿에 대한 업데이트를 사용할 수 있는 경우 업데이트필요상태는 템플릿 업데이트가 필요한 선택 윤곽 사이트에만 나타납니다. 업데이트가 필요하지 않은 최신 버전의 선택 윤곽 템플릿을 사용하여 빌드된 다른 선택 윤곽 사이트가 있을 수 있습니다.

템플릿 업데이트에 대한 자세한 내용을 보려면 페이지 오른쪽에 있는 세부 정보보기링크를 클릭하십시오.

링크를 클릭하면 템플릿 업데이트 세부 정보가있는 대화 상자가 표시됩니다.

딸깍 하는 소리취소대화 상자를 닫거나업데이트 시작템플릿 업데이트를 시작합니다.

경험에 템플릿 업데이트 적용

템플릿을 업데이트하려면 다음 단계를 따르십시오.

  1. 세부 정보보기링크를 클릭하여 템플릿에 대한 변경 사항을 검토합니다.
    참고: 당신은 볼 수 있습니다템플릿 상태 NS 이력서 업데이트템플릿 업데이트가 적용되었고 경험이 게시되지 않은 경우.
  2. 업데이트시작을 클릭합니다 . 템플릿 업데이트가 경험에 적용되고 갤러리 사이트 편집기가 열립니다.
    참고: 템플릿 업데이트가적용되고 경험이 게시되지 않은 경우 템플릿 업데이트다시 시작버튼이 표시될 수 있습니다.
  3. 사이트미리 보기링크를 클릭합니다. 새 템플릿을 사용하는 미리보기가 새 브라우저 탭에서 열립니다. 사이트를 검토하여 플레이어 사용자 정의, 사용자 정의 CSS / JavaScript, 사용자 정의 머리글 / 바닥 글 및 기타 사용자 정의가 올바르게 표시되는지 확인하십시오.
  4. (선택 과목)클릭업데이트를 취소링크 또는업데이트 삭제버튼템플릿 업데이트 삭제 .
  5. 업데이트된 템플릿으로 사이트를게시하려면 게시를 클릭합니다 .

업데이트 된 템플릿을 사용하여 사이트를 게시하면 이전 템플릿에 액세스 할 수 없습니다.

템플릿 업데이트 취소

업데이트취소링크 또는 업데이트취소단추를 클릭하여 템플릿 업데이트를 취소합니다.

업데이트를 취소하라는 메시지가 표시됩니다.

업데이트폐기를 클릭하여 사이트를 이전 버전의 템플릿으로 되돌립니다. Brightcove는 최신 버전의 템플릿으로 업데이트 할 것을 권장합니다. 업데이트를 폐기하면 어느 시점에서 이전 버전이나 템플릿이 Brightcove에서 지원되지 않습니다.
