카탈로그 템플릿 설정 사용자 화

{{ page.description }}

카탈로그 템플릿을 사용하여 포털 경험을 만들면 몇 가지 템플릿 별 설정을 구성 할 수 있습니다. 카탈로그 템플릿에 대한 설정을 구성하려면 환경을 편집하고외모와 행동> 템플릿 설정왼쪽 탐색에서

다음 설정을 구성 할 수 있습니다.

  • 기본 탐색
  • 기능 캐 러셀
  • 캐 러셀 페이징 버튼
  • 비디오 썸네일
  • 썸네일 재생 아이콘
  • 비디오 설명

기본 탐색

기본 탐색설정은 사이트 탐색의 레이아웃을 제어합니다.

  • 왼쪽 탐색 -컬렉션이 3 개 이상일 경우 왼쪽 탐색 창에 컬렉션 이름이 표시됩니다. 이 옵션을 사용하면홈자리 표시자 텍스트 대상체가 레이아웃에 추가되어 뷰어가 홈 페이지로 돌아갈 수 있습니다. 이 텍스트는 필요한 경우 사용자 정의 할 수 있습니다.
  • 접힌 메뉴 탐색 -접을 수있는 햄버거 메뉴가 표시됩니다.

기능 캐 러셀

기능 회전 목마설정은 현재 컬렉션의 비디오가 표시되는 방식을 제어합니다.

  • 썸네일 그리드 -동영상 썸네일이 플레이어 아래에 표시됩니다.
  • 동영상만 - 하나의 동영상만 나타납니다. < > 컬렉션의 다른 비디오로 이동할 수 있는아이콘이 플레이어에 나타납니다.

캐 러셀 페이징 버튼

회전 목마 페이징 버튼설정은 회전 목마 페이징 버튼이 표시되는 시기를 결정합니다.

  • 항상 페이징 버튼 표시 -페이징 버튼은 항상 표시됩니다.
  • 호버시에만 페이징 버튼 표시 -페이징 버튼은 마우스가 플레이어 또는 비디오 정보 위에있을 때만 표시됩니다.

비디오 썸네일

[ 비디오 축소판]설정은 비디오 축소판 이미지의 크기를 제어합니다.

  • 작은
  • 큰

썸네일 재생 아이콘

썸네일 재생 아이콘설정은 플레이어에 재생 버튼이 표시되는지 여부를 결정합니다.

  • 재생 아이콘 표시 안 함 -재생 아이콘이 표시되지 않습니다.
  • 기본적으로 재생 아이콘 표시 -재생 아이콘이 표시됩니다.

비디오 설명

비디오 설명설정은 표시되는 비디오 설명의 길이를 제어합니다.

  • 설명 자르기 -잘린 비디오 설명을 표시합니다.
  • 전체 설명보기 -전체 비디오 설명을 표시합니다
