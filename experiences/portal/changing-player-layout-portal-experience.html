포털 경험에서 플레이어 레이아웃 변경

{{ page.description }}

경험을 만들 때 선택한 포털 템플릿에 따라 일부 페이지 템플릿은 페이지의 플레이어 레이아웃을 변경하는 기능을 제공 할 수 있습니다.

참고: 모든 페이지 템플릿이 플레이어 레이아웃을 커스터마이징할 수 있는 기능을 제공하는 것은 아닙니다.

플레이어 레이아웃을변경할 수 있다면 페이지 이름 옆에 레이아웃 변경드롭다운이 나타납니다.

플레이어 레이아웃 변경

플레이어 레이아웃을 변경하려면 레이아웃변경링크를 클릭하고 새 플레이어 레이아웃을 선택합니다.

다음 레이아웃을 사용할 수 있습니다.

  • 표준
  • 큰
  • 사이드 배너
  • 두 측면 영역
  • 왼쪽 열

사용 가능한 레이아웃은 선택한 템플릿과 선택한 페이지에 따라 다릅니다. 현재 사용중인 레이아웃은 확인 표시로 표시됩니다. 사이트가 새 레이아웃으로 업데이트됩니다.
