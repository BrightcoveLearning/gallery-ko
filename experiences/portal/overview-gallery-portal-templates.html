페이지 내용

    갤러리 포털 템플릿 개요

    이 항목에서는 사용 가능한 모든 포털 템플릿에 대한 참조를 제공합니다.
    참고: 페이지 내 환경 템플릿에 대한 자세한 내용은갤러리 페이지 내 환경 템플릿 개요를참조하십시오.

    새 갤러리 환경을 만들 때 경험의 전체적인 모양과 느낌을 제어하는 템플릿을 선택합니다. 만들려는 환경 유형을 선택하면 템플릿 목록이 필터링됩니다.

    포털을클릭하면 사이트 요구 사항에 따라 포털 템플릿을 필터링하는 데 템플릿 선택기를 사용할 수 있습니다.

    참고: 템플릿을 선택한 후에는 템플릿을 변경할 수 없습니다.

    포털 , 랜딩 페이지및이벤트환경을 만들 때 사용할 수 있는 템플릿은 다음과 같습니다.

    • 모자이크
    • 클래식
    • 선택 윤곽
    • 게시자
    • 디스커버리
    • 라이브 이벤트 포털
    • 랜딩 페이지
    • 쇼케이스
    • 크로니클
    • 카탈로그

    페이지 내 환경 템플릿에 대한 자세한 내용은갤러리 페이지 내 환경 템플릿 개요를참조하십시오.

    참고: 브라이트코브 컨설팅을 사용하여 사용자 정의 템플릿을 개발할 수도 있습니다. 자세한 내용은 계정 관리자에게 문의하십시오.

    포털 템플릿은 사용자 정의할 수 있는 페이지 템플릿 집합을 제공합니다. 페이지 템플릿은 갤러리 사이트 편집기를 연 다음 왼쪽 탐색 메뉴에서PAGE를클릭하여 볼 수 있습니다.

    페이지 이름을 클릭하여 편집기에서 페이지 레이아웃을 엽니다. 위의 예에서, 홈 , 재생및카테고리페이지를 사용할 수 있습니다.

    참고: 일부 페이지는 플레이어 레이아웃을 변경하는 기능도 제공합니다. 자세한 내용은포털 환경에서 플레이어 레이아웃 변경을참조하십시오.

    모자이크

    모자이크 템플릿은 인-페이지 비디오 플레이어가 있는 연속 이미지의 연속 배열을 제공합니다. 관련 비디오는 아래에 나와 있습니다. 모자이크 템플릿을 사용하면 사이트가 시각적으로 돋보입니다. 모자이크 템플릿은 매우 시각적인 사이트를 제공하며 그래픽 기반 환경을 원할 경우에 권장됩니다. 이 템플릿은 큰 범주 기반이 없는 소규모 콘텐츠 라이브러리에 가장 적합합니다. 페이지 수준 광고 및 클릭유도문안도 지원됩니다.

    모자이크 템플릿은 두 개의 페이지 레이아웃 템플릿을 제공합니다.

    • 홈
    • 범주

    모자이크 템플릿을 사용하는 샘플 사이트를 보려면 여기를 클릭하십시오.

    클래식

    클래식 템플릿은 콘텐츠 라이브러리를 통해 선명한 탐색을 제공하는 편집 내용과 비디오를 원활하게 결합합니다. 클래식 템플릿은 편집 내용과 함께 보여주고 강조하려는 짧은 구체적 비디오 콘텐츠가 있는 경우에 권장됩니다. 이 템플릿은 대규모 콘텐츠 라이브러리에 가장 적합한 콘텐츠 계층 구조를 지원합니다. 클래식 템플릿을 사용할 때는 다음에 유의하십시오.

    1. 첫 번째 컬렉션은 회전식 슬라이드에 대한 것으로, 페이지 상단의 스크롤 영역입니다.
    2. 두 번째 컬렉션은 홈페이지의 섹션 1에 대한 것입니다.
    3. 세 번째 컬렉션은 홈페이지의 섹션 2에 대한 것입니다.
    4. 나머지 컬렉션은 왼쪽 탐색 공간을 채우는 데 사용됩니다.

    클래식 템플릿은 세 개의 페이지 레이아웃 템플릿을 제공합니다.

    • 홈
    • 범주
    • 비디오 세부 정보

    페이지 수준 광고 및 클릭유도문안도 지원됩니다. 클래식 템플릿을 사용하는 샘플 사이트를 보려면 여기를 클릭하십시오.

    움직이는 텍스트

    마키 템플릿은 아래의 주요 비디오 촬영 센터 작업실 및 전체 비디오 라이브러리와 함께 단일 페이지로 된 일관된 경험을 제공합니다. 선보일 주 비디오가 하나 있다면 마키 템플릿을 사용하는 것이 좋습니다.

    마키 템플릿은 세 개의 페이지 레이아웃 템플릿을 제공합니다.

    • 홈
    • 재생
    • 범주

    마키 템플릿을 사용하는 샘플 사이트를 보려면 여기를 클릭하십시오. 페이지 수준 광고 및 클릭유도문안도 지원됩니다.

    선택 윤곽 템플릿은 또한 페이지 이름 옆에 있는 레이아웃 변경 링크를 클릭하여 모든 페이지의 플레이어레이아웃을 변경할수 있는 기능을 제공합니다. 현재 레이아웃은 확인 표시와 함께 표시됩니다.

    이용할 수 있는 레이아웃은 다음과 같습니다.

    • 표준
    • 크게
    • 사이드 배너
    • 양 사이드 영역

    사이드 배너 또는 양 사이드 영역 레이아웃을 사용할 경우 템플릿에 추가 메시지 공간이 추가됩니다. 이 영역의 크기 제한은 다음과 같습니다.

    • 측면 배너 레이아웃 - 이미지가 50px의 높이로 잠긴 다음 가로 세로 비율에 맞게 너비로 조정됩니다.
    • 두 측면 영역 레이아웃 - 이미지는 300 픽셀의 높이로 잠긴 다음 가로 세로 비율로 너비를 조정합니다

    플레이어 레이아웃 변경에 대한 자세한 내용은포털 환경에서 플레이어 레이아웃 변경을참조하십시오.

    퍼블리셔

    퍼블리셔 템플릿에는 비디오 및 이미지 기반 홈 페이지가 있습니다. 퍼블리셔는 쉽게 검색할 수 있는 비디오, 비디오 설명 위에 마우스 커서 놓기, 콘텐츠의 범주 및 하위 범주를 지원합니다. 페이지 수준 광고 및 클릭유도문안도 지원됩니다.

    퍼블리셔 템플릿은 세 개의 페이지 레이아웃 템플릿을 제공합니다.

    • 홈
    • 범주
    • 비디오 세부 정보

    페이지 수준 광고 및 클릭유도문안도 지원됩니다. 퍼블리셔 템플릿을 사용하는 샘플 사이트를 보려면 여기를 클릭하십시오.

    검색

    검색 템플릿은 많은 양의 콘텐츠 라이브러리에 적합하며 뷰어를 통해 콘텐츠를 표 형식으로 탐색할 수 있습니다. 이것은 편집 내용에 대해 공간이 제한된 그래픽 기반 템플릿입니다.

    검색 템플릿은 세 개의 페이지 레이아웃 템플릿을 제공합니다.

    • 홈
    • 범주
    • 동영상 세부 정보
