새로운 갤러리 환경을 만들 때 환경의 전체적인 모양과 느낌을 제어하는 템플릿을 선택하게됩니다. 만들려는 경험 유형을 선택하면 템플릿 목록이 필터링됩니다.

포털을 클릭하면 템플릿 선택기를 사용하여 사이트 요구 사항에 따라 포털 템플릿을 필터링할 수 있습니다.

참고: Brightcove Consulting의 도움을 받아 맞춤형 템플릿을 개발할 수도 있습니다. 자세한 내용은 계정 관리자에게 문의하십시오.

포털 템플릿은 사용자 정의 할 수있는 페이지 템플릿 집합을 제공합니다. 갤러리 사이트 편집기를 연 다음 왼쪽 탐색 메뉴에서 PAGES 를 클릭하여페이지템플릿을 볼 수 있습니다.

페이지 이름을 클릭하여 편집기에서 페이지 레이아웃을 엽니 다. 위의 예에서는홈 , 재생및범주페이지를 사용할 수 있습니다.

참고: 일부 페이지는 플레이어 레이아웃을 변경할 수있는 기능도 제공합니다. 자세한 내용은 포털 경험에서 플레이어 레이아웃변경을 참조하십시오 .

Mosaic 템플릿은 인 페이지 비디오 플레이어로 전체 이미지의 연속 배열을 제공합니다. 관련 동영상이 아래에 표시됩니다. Mosaic 템플릿을 사용하면 비주얼이 사이트를 지배합니다. Mosaic 템플릿은 매우 시각적 인 사이트를 제공하며 그래픽 중심의보기 환경을 원하는 경우 권장됩니다. 이 템플릿은 큰 카테고리 기반이없는 작은 콘텐츠 라이브러리에 가장 적합합니다. 페이지 수준 광고 및 클릭 유도 문안도 지원됩니다.

Mosaic 템플릿은 두 가지 페이지 레이아웃 템플릿을 제공합니다.

  • 범주

권위 있는

클래식 템플릿은 편집 콘텐츠와 비디오를 매끄럽게 결합하여 콘텐츠 라이브러리를 통해 선명한 탐색을 제공합니다. 클래식 템플릿은 편집 콘텐츠로 보여주고 강조하려는 특정 비디오 콘텐츠가 많을 때 권장됩니다. 이 템플릿은 대용량 콘텐츠 라이브러리에 가장 적합한 콘텐츠 계층 구조를 지원합니다. Classic 템플릿을 사용할 때는 다음과 같이 하십시오.

  1. 첫 번째 컬렉션은 캐 러셀 용이며 페이지 상단의 스크롤 영역입니다.
  2. 두 번째 컬렉션은 홈페이지의섹션 1입니다.
  3. 세 번째 컬렉션은 홈페이지의섹션 2입니다.
  4. 나머지 컬렉션은 왼쪽 탐색을 채우는 데 사용됩니다.

큰 천막

Marquee 템플릿은 중앙 무대를 차지하는 주요 비디오와 바로 아래에 전체 비디오 라이브러리를 통해 일관된 단일 페이지 경험을 제공합니다. 표시하려는 기본 동영상이 하나있는 경우 Marquee 템플릿을 사용하는 것이 좋습니다.

Marquee 템플릿은 세 가지 페이지 레이아웃 템플릿을 제공합니다.

  • 집
  • 재생
  • 범주

딸깍 하는 소리여기 Marquee 템플릿을 사용하여 샘플 사이트를 보려면 페이지 수준 광고 및 클릭 유도 문안도 지원됩니다.

선택 윤곽 템플릿은 클릭하여 모든 페이지의 플레이어 레이아웃을 변경할 수 있는 기능도 제공합니다. 레이아웃 변경페이지 이름 옆에 있는 링크. 현재 레이아웃은 확인 표시로 표시됩니다.

다음 레이아웃을 사용할 수 있습니다.

  • 표준
  • 큰
  • 사이드 배너
  • 두 측면 영역

측면 배너 또는 양면 영역 레이아웃을 사용하는 경우 추가 메시징 공간이 템플릿에 추가됩니다. 이러한 영역의 치수 제한은 다음과 같습니다.

  • 사이드 배너 레이아웃 -이미지가 50px 높이로 고정 된 다음 가로 세로 비율에 맞게 조정됩니다.
  • 양면 영역 레이아웃 -이미지가 300px 높이로 고정 된 다음 가로 세로 비율에 맞게 너비가 조정됩니다.

플레이어 레이아웃 변경에 대한 자세한 내용은 포털 경험에서 플레이어 레이아웃변경을 참조하십시오 .

라이브 이벤트 포털

라이브 이벤트 포털 템플릿은 청중에게 라이브 콘텐츠를 제공하도록 설계되었습니다. 라이브 이벤트 포털 템플릿은 이벤트 전, 도중 및 사후 상태를 제공하고 페이지 수준 광고 및 클릭 유도 문안을 지원합니다. 타사 통합을 사용하여 채팅 및 댓글 기능을 추가 할 수 있습니다.

라이브 이벤트 포털 템플릿은 다섯 페이지 레이아웃 템플릿을 제공합니다.

  • 홈페이지: 이벤트 전
  • 홈페이지: 라이브
  • 홈페이지: 이벤트 후
  • 비디오 아카이브
  • 비디오 세부 정보

딸깍 하는 소리여기라이브 이벤트 포털 템플릿을 사용하여 샘플 사이트를 보려면 라이브 이벤트 포털 템플릿 사용에 대한 자세한 내용은 다음을 참조하십시오. 라이브 이벤트 포털 만들기 경험 .
