포털 경험을 위한 비디오 설정 사용자 정의

{{ page.description }}

비디오 설정을 사용자 정의하려면 포털 경험을 편집하고비디오 및 재생> 동영상왼쪽 탐색에서 비디오 설정을 사용하면 포털 경험에 대해 다음을 구성 할 수 있습니다.

  • 정렬
  • 비디오 다운로드
  • 관련 링크
  • 장황한 설명
  • 비디오 보안

비디오 설정을 변경 한 후 경험을 다시 게시해야합니다.

비디오 정렬 활성화

비디오 정렬 활성화 (재생 목록 기반 컬렉션에는 사용할 수 없음) -선택하면 시청자가 컬렉션의 동영상을 정렬 할 수 있습니다. 재생 목록을 사용하여 만든 컬렉션에는 정렬을 사용할 수 없습니다. 이 설정은 다음 템플릿에서만 사용할 수 있습니다.

  • 목록
  • 발견
  • 라이브 이벤트 포털
  • 큰 천막
  • 모자이크
  • 발행자
  • 권위 있는

비디오 다운로드 구성

비디오 다운로드설정은 시청자가 동영상을 다운로드할 수 있는 기능을 제공합니다. 다운로드 링크를 사용하여 최고 품질의 변환을 다운로드 할 수 있습니다.

다음 옵션을 사용할 수 있습니다.

  • 비디오 다운로드 허용 안함 -비디오 다운로드는 허용되지 않습니다.
  • 각 비디오에 대한 다운로드 링크 제공 -각 동영상에 대한 다운로드 링크가 제공됩니다.
  • 사용자 정의 필드 값을 기반으로 다운로드 링크 제공 -비디오 다운로드는 비디오에 할당 된 사용자 정의 필드 값을 기반으로 제공됩니다. 이 옵션을 선택할 때 사용자 정의 필드를 정의한 다음 해당 필드를 다음으로 설정합니다. 진실다운로드할 수 있는 모든 비디오의 경우. 각 비디오에 대한 다운로드 링크를 구성하는 전체 단계는 포털 환경에서 비디오다운로드를 참조하십시오 .
참고: 비디오에 사용 가능한 MP4 변환이 하나 이상 있어야 합니다.

관련 링크 표시

각 비디오에 대한 관련 링크 표시 -체크하면 동영상 이름 옆에 동영상 관련 링크가 표시됩니다. 비디오에 대한 관련 링크를 구성하는 방법에 대한 자세한 내용은 미디어 모듈을 사용하여 비디오 속성편집을 참조하십시오 .

자세한 설명 표시

비디오 세부 사항 페이지에 자세한 설명 (가능한 경우) 사용 -체크시 동영상 상세 페이지에 동영상 상세 설명이 표시됩니다.

비디오 보안 구성

사이트의 비디오를 사용자 정의 검색 또는 스마트 재생 목록을 통해 선택한 비디오로 제한 - 체크하면 언제검색이 활성화되었습니다경험을 위해 컬렉션의 비디오만 검색됩니다(검색은 스마트 재생 목록을 사용하거나 사용자 지정 검색을 사용하여 컬렉션에 추가된 비디오만 반환합니다). 또한 시청자가 재생 목록에서 제거 된 동영상에 액세스하려고하면 경험 홈 페이지로 다시 리디렉션됩니다.

이 옵션을 선택하지 않으면 경험 검색이 활성화되면 Video Cloud 계정의 모든 비디오가 검색됩니다 (태그 포함 / 제외 옵션이 사용되지 않는 경우).

참고: 이 옵션이 나타나지 않으면 브라이트코브 계정 관리자에게 문의하여 활성화하십시오.
