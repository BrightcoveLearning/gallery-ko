갤러리 포털 경험 인덱싱

{{ page.description }}

포털 경험이 게시되면 검색 엔진에서 사이트를 홍보하고 색인화하는 데 도움이되는 몇 가지 작업을 수행 할 수 있습니다.

  • 이미 색인이 생성 된 기존 웹 속성에서 새 사이트로 연결합니다. 예를 들어, 새 사이트에 대한 링크가 포함 된 블로그 게시물을 작성하거나 기본 웹 사이트의 사이트에 대한 링크를 추가 할 수 있습니다.
  • 그것을 공유하고 소셜 네트워크에서 홍보하십시오.
  • 고객이 사이트를 방문하도록 장려하고 자신이 사이트를 사회적으로 공유 할 수 있기를 바랍니다.
  • URL 또는 사이트 맵을 검색 엔진에 제출하여 사이트 색인을 생성합니다.

이 항목에서는 검색 엔진에 제출할 수있는 포털 경험에 대한 사이트 맵 생성에 중점을 둡니다.

포털 경험을위한 사이트 맵 생성

사이트 맵은 사이트의 페이지에 대해 검색 엔진에 알리는 방법입니다. 갤러리는 웹 사이트의 페이지를 나열하는 XML 사이트 맵을 생성합니다. 그런 다음 검색 엔진은 XML을 구문 분석하여 사이트에있는 콘텐츠를 이해합니다.

포털 경험의 사이트맵에 액세스하려면 경험을 편집하고사이트 구성> 검색 엔진 최적화왼쪽 탐색에서 사이트 맵의 URL은 페이지 하단에 표시됩니다.

참고: 경험이 게시될 때까지 사이트 맵을 사용할 수 없습니다.

Sitemap은 24 시간 동안 캐시됩니다. 24 시간 후 캐시에서 제거되고 다시 액세스 할 때 다시 생성됩니다. 비디오가 포털 경험에 추가되면 사이트 맵이 다시 생성되거나 (24 시간 후) 사이트가 다시 게시 될 때까지 사이트 맵에 포함되지 않습니다.

사이트 맵 URL을 복사하여 Google / Bing 웹 마스터 도구에 붙여 넣을 수 있습니다. 검색 엔진에 사이트 맵을 제출하는 방법에 대한 자세한 내용은 다음 링크를 참조하십시오.

  • Google 웹 마스터 도구에 사이트 맵 제출
  • Bing에 사이트 맵 제출

Google에 사이트 맵을 제출할 때 프로세스의 한 단계는 사이트 소유권을 확인하는 것입니다. Brightcove는 HTML 태그 옵션 사용을 권장합니다. Google에서 생성한 메타 태그는검색 엔진 최적화설정의 일부로 경험에 추가할 수 있습니다.

또한 Gallery는 검색 엔진이 사이트맵을 사용하도록 지시하는robots.txt파일을 만듭니다.
