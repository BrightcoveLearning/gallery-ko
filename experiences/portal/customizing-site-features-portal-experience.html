포털 경험의 사이트 기능 사용자 정의

{{ page.description }}

포털 경험에 대한 사이트 기능을 구성하려면 경험을 편집하고사이트 기능왼쪽 탐색에서 다음 설정을 사용할 수 있습니다.

  • 사이트 검색 -사이트 검색 활성화
  • 사회적인 -비디오 공유 구성 및 연결 유지 옵션
  • 코멘트 -댓글 사용

사이트 검색 구성

사이트 검색설정을 사용하면 사이트 검색을 활성화할 수 있습니다. 사이트 검색 설정에 액세스하려면사이트 기능> 사이트 검색왼쪽 탐색에서 사이트 검색활성화를 선택하면 시청자가 동영상 콘텐츠에 대해 검색을 수행할 수 있도록 사이트에 검색 필드가 나타납니다. 동영상 이름, 짧고 긴 설명 및 태그가 검색됩니다. 기본적으로 검색이 활성화되면 Video Cloud 계정의 모든 비디오가 검색됩니다. 검색은 갤러리 환경에 나타나는 비디오에만 국한되지 않습니다.

검색 할 때 반환되는 비디오를 제한하려면 다음과 같은 옵션이 있습니다.

  • 태그 포함 -검색은 검색 창에 입력 한 검색어와 일치하고이 태그로 태그가 지정된 동영상 만 반환합니다 (단일 태그 만 지원됨).
  • 태그 제외 -이 태그가있는 동영상은 검색 결과의 일부로 반환되지 않습니다 (단일 태그 만 지원됨).
  • 비디오 보안설정 활성화 - 비디오 설정의 일부로 비디오 보안을 활성화할 수 있습니다. 활성화되면 컬렉션의 비디오 만 검색됩니다 (검색은 스마트 재생 목록 또는 사용자 지정 검색을 사용하여 컬렉션에 추가 된 비디오 만 반환 함). 자세한 내용은 포털 경험에 대한 비디오 설정사용자 지정을 참조하십시오 .
참고: 컬렉션을 만들 때 여러 비디오 소스가 사용되는 경우에는 검색이 지원되지 않습니다.
참고: 태그포함및제외태그에 입력한 값은 사이트에 추가된 모든 컬렉션에도 적용됩니다.

소셜 설정 구성

소셜설정을 사용하면 시청자가 인기 소셜 미디어 사이트에 콘텐츠를 쉽게 공유할 수 있습니다. 시청자가 소셜 미디어 사이트에 빠르게 액세스 할 수있는 방법을 제공 할 수도 있습니다. 소셜 설정 구성에 대한 자세한 내용은 포털 경험에 대한 소셜 설정사용자 지정을 참조하십시오 .

주석 구성

댓글설정은 Disqus에 커뮤니티 포럼을 사용하여 댓글을 캡처할 수 있는 기능을 제공합니다. 주석 설정을 구성하려면사이트 기능> 코멘트왼쪽 탐색에서

참고: 일부 템플릿에서 주석설정을 사용할 수 있는 것은 아닙니다.

Disqus 닉네임을 입력하면 시청자가 사이트에 댓글을 추가 할 수 있습니다.

Disqus에 대해 자세히 알아보고 사이트의 짧은 이름을 생성하려면 Disqus의웹 사이트를 방문하십시오.
