참고: 사용 가능한 설정은 선택한 사이트 템플릿을 기반으로합니다. 템플릿이 옵션을 지원하지 않으면 표시되지 않습니다.

Google Analytics와 통합

갤러리에는 Google 애널리틱스로 추적 데이터를 보내는 기능이 포함되어 있습니다. Google 애널리틱스에서 추적 할 새 속성을 만들면 Google은 사이트에 추가해야하는 추적 ID와 추적 코드를 제공합니다. 추적 ID는 데이터를 보낼 계정과 속성을 Google 애널리틱스에 알리는 데 사용되며 Google 애널리틱스 필드에 입력해야합니다. 추적 코드는 다음 방법 중 하나로 사이트에 추가 할 수 있습니다.

  • 사용자 지정 머리글/바닥 글 HTML - 사이트에사용자 지정 머리글/바닥글 HTML을 사용하는 경우 머리글/바닥글 HTML의 일부로 추적 코드를 붙여 넣을 수 있습니다.
  • 사용자 지정 JavaScript 파일 지정 - 추적 코드를 도메인에 호스팅된.js 파일에 저장한 다음 사이트의 사용자 지정 JavaScript 속성을 사용하여 추적 코드를 가리킵니다.

사이트 설정에서 추적 ID를 추가하면 기본 "페이지 뷰"분석이 제공됩니다. 갤러리는 동일한 것을 위해 필요한 최소한의 JavaScript를 추가합니다. 맞춤 헤더의 일부 또는 맞춤 자바 스크립트 파일에 추적 코드를 추가하면 더 구체적인 분석을 추가 할 수 있습니다. 예를 들어 클릭 또는 기타 이벤트에 대한 사용자 지정 코드를 추가 할 수 있습니다.

Google 애널리틱스에 대해 자세히 알아보고 사이트의 추적 ID를 만들려면 Google 애널리틱스웹 사이트를 방문하십시오.

Google 태그 관리자와 통합

갤러리에는 Google 태그 관리자의 코드를 포함하는 기능이 있습니다. < 머리>또는 < 본문>섹션에 적절한 코드를 붙여 넣습니다.

Google 태그 관리자에 대해 자세히 알아보려면 Google웹 사이트를 방문하십시오.

Eloqua와 통합

갤러리에는 웹 사이트 방문을 추적 할 수있는 Eloqua 페이지 추적 스크립트가 포함되어 있습니다. 사이트를 Eloqua와 통합하려면 Eloqua 사이트 ID를 입력해야합니다.

Marketo와 통합

갤러리에는 비디오 사이트 방문을 추적 할 수있는 Marketo 페이지 추적 스크립트가 포함되어 있습니다. 사이트를 Marketo와 통합하려면 Marketo 계정 ID를 입력해야합니다.
