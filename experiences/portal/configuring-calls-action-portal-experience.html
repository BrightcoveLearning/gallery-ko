포털 경험에서 클릭 유도문안 구성

{{ page.description }}

포털 경험을 만드는 데 사용 된 템플릿에 따라 일부 템플릿은 클릭 유도 문안 콘텐츠를위한 지정된 영역을 제공합니다. 여기에는 사용자 정의 HTML, 이미지 또는 광고가 포함될 수 있습니다. 템플릿에서 이러한 영역은배너, 비디오 그리드 CTA/광고, 영웅 CTA/광고, 카테고리 CTA/광고또는비디오 CTA/광고로표시되며밝은 회색 배경색입니다. 페이지에 대한 클릭 유도문안을 구성하려면 페이지의 클릭 유도문안 영역을 클릭하거나클릭 유도문안왼쪽 탐색에서 페이지 템플릿을 선택합니다. 클릭 유도 문안 페이지가 열립니다.

클릭 유도 문안 페이지는 선택한 템플릿에 따라 다릅니다. 위의 대화 상자에서는 사이트를 만들 때 Showcase 템플릿이 사용되었습니다. 홈 페이지 템플릿은 다음 영역에서 클릭 유도 문안을 지원합니다.

  • ㅏ -페이지 헤더
  • B - 반복 CTA를 사용하여 비디오 피드의 중간 또는 끝에서
  • 씨 -동영상 재생을 위해 오버레이가 열리면
  • 디 -휴대 기기에서 사이트 탐색 아래
  • E - 모바일 장치에서반복 가능한 CTA를 사용하여 비디오 피드의 중간 또는 끝

클릭 유도 문안을 구성하려면 해당 지역을 클릭 한 다음 해당 지역에 대한 조치 유형을 선택하십시오.

  • 없음
  • 광고
  • 맞춤 HTML
  • 영상

반복 가능한 CTA 구성

클릭 유도 문안을 구성 할 때 지정된 영역 중 일부가 반복 가능한 CTA를 지원했습니다.

반복 가능한 CTA 영역을 구성하려면 해당 영역을 선택한 다음 CTA가 표시되어야하는시기를 선택하십시오.

그런 다음 자리에서 표시할 객체 유형, 광고 , 사용자 정의 HTML또는이미지를 선택합니다 . 스폿을추가하려면 다른 반복 지점 추가를 클릭합니다 .

광고 구성

광고를 구성하려면광고를 선택합니다 . 선택한 영역에 따라 권장 광고 크기가 변경됩니다. 문서 헤더 광고코드와광고 단위 코드에 붙여넣습니다 .

저장을클릭하여 변경 사항을 저장합니다.

단일 비디오를 재생할 수 있는 페이지 (라이브 및 비디오 세부 정보 페이지) 의 경우광고를선택하면 두 가지 추가 사용자 지정 옵션을 사용할 수 있습니다.

  • 사용자 정의 필드 -맞춤 입력란 값에 따라 광고 단위 코드를 표시하도록 선택할 수 있습니다. 예를 들어 맞춤 값이 '축구'인 동영상은 축구 광고를 표시합니다. 사용자 지정 필드 값이 없는 동영상에는그렇지 않으면광고 단위 코드를 사용하여 광고가 표시됩니다.
  • 컴패니언 광고 사용 -광고 서버가 컴패니언 광고를 제공하도록 구성된 경우 동영상 재생시 컴패니언 광고가 실행될 때 클릭 유도 문안 영역에 표시됩니다.
    참고: 사용자 지정 필드나 컴패니언 광고 기준이 모두 충족되지 않을 때 백업 광고를 표시하려면그렇지 않은 경우 , 표시 아래에 광고 단위 코드를 제공해야 합니다. 그렇지 않으면 빈 장소가 표시됩니다.

사용자 정의 HTML 구성

표시할 사용자 정의 HTML을 구성하려면사용자 정의 HTML을 선택합니다 . HTML에 대해 제안 된 크기는 선택한 영역에 따라 변경됩니다. 표시 할 사용자 정의 HTML을 붙여 넣습니다.

저장을클릭하여 변경 사항을 저장합니다.

사용자 정의 HTML을 구성 할 때 구성되는 템플릿 및 지역에 따라 표시되는 CTA를 사용자 정의하는 추가 옵션이있을 수 있습니다. 예를 들어 Showcase 템플릿의 오버레이 CTA를 구성 할 때 다음 옵션이 있습니다.

  • 모든 동영상에 대해 동일한 클릭 유도 문안 표시 -모든 동영상에 동일한 CTA가 사용됩니다.
  • 맞춤 값을 기반으로 클릭 유도 문안 표시 -CTA는 동영상과 관련된 맞춤 메타 데이터에 따라 변경 될 수 있습니다.

여러 구성을 추가 할 수도 있습니다.

이미지 구성

표시할 이미지를 구성하려면 [ 이미지] 를 선택합니다 . 이미지의 권장 크기는 선택한 영역에 따라 변경됩니다. 파일 시스템에서 이미지를 끌어다 놓거나파일을 선택업로드할 이미지를 찾아 선택합니다. 이미지에 하이퍼링크를 추가하려면 이미지연결 대상필드에 URL을 입력합니다.

저장을클릭하여 변경 사항을 저장합니다.
