포털 경험을 위한 검색 엔진 최적화 설정 구성

포털 경험에 대한 검색 엔진 최적화 설정을 구성하려면 경험을 편집하고사이트 구성> 검색 엔진 최적화왼쪽 탐색에서 다음 설정을 사용할 수 있습니다.

  • 사이트 세부 정보 -사이트 이미지, 키워드 및 메타 태그 구성
  • 사이트 맵 -사이트 맵 및 검색 피드 URL에 액세스

사이트 세부 정보 구성

사이트 세부 정보 설정을 구성하여 검색 엔진에서 포털을 인덱싱하는 방법을 제어 할 수 있습니다. 검색 엔진 최적화 설정에 액세스하려면 환경을 편집하고사이트 구성> 검색 엔진 최적화왼쪽 탐색에서

다음 설정을 구성 할 수 있습니다.

  • 검색 엔진 결과 및 사이트가 공유 될 때 사이트 설명 사용 -체크하면 <meta> 설명 태그에 Facebook 및 Twitter 메타 태그를 포함한 사이트 설명이 사용됩니다. 선택하지 않으면 추천 동영상에 대한 설명이 사용됩니다.
  • 사이트 이미지 -포털 사이트의 홈페이지를 페이스 북이나 트위터에서 공유 할 때 표시되는 이미지입니다.
  • 키워드 -모든 사이트 페이지에 추가 될 메타 키워드. “새, 동물, 물고기, 동물 세계”를 추가하면 HTML 페이지에 다음이 추가됩니다.
    <meta name="keywords" content="birds,animals,fish,animal world">
  • 메타 태그 -사이트에 대한 추가 메타 태그를 추가하는 데 사용됩니다. 이러한 태그는 모든 사이트 페이지에 추가됩니다. 예:
    <meta name="myCustomTag" content="myCustomValue">
  • 페이지 메타 데이터에 사용자 정의 비디오 필드 포함 -체크시 사용자 정의 필드를 메타 태그로 추가합니다. 예를 들어 값이 2018인 사용자 지정연도필드는 다음을 추가합니다.
    <meta name="year" content="2018">
  • 페이지 메타데이터에 정식 URL 포함 - 이 옵션을 선택하면 소스에rel=canonical메타 태그를 추가하여 SEO를 개선할 수 있습니다. 이 기능은 Google에 동일한rel=canonical링크로 모든 페이지를 단일 항목으로 처리하도록 지시하는 기능입니다. 예를 들어, mysite.com/비디오/마이패지.html은 Mysite.com/비디오/마이페이지와 동일합니다.
  • URL에서 비디오 제목 제거 -선택하면 동영상 이름이 동영상 상세 페이지에 대한 URL의 일부로 포함되지 않으며 동영상 ID 만 URL에 포함됩니다.

사이트 맵 및 검색 피드보기

사이트 맵 설정은 사이트 맵 및 검색 피드에 대한 URL을 표시합니다. 사이트 맵을 다운로드하여 검색 엔진에 제출할 수 있습니다. 검색 피드는 대부분의 검색 엔진 소프트웨어에서 갤러리 콘텐츠를 검색 결과에 추가하는 방법으로 사용할 수 있습니다. 검색 엔진 최적화 설정에 액세스하려면 환경을 편집하고사이트 구성> 검색 엔진 최적화왼쪽 탐색에서

사이트 맵에 비디오 태그를 포함하려면 사이트 맵에 비디오 태그포함옵션을 선택합니다. 처음 32 개의 태그 만 포함됩니다. 사이트 맵에 태그를 추가하면 방문자가 비디오 태그를 기반으로 비디오를 찾을 수 있도록 검색 엔진 웹 크롤러에 태그가 노출됩니다.

참고: 이러한 URL은 사이트가 게시될 때까지 사용할 수 없습니다.

사이트 맵은 24 시간 동안 캐시됩니다. 24 시간 후 캐시에서 제거되고 다시 액세스 할 때 다시 생성됩니다. 비디오가 포털 경험에 추가되면 사이트 맵이 다시 생성되거나 (24 시간 후) 사이트가 다시 게시 될 때까지 사이트 맵에 포함되지 않습니다.

사이트맵 제출을 빌드하는 방법에 대한 자세한 내용은 Google 설명서를 참조하십시오 .
