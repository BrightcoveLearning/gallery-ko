포털 경험에 대한 액세스 제어

{{ page.description }}

액세스 제어 프로필은 포털 경험에 대한 액세스를 제어하는 데 사용됩니다. 액세스 제어 프로필은 갤러리 설정의 일부로 생성되며 다음과 같이 사이트에 대한 액세스를 제한 할 수 있습니다.

  • IP 주소를 기반으로 액세스 제한
  • 사용자에게 자격 증명 (SSO) 입력 요구
  • 사용자에게 액세스 코드 입력 요구

액세스 제어 프로파일 작성에 대한 자세한 내용은 액세스 제어 프로파일구성을 참조하십시오 . 프로필이 생성되면 포털 경험에 할당하여 액세스를 제어 할 수 있습니다. 사이트에 액세스 제어 프로필을 할당하려면 다음 단계를 따르십시오.

  1. 포털 경험을 편집하십시오.
  2. 클릭사이트 구성> 액세스 제어왼쪽 탐색에서
  3. 사이트에 대한 액세스 제어 프로필을 선택합니다. 기본적으로 모든 사이트는 공개됩니다.
  4. 저장을 클릭합니다 .

액세스 제어 설정을 변경 한 후 사이트를 다시 게시해야합니다.
