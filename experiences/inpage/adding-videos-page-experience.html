인 페이지 경험에 비디오 추가

{{ page.description }}

Video Cloud 비디오 및 YouTube 비디오를 인 페이지 경험에 추가 할 수 있습니다. 인페이지 경험에 비디오를 추가하려면 경험을 편집한 다음비디오왼쪽 탐색에서

참고: YouTube 동영상을 경험에 추가하려면 먼저 YouTube 설정을구성해야 합니다 .

딸깍 하는 소리동영상 추가비디오 선택기를 연 다음원천 :

  • 비디오 클라우드 -Video Cloud 비디오 콘텐츠가 사용됩니다.
  • YouTube - YouTube 동영상 콘텐츠가 사용됩니다 (자세한 내용은 이 문서의 YouTube 통합 세부 정보섹션 참조)
참고: 비디오 클라우드와 YouTube의 동영상을 동일한 인페이지 환경에 추가할 수 있습니다.

YouTube소스를 선택한 경우 사용할 YouTube 채널을 선택할 수 있는 채널 선택기가 나타납니다.

비디오 클라우드소스를 선택한경우 비디오를 추가하는 방법에 대한 방법을 선택합니다.

  • 비디오 선택 -동영상은 수동으로 선택하여 사용됩니다.
  • 재생 목록 선택 -재생 목록의 동영상이 사용됩니다.
  • 참조 ID 사용 -재생 목록 참조 ID의 동영상이 사용됩니다.

동영상이 선택되면 페이지 본문에 표시됩니다. 이 예에서는 Video Cloud 재생 목록이 사용되었습니다.

참고: 경험을 변경한 후에는 경험을다시 게시하여 해당 경험을 표시하는 사이트에 변경 사항이나타나도록 해야 합니다.

수동으로 비디오 선택

비디오를 수동으로 선택하려면 비디오 선택방법을선택합니다 . Video Cloud 소스를 사용하면 비디오 선택기가 Video Cloud 계정에서 가장 최근에 업데이트 된 100 개의 비디오 목록을 표시합니다. 페이지 아래로 이동하면 새 동영상을 가져옵니다. YouTube 소스를 사용하면 선택한 채널의 모든 비디오가 표시됩니다. 비디오를 클릭하여 선택하고 다시 클릭하여 선택을 취소하십시오. 검색 필드를 사용하여 비디오를검색합니다 . 이 예에서는 Video Cloud 소스가 선택되었습니다.

작업을 마쳤으면추가를 클릭합니다 .

비디오 검색

검색필드는 비디오를 검색하는 데 사용할 수 있습니다. Video Cloud 비디오를 검색 할 때 Gallery는 비디오 이름, 짧고 긴 설명, 태그 및 사용자 정의 필드를 검색합니다. YouTube 동영상을 검색 할 때 Gallery는 YouTube 검색 API를 사용합니다.

검색되는 필드를더 세밀하게 제어하고 텍스트를 포함 및 제외하려면 추가 검색 옵션을클릭합니다. 이 옵션은 Video Cloud 소스가 사용되는 경우에만 사용할 수 있습니다.

재생 목록 사용

재생 목록에서 동영상을 사용하려면 재생 목록 선택방법을선택합니다 . Video Cloud 소스를 사용하면 비디오 선택기는 Video Cloud 계정에서 가장 최근에 업데이트 된 20 개의 재생 목록 목록을 표시합니다. YouTube 소스를 사용하면 비디오 선택기에 선택한 채널의 모든 재생 목록 목록이 표시됩니다. 검색 필드를 사용하여 재생 목록을 검색 할 수 있습니다. 이 예에서는 Video Cloud 소스가 선택되었습니다.

하나의 재생 목록 만 선택할 수 있습니다. 완료되면선택을 클릭합니다 .

재생 목록 참조 ID 사용

재생 목록 참조 ID를 지정하여 재생 목록의 비디오를 사용하려면방법 NS 참조 ID 사용 . 이 옵션은 Video Cloud 소스를 사용하는 경우에만 지원됩니다. 비디오 선택기는재생 목록 참조 ID를 입력할 수 있는 필드를 제공합니다 . 재생 목록 참조 ID를 사용하면 미디어 모듈을 사용하여 참조 ID를 다른 재생 목록에 다시 할당하여 경험의 비디오를 쉽게 변경할 수 있습니다. 경험에 표시되는 동영상을 변경해야 하는 경우 경험을 편집하고 다시 게시할 필요가 없습니다.

하나의 재생 목록 참조 ID 만 사용할 수 있습니다. 완료되면선택을 클릭합니다 .

비디오에 상호 작용 추가

상호 작용 및 컴패니언 구성 요소를 비디오에 추가하고 비디오가 재생 될 때 표시 될 수 있습니다. 상호 작용에는 다음이 포함됩니다. 연결그리고카드비디오가 재생될 때 플레이어의 일부로 표시됩니다. 컴패니언 구성 요소에는 HTML , 이미지및텍스트객체가 포함되며 플레이어 외부에 표시됩니다.

상호 작용으로 구성된 비디오는 상호 작용 유형을 나타내는상호 작용열에 아이콘이 표시됩니다. 다음 예에서는링크및카드상호 작용과컴패니언 이미지가구성되었습니다.

상호 작용 및 컴패니언 구성 요소 추가 및 구성에 대한 자세한 내용은 다음을 참조하십시오.

  • 인 페이지 경험에 상호 작용 추가
  • 인 페이지 경험을위한 컴패니언 구성 요소 구성

인 페이지 경험에서 비디오 제거

인페이지 경험에서 비디오를 제거하는 프로세스는방법비디오를 추가하기 위해 선택:

  • 비디오 - 제거할 비디오 옆에 있는 확인란을 선택한 다음 비디오제거를 클릭합니다 . 삭제 아이콘() 작업열에 있습니다.
  • 재생 목록 선택 - 재생 목록제거를 클릭하여 모든 비디오를 제거합니다.
  • 참조 ID 사용 - 재생 목록제거를 클릭하여 모든 비디오를 제거합니다.

비디오가 제거되면 해당 비디오에 구성된 상호 작용 또는 컴패니언 구성 요소는삭제되지 않습니다. 그런 다음 비디오가 경험에 다시 추가되면 상호 작용 및 컴패니언 구성 요소가 다시 나타납니다.

YouTube 통합 세부 정보

  • YouTube 동영상을 다운로드 할 수 없습니다.
  • YouTube 비디오를 비디오 사이트에 추가하려면 먼저 YouTube 설정을 구성해야합니다.
  • YouTube 동영상은 공개로 개인 정보 보호 설정이 있어야합니다.
  • 현재 갤러리는 YouTube 동영상에 대한 분석 데이터를 수집하지 않습니다. 인 페이지 경험의 YouTube 동영상은 경험 분석 대시 보드에 분석 데이터를 표시하지 않습니다.
  • IMA 플러그인이있는 Brightcove 플레이어 내에서 재생되는 YouTube 동영상은 휴대 기기에 광고를 표시하지 않습니다.
  • ' YouTube를 사용하는 고객은 YouTube 이용 약관의 적용을 받습니다.
  • 검색시 모든 컬렉션은 YouTube 동영상으로 구성되어야하며 동영상은 선택한 채널의 소유 여야합니다.
