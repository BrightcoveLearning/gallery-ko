인페이지 경험 비디오 분석 검토

{{ page.description }}

인 페이지 경험 비디오 분석은 비디오 성능 및 참여에 대한 분석을 제공합니다. 비디오 분석을 검토하려면 경험을 편집하고경험 분석> 비디오왼쪽 탐색에서

이 항목에는 다음 섹션이 포함됩니다.

  • 시간 범위 선택
  • 성능 분석 검토
  • 참여 분석 검토
참고: 인 페이지 경험 비디오 분석은 플레이어 ID와 경험 ID의 조합을 사용하여 추적됩니다. 경험에 대한 플레이어를 변경하면 해당 경험에 대한 분석이 재설정됩니다. 플레이어를 다시 변경하면 경험에 대한 분석이 반환됩니다.

시간 범위 선택

보고서 기간을 지정하려면 빠른 날짜를 클릭하거나 달력 컨트롤을 사용할 수 있습니다.

빠른 날짜 사용

일련의 빠른 날짜가 페이지 상단에 있습니다. 다음과 같은 빠른 날짜를 사용할 수 있습니다.

  • 오늘
  • 어제
  • 지난 7 일
  • 지난 30 일
  • 이번 달 - 달력 월 누계 - 예를 들어, 오늘이 3월 17일 경우 이 보고서에는 3월 1일부터 3월 17일이 표시됩니다.
  • 지난 달 - 이전 달력 월 - 예를 들어, 오늘이 2월 14일 경우 이 보고서에는 1월 1일부터 1월 31일이 표시됩니다.

보고서를 업데이트하려면 빠른 날짜를 클릭하십시오.

달력 컨트롤 사용

달력 컨트롤을 클릭하여 시작 및 종료 날짜를 선택할 수도 있습니다.

적용을클릭하여 보고서를 업데이트합니다.

보고 시간대

데이터는 계정의보고 시간대를 기반으로합니다. 보고 시간대는보고 데이터의 일 경계를 계산하는 데 사용됩니다. 보고 시간대 설정을 변경하면 향후 데이터에만 영향을 미치며 변경 사항은 소급 적용되지 않습니다. 시간대 설정을 변경하면 변경 사항이 적용된 날짜 경계를 볼 때 데이터가 평평하거나 급증 할 수 있습니다. 시간대 설정에 대한 업데이트는 즉시 적용되지 않을 수 있으며 분석 데이터는 시스템이 변경 사항을 처리 할 때까지 이전 설정을 계속 참조 할 수 있습니다.

보고 시간대는 계정 설정의 일부로 구성 할 수 있습니다. 보고 시간대 변경에 대한 자세한 내용은 계정 설정관리를 참조하십시오 .

성능 분석 검토

성능 분석을 검토하려면경험 분석> 비디오왼쪽 탐색에서성능페이지 헤더에서. 기본적으로 성능 데이터는비디오에 의해 표시됩니다 . 경험의 비디오에 대한 다음 데이터가 표시됩니다.

  • 비디오 이름 및 ID
  • 비디오 조회수 -스트림이 시작될 때 녹화 된 비디오 재생 시작 횟수 (되감기 또는 다시 재생은 포함되지 않음) 개별 시청자의 척도가 아닙니다.
  • 비디오 노출 -동영상이 플레이어에로드되고 상호 작용할 준비가 된 횟수
  • 재생률 (%) -동영상이로드 된 후 재생 한 사용자 비율 (동영상 조회수를 동영상 노출 수로 나눈 값)
  • 평균 조회 시간 -시청자가 동영상을 시청 한 평균 시간
  • 평균 조회 율 -동영상을 본 평균 비율 (평균 시청 시간을 동영상 길이로 나눈 값)

디바이스별 성능을보려면 [장치] 를 선택합니다. 가능한 장치 유형은 다음과 같습니다.

  • 데스크탑 -데스크톱 및 노트북 컴퓨터 포함
  • 태블릿 -iPad, Android 태블릿 등이 포함됩니다.
  • 변하기 쉬운 - 휴대 전화
  • 연결된 TV / 게임 콘솔 / 컴패니언 장치 -스마트 TV, 셋톱 박스 및 게임 콘솔 포함
  • 다른 -위의 어느 것도 매핑되지 않은 사용자 에이전트의 경우

공연은국가및날짜별로 볼 수도있습니다.

경험의 비디오에 대한 다음 데이터가 표시됩니다.

  • 기기 종류
  • 비디오 조회수 -스트림이 시작될 때 녹화 된 비디오 재생 시작 횟수 (되감기 또는 다시 재생은 포함되지 않음) 개별 시청자의 척도가 아닙니다.
  • 비디오 노출 -동영상이 플레이어에로드되고 상호 작용할 준비가 된 횟수
  • 재생률 (%) -동영상이로드 된 후 재생 한 사용자 비율 (동영상 조회수를 동영상 노출 수로 나눈 값)
  • 평균 조회 시간 -시청자가 동영상을 시청 한 평균 시간

참여 분석 검토

성능 분석을 검토하려면경험 분석> 비디오왼쪽 탐색에서약혼페이지 헤더에서.  기본적으로 참여 데이터는 다음으로 표시됩니다. 동영상하지만 다음으로 표시할 수도 있습니다. 일 .

참여 분석은 경험의 비디오에 대해 다음 데이터를 표시합니다.

  • 비디오 이름 및 ID
  • 참여 점수 -동영상 조회 당 평균 시청률 (모든 시청률을 동영상 조회수로 나눈 값으로 계산). 이 계산 방법에 대한 자세한 내용은 비디오 참여 점수계산을 참조하십시오.
  • 조회수 1 % -동영상의 1 % 지점에서 기록 된 조회수
  • 25 % 조회수 -동영상의 25 % 지점에서 기록 된 조회수
  • 50 % 조회수 -동영상의 50 % 지점에서 기록 된 조회수
  • 75 %에서 조회 -동영상의 75 % 지점에서 기록 된 조회수
  • 100 % 조회수 -동영상의 100 % 지점에서 기록 된 조회수
  • 소셜 공유 -동영상이 소셜 미디어 계정에 공유 된 횟수
