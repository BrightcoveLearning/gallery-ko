인페이지 경험 클릭 유도문안 분석 검토

{{ page.description }}

클릭 유도 문안 분석은 인 페이지 경험에 대한 클릭 연결 및 양식 전환에 대한 분석을 제공합니다. 클릭 유도문안 분석을 검토하려면 경험을 수정하고경험 분석> 클릭 유도문안왼쪽 탐색에서

시간 범위 선택

보고서 기간을 지정하려면 빠른 날짜를 클릭하거나 달력 컨트롤을 사용할 수 있습니다.

빠른 날짜 사용

일련의 빠른 날짜가 페이지 상단에 있습니다. 다음과 같은 빠른 날짜를 사용할 수 있습니다.

  • 오늘
  • 어제
  • 지난 7 일
  • 지난 30 일
  • 이번 달 - 달력 월 누계 - 예를 들어, 오늘이 3월 17일 경우 이 보고서에는 3월 1일부터 3월 17일이 표시됩니다.
  • 지난 달 - 이전 달력 월 - 예를 들어, 오늘이 2월 14일 경우 이 보고서에는 1월 1일부터 1월 31일이 표시됩니다.

보고서를 업데이트하려면 빠른 날짜를 클릭하십시오.

달력 컨트롤 사용

달력 컨트롤을 클릭하여 시작 및 종료 날짜를 선택할 수도 있습니다.

적용을클릭하여 보고서를 업데이트합니다.

클릭률 분석 검토

클릭률 분석은 인 페이지 경험에서 상호 작용을 추적합니다. 다음 개체는 추적 할 클릭 방문 분석을 트리거합니다.

  • 링크 URL로 구성된 링크 상호 작용
  • 링크 URL로 구성된 카드 상호 작용
  • 클릭 가능한 URL로 구성된 이미지 구성 요소

클릭률 분석을 검토하려면경험 분석> 클릭 유도문안왼쪽 탐색에서클릭률페이지 헤더에서.

클릭률 분석은 경험에 대한 다음 데이터를 표시합니다.

  • 클릭 유도문안 이름 - 클릭 유도문안 이름:
    • 링크-링크 제목을 표시합니다.
    • 카드-카드 제목을 표시합니다.
    • 이미지-생성 중에 할당 된 이름을 표시합니다.
    • 사용자 정의 HTML-생성 중에 할당 된 이름을 표시합니다.
  • 업데이트 됨 -상호 작용에 대한 마지막 업데이트 날짜 및 시간
  • 유형 -상호 작용, 링크 또는 카드 유형
  • 노출수 - 상호작용이 표시된 횟수(카드의 경우 카드 아이콘()가 나타납니다. 인상을 기록하기 위해 카드를 표시할 필요는 없습니다.)
  • 클릭률 -링크를 클릭하거나 카드의 링크를 클릭하거나 이미지 구성 요소의 클릭 연결 URL을 클릭 한 횟수
  • 클릭률 (%) -링크를 클릭 한 시청자 비율 (클릭률을 노출 수로 나눈 값)

양식 변환 분석 검토

양식 변환 분석은 인 페이지 환경에서 양식 제출 및 사용자 정의 HTML 링크 활동을 추적합니다. 다음 개체는 추적 할 양식 전환 분석을 트리거합니다.

  • HTML 컴패니언 구성 요소에서 제출 된 양식
  • 사용자 정의 HTML 구성 요소에서 제출 된 양식
  • 사용자 정의 HTML 구성 요소 내의 클릭 가능한 URL

양식 전환 분석을 검토하려면경험 분석> 클릭 유도문안왼쪽 탐색에서양식 변환페이지 헤더에서.

양식 전환 분석은 경험에 대해 다음 데이터를 표시합니다.

  • 클릭 유도 문안 이름 -구성 요소의 이름 (구성 요소가 경험에 추가 될 때 이름이 지정됨)
  • 업데이트 됨 -구성 요소의 마지막 업데이트 날짜 및 시간
  • 유형 - 상호 작용 유형(HTML 컴패니언 구성 요소는 () 아이콘
  • 노출 수 -구성 요소가 표시된 횟수
  • 양식 변환 -양식이 제출되었거나 사용자 정의 HTML 구성 요소 내의 URL이 클릭 된 횟수
  • 양식 전환율 (%) -양식을 제출 한 시청자 비율 (양식 전환을 노출 수로 나눈 값)
