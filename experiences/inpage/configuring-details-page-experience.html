인 페이지 경험에 대한 세부 정보 구성

{{ page.description }}

경험 이름, 설명 및 레이아웃 크기는 모두 개요 설정의 일부로 구성 할 수 있습니다. 개요 설정을 구성하려면 환경을 편집한 다음개요왼쪽 탐색에서

개요 페이지가 표시됩니다.

  • 체험 유형 -인 페이지 경험에 사용되는 템플릿을 표시합니다.
  • 경험 제목 및 설명 - 처음 새 경험을 만들 때 입력하라는 메시지가 표시됩니다. 경험 제목그리고경험 설명 . 이러한 값은 여기에서 편집 할 수 있습니다.
  • 경험 언어 -UI 메시지를 표시하는 데 사용할 언어를 선택하십시오.
  • 레이아웃 크기 -인 페이지 경험의 크기를 결정합니다. 자세한 내용은 경험 크기설정을 참조하십시오.

완료되면저장을클릭하여 변경 사항을 저장합니다.

참고: 경험을 변경한 후에는 경험을다시 게시하여 해당 경험을 표시하는 사이트에 변경 사항이나타나도록 해야 합니다.

경험 크기 설정

인 페이지 경험은 다음 세 가지 방법 중 하나로 크기를 조정할 수 있습니다.

  • 반응 형 -경험은 컨테이너에 맞게 자동으로 크기가 조정됩니다.
    참고: 이 옵션을 선택할 때 권장 임베드 코드를 사용해야합니다. No JavaScript 포함 코드는 응답하지 않습니다.
  • 고정 크기 - 경험의 크기는 지정된너비및높이값에 따라 크기가 조정됩니다.
  • 고정 너비 - 경험의 너비가 지정된너비값으로 설정됩니다.

완료되면저장을클릭하여 변경 사항을 저장합니다.

참고: 레이아웃 크기를 변경하려면 환경을 다시 게시하고 기존의 모든 포함 게시 코드를 업데이트해야합니다. 자세한 내용은 인페이지 환경미리 보기 및게시를 참조하십시오.
