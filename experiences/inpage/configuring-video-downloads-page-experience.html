인페이지 경험을 위한 비디오 다운로드 구성

{{ page.description }}

갤러리 인 페이지 경험은 시청자가 비디오 콘텐츠를 다운로드 할 수있는 다운로드 링크를 표시하도록 구성 할 수 있습니다. 사용자 지정 필드 값에 따라모든 비디오또는특정 비디오에 대한 다운로드 링크가 표시되도록 페이지 내 경험을 구성할 수있습니다. 비디오를 다운로드 할 수있는 경우 비디오 옆에 다운로드 링크가 나타납니다. 사용 가능한 최고 품질의 변환이 다운로드됩니다.

참고: iOS 기기에서는 비디오 다운로드가 지원되지 않습니다.

경험에 있는 모든 비디오에 대한 다운로드 링크를 제공하려면 페이지 내 경험에 대한 설정구성설명서를 참조하십시오. 지정된 비디오 만 다운로드하도록 선택적으로 허용하려면 다음 단계를 수행해야합니다.

  1. 다운로드 가능한 콘텐츠를 나타내는 사용자 지정 필드 만들기
  2. 사용자 정의 필드에 적절한 값을 지정하십시오.
  3. 다운로드를 허용하도록 인 페이지 환경 구성

다운로드 가능한 콘텐츠를 나타내는 사용자 정의 필드 만들기

다운로드 할 수있는 비디오는 사용자 정의 필드의 값을 설정하여 표시됩니다. 먼저 비디오를 다운로드 할 수 있음을 나타내는 데 사용할 사용자 정의 필드를 계정에 만듭니다.

  1. 비디오 클라우드 스튜디오에 로그인합니다.
  2. 탐색 헤더에서 ADMIN 을클릭한 다음비디오 필드 를 클릭합니다 . ADMIN 메뉴에 액세스하려면 계정 관리자 여야합니다.
  3. 사용자 지정 필드추가를 클릭합니다 .
  4. 비디오를 다운로드 할 수 있는지 여부를 나타내는 데 사용할 새 사용자 정의 필드를 만듭니다. 할당 된 이름과 값은 중요하지 않습니다. 아래 예에서는다운로드 가능한 새 사용자 정의 필드가생성되고 있습니다. NS 유형로 설정됩니다목록 , 2개의 값, 진실그리고거짓 . 다운로드할 수 있는 비디오는 이 값이 다음으로 설정됩니다. 진실비디오 속성의 일부로.
  5. 딸깍 하는 소리변경 사항을 저장하다그리고구하다 .

사용자 정의 필드 값 할당

다운로드 할 수있는 모든 비디오에는 생성 된 사용자 정의 필드에 할당 된 값이 있어야합니다.

  1. 미디어모듈을 엽니다.
  2. 비디오를 다운로드 가능으로 표시하려면 비디오 이름을 클릭하여 비디오 속성을 엽니 다.
  3. 사용자 정의 필드섹션을 찾아편집을 클릭합니다 .
  4. 사용자 정의 필드의 값을 적절하게 설정하십시오. 이 예에서 값은다운로드 가능로 설정됩니다진실 .
  5. 저장을 클릭합니다 .
  6. 필요에 따라 추가 비디오의 비디오 속성을 편집합니다.
참고: 미디어 모듈의빠른 편집 기능을사용하여 여러 비디오의 사용자 지정 필드 값을 동시에 편집할 수 있습니다.

다운로드를 허용하도록 인 페이지 환경 구성

사용자 정의 필드 값을 기반으로 비디오 다운로드를 허용하도록 인 페이지 환경을 구성하려면 다음 단계를 따르십시오.

  1. 갤러리 모듈을 엽니 다.
  2. 경험을 찾은 다음편집을 클릭합니다 .
  3. 딸깍 하는 소리설정> 동영상왼쪽 탐색에서
  4. 비디오 다운로드섹션에서 사용자 지정 필드 값을 기반으로 각 비디오에 대한 다운로드 링크제공옵션을 선택합니다.
  5. 위의 4 단계에서 생성 한 사용자 정의 필드 이름과 값을 입력합니다. 이 예에서 생성된 사용자 정의 필드의 이름은다운로드 가능값은 다음으로 설정되었습니다. 진실다운로드할 수 있는 동영상의 경우.
  6. 저장을클릭한 다음 경험을 다시 게시합니다.
  7. 다운로드가능한사용자 지정 필드가 true로 설정된 모든 비디오에 대해 다운로드 링크가 표시되는지 확인합니다 .
