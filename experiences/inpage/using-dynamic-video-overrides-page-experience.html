인페이지 경험에서 동적 비디오 재정의 사용

{{ page.description }}

인 페이지 경험에서 비디오를 변경하려는 경우가있을 수 있습니다. 예를 들어, 경험의 비디오는 사용자가 선택하거나 사이트의 다른 부분으로 이동하여 변경해야 할 수 있습니다. 각각 다른 비디오 세트가있는 새로운 인 페이지 경험을 만드는 대신 임베드 코드를 수정하여 적절한 비디오를 경험에 할당 할 수 있습니다. 소스 코드의 변경 사항은 사용중인 소스 코드 유형에 따라 다릅니다.

  • 추천 소스 코드
  • 자바 스크립트 소스 코드 없음

추천 소스 코드

일반적인 인 페이지 경험 포함 코드 (권장 옵션)는 다음과 같습니다.

<div data-experience="597a2aa114b2ed00103052af"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

인페이지 환경에 연결된 비디오를 동적으로 재정의하려면 포함 코드를 변경하여div태그에 속성을 추가하여 사용할 비디오 또는 재생 목록을 지정할 수 있습니다.

개별 비디오 지정

사용할 개별 비디오를 지정하려면data-video-ids매개변수div태그를 지정한 다음 쉼표로 구분된 동영상 ID 목록을 지정합니다.

<div data-experience="597a2aa114b2ed00103052af" 
     data-video-ids="12345,56789"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

재생 목록 지정

사용할 재생 목록을 지정하려면data-playlist-id매개변수div태그를 클릭한 다음 사용할 재생 목록 ID를 지정합니다.

<div data-experience="597a2aa114b2ed00103052af" 
     data-playlist-id="12345"></div>
<script src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/live.js"></script>

자바 스크립트 소스 코드 없음

일반적인 인 페이지 경험 포함 코드 (JavaScript 옵션 없음)는 다음과 같습니다.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

인페이지 환경에 연결된 비디오를 동적으로 재정의하려면 포함 코드를 변경하여src속성에 매개 변수를 추가할 수 있습니다.

개별 비디오 지정

사용할 개별 비디오를 지정하려면videoIds매개 변수를 추가한 다음 쉼표로 구분된 비디오 ID 목록을 지정합니다.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?videoIds=12345,56789""
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

재생 목록 지정

사용할 재생 목록을 지정하려면playlistId매개 변수를 추가한 다음 사용할 재생 목록 ID를 지정합니다.

<iframe style="display:block;border:none;margin-left:auto;margin-right:auto;"
src="https://players.brightcove.net/20318290001/experience_597a2aa114b2ed00103052af/index.html?playlistId=12345"
allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
