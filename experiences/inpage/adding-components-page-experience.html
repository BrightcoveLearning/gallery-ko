인페이지 경험에 구성 요소 추가

{{ page.description }}

인 페이지 경험에 구성 요소를 추가하여 경험을 사용자 지정하고 브랜딩 할 수 있습니다. 인페이지 경험에 구성 요소를 추가하려면 경험을 편집한 다음형세왼쪽 탐색에서

더하기 아이콘() 구성 요소를 추가하려는 위치에서 추가할 구성 요소 유형을 선택합니다.

다음 유형의 구성 요소를 추가 할 수 있습니다.

  • 광고
  • 잡담
  • 맞춤 HTML
  • 영상
  • 본문
  • 3Play
  • 트위터 피드
참고: 경험을 변경한 후에는 경험을다시 게시하여 해당 경험을 표시하는 사이트에 변경 사항이나타나도록 해야 합니다.

광고 구성 요소 추가

경험에 광고를 추가하려면 다음과 같이 하십시오.

  1. 구성 요소 선택기에서광고를 클릭합니다.
  2. 광고의이름을입력합니다.
  3. 헤더 광고 코드및광고 단위 코드를 입력합니다 . 헤더 광고 코드는 경험의 모든 광고에서공유됩니다.
  4. 광고를 언제 표시할지 선택합니다.
  5. 저장을 클릭합니다 . 광고가 경험에 나타납니다.
참고: 여러 광고가 포함된 인페이지 환경이 있는 경우 각광고 단위 코드에 고유한div ID가 있는지 확인합니다. 의 및 컨테이너 매개 변수의 id를 해당div id와adblock1일치하도록 편집해야 할 수 있습니다.

채팅 구성 요소 추가

채팅 기능을 제공하기 위해 타사 채팅 위젯을 경험에 추가 할 수 있습니다. 경험에 채팅 구성 요소를 추가하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리채팅구성 요소 선택기에서.
  2. 채팅 구성 요소의이름을입력합니다.
  3. 채팅 HTML 코드를 입력하세요.
  4. 채팅이 표시되어야하는시기를 선택합니다.
  5. 저장을 클릭합니다 . 채팅은 경험에 렌더링됩니다.

사용자 정의 HTML 구성 요소 추가

사용자 지정 구성 요소를 사용하여 사용자 지정 HTML을 경험에 추가 할 수 있습니다. 예를 들어, 마케팅 자동화 플랫폼 (MAP)을 사용하여 사용자 정의 리드 양식을 만든 경우 MAP의 리드 양식 HTML을 사용자 정의 HTML 구성 요소를 사용하여 인 페이지 경험에 추가 할 수 있습니다.

경험에 사용자 지정 HTML을 추가하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리관습구성 요소 선택기에서.
  2. HTML 구성 요소의이름을입력합니다.
  3. 경험에 추가 할 사용자 정의 HTML을 입력하십시오.
  4. HTML이 표시되어야하는시기를 선택하십시오.
  5. 저장을 클릭합니다 . HTML은 경험에서 렌더링됩니다.

HTML을 입력하지 않으면 HTML 자리 표시자가 표시됩니다.

참고: 갤러리 제작 환경은 보안상의 이유로 사용자 정의 HTML 구성 요소에서 스크립트를 제거합니다. 스크립트는 인 페이지 경험을 미리보고 게시 할 때 실행됩니다. 이로 인해 사용자 정의 HTML 구성 요소가 다르게 보일 수 있습니다. 특히 JavaScript 콘텐츠가 많은 경우 더욱 그렇습니다.

이미지 구성 요소 추가

경험에 이미지를 추가하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리영상구성 요소 선택기에서.
  2. 이미지의이름을입력합니다.
  3. 이미지를 지정하려면 다음을 수행 할 수 있습니다.
    • 파일 시스템에서 이미지를 끌어서 놓기 영역에 놓으십시오.
    • 찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다.
    • 원격 이미지 파일의 URL을 입력하십시오.
  4. 이미지크기를 설정합니다 .
    • 이미지가 컨테이너에 맞게 크기가 조정됩니다. -컨테이너보다 크기가 큰 이미지는 크기에 맞게 축소됩니다. 작은 이미지는 업 스케일되지 않습니다.
    • 너비와 높이 -이미지의 너비와 높이를 수동으로 입력
  5. 이미지정렬을 설정합니다 .
  6. 클릭스루 URL을 입력합니다 . 이미지를 클릭 할 때 이동할 URL입니다. 새 창에서 링크를열 수도있습니다.
  7. 이미지가 표시되어야하는시기를 선택하십시오.
  8. 저장을클릭하여 이미지를 경험에 추가합니다.

이미지를 선택하지 않으면 이미지 자리 표시자가 표시됩니다.

텍스트 구성 요소 추가

경험에 텍스트를 추가하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리텍스트구성 요소 선택기에서.
  2. 경험에 추가 할 텍스트를 입력하십시오. 도구 모음을 사용하여 텍스트 서식을 지정할 수 있습니다.
  3. 텍스트의이름을입력합니다.
  4. 텍스트가 표시되어야하는시기를 선택하십시오.
  5. 저장을클릭하여 텍스트를 추가합니다. 텍스트가 경험에 나타납니다.

텍스트를 입력하지 않으면 텍스트 자리 표시자 표시됩니다.

3Play 구성 요소 추가

3Play 구성 요소를 사용하여 경험에 3Play 대화 형 대본을 추가 할 수 있습니다. 이 구성 요소는 3Play Media의 비디오 캡션이 생성된 경우에만 작동합니다 . 이 구성 요소가 경험에 추가되면 대화식 성적 기록표시링크가 나타납니다. 클릭하면 비디오 대본이 나타납니다. 다음 예에서는 비디오 플레이어 아래에 3Play 구성 요소가 추가되었으며재생 중 표시되도록구성되었습니다.

라이브 예제는이 링크를참조하여 비디오를 재생하십시오.

참고: 계정에 대해 3Play 구성 요소를 활성화하려면 계정 관리자 또는브라이트코브 지원에 문의하십시오 .

경험에 3Play 구성 요소를 추가하려면

  1. 구성 요소 선택기에서 3Play를 클릭합니다.
  2. 3Play 구성 요소의이름을입력합니다.
  3. 3Play 프로젝트 ID를 입력합니다 . 3Play 프로젝트 ID를 찾으려면 3Play 계정에 로그인하고 페이지 헤더의계정 개요링크를 클릭합니다. 프로젝트 ID가 해당 페이지에 나타납니다.
  4. 3Play 구성 요소가 표시되어야하는시기를 선택합니다.
  5. 저장을클릭하여 3Play 구성 요소를 추가합니다. 3Play 구성 요소가 경험에 나타납니다.

Twitter 피드 구성 요소 추가

경험에 Twitter 피드를 추가하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리트위터 피드구성 요소 선택기에서.
  2. 트위터 피드의이름을입력합니다.
  3. 트위터 사이트에서 위젯을 만듭니다 . 트위터 위젯 생성에 대한 도움말은 Twitter 개발자 문서를 참조하십시오 .
  4. Twitter 위젯의 HTML을 복사하여트위터 피드설정.
  5. Twitter 피드가 표시되어야하는시기를 선택하십시오.
  6. 저장을클릭하여 Twitter 피드를 추가합니다. Twitter 피드가 경험에 표시됩니다.
