인 페이지 경험 사용자 지정

{{ page.description }}

텍스트, 이미지, 광고, 사용자 지정 HTML 및 트위터 피드와 같은 개체를페이지 내 경험에 추가하여 경험을 사용자 정의하고브랜드화할 수 있습니다. 개체를 추가하려면인페이지 경험 , 경험을 편집한 다음형세왼쪽 탐색에서 인페이지 환경을편집하면레이아웃모드가 기본적으로 열립니다.

장치 유형 선택

페이지 상단의 장치 선택기를 사용하여 편집할 수 있습니다. 인페이지 경험데스크톱, 태블릿 또는 모바일 레이아웃을 사용합니다.

기본적으로데스크톱레이아웃은 환경을 편집할 때 표시됩니다. 모바일 레이아웃을 선택하면 모바일 레이아웃이 표시됩니다.

경험을 미리 보면 다른 브라우저 탭에서 경험이 열리고 갤러리 UI 요소없이 경험을 미리 볼 수 있습니다.

경험 상태 전환

N 페이지경험에는 시청자가 상호 작용할 때 변경되는 시각적 상태가 다릅니다. 이러한 각 상태의 모양은 페이지 상단의 링크를 클릭하여 사용자 정의 할 수 있습니다.

다음 상태를 사용자 정의 할 수 있습니다.

  • 플레이하기 전에 -비디오 재생이 시작되기 전에 처음 렌더링 될 때 경험이 표시되는 방식
  • 놀이 -동영상이 재생 될 때 경험이 표시되는 방식
  • 플레이 후 -동영상 재생 후 경험이 표시되는 방식
참고: 라이브 이벤트 인 페이지 템플릿은 대부분의 다른 인 페이지 템플릿과 다르게 구성됩니다. 자세한 내용은 라이브 이벤트 인페이지 환경만들기를 참조하십시오 .

재생및 AFTER PLAY상태에서〈뒤로링크를 클릭하면 경험이이전 플레이로 돌아갑니다.상태. 경험이 입력됩니다플레이 후다음 비디오 자동 재생 옵션이 활성화되었는지 여부에 관계없이 상태를 유지합니다.

참고: 일부 템플릿이〈뒤로링크를 표시하지는 않습니다.

다른 경험 상태를 편집하려면 상태를 클릭 한 다음경험에 구성 요소 추가필요에 따라. 레이아웃메뉴는 각 상태에 대한 추가 사용자 지정 옵션을 제공합니다. 자세한 내용은 모양사용자 정의섹션을 참조하십시오.

모양 사용자 지정

레이아웃메뉴는 인페이지 환경을 사용자 지정하는 데 사용할 수 있는 추가 설정을 제공합니다.

  • 표시
  • 비디오 정보
  • 썸네일
  • 나누는

사용 가능한 레이아웃 옵션은 선택한 템플릿을 기반으로합니다.

프레젠테이션 설정

프레젠테이션설정은 경험의 프레젠테이션을 제어합니다. 딸깍 하는 소리프레젠테이션왼쪽 탐색에서 프레젠테이션을 선택한 다음구하다 .

참고: 수평 재생 목록 템플릿은프레젠테이션옵션을 제공하지 않습니다.

비디오 정보 설정

비디오 정보설정은비디오이름과간단한 설명이 표시되는위치를 제어합니다. 딸깍 하는 소리비디오 정보왼쪽 탐색에서 위치를 선택한 다음구하다 .

썸네일 설정

[축소판]설정은 비디오 썸네일과 관련하여 비디오 이름이 표시되는 위치를 제어합니다. 딸깍 하는 소리썸네일왼쪽 탐색에서 위치를 선택한 다음구하다 .

공유 설정

공유설정은 소셜 공유 아이콘이 표시되는 위치를 제어합니다. 딸깍 하는 소리나누는왼쪽 탐색에서 위치를 선택한 다음구하다 .

인 페이지 환경에 구성 요소 추가

레이아웃 편집기를 사용하여 경험에 구성 요소를 추가 할 수 있습니다. 더하기 아이콘() 구성 요소를 추가하려는 위치에서 추가할 구성 요소 유형을 선택합니다.

인페이지 환경에 추가할 수 있는 구성 요소에 대한 자세한 내용은 인페이지 환경에 구성 요소추가를 참조하십시오 .

구성 요소 편집 및 삭제

인 페이지 환경에서 구성 요소를 편집하거나 삭제하려면 구성 요소 위에 마우스를 올려 놓습니다. 수정/삭제 아이콘 나타날거야. 편집을클릭하면 구성 요소를 편집할 수 있는 구성 요소 편집기가 열립니다. 삭제를클릭하면 경험에서 구성 요소가 삭제됩니다.

참고: 구성 요소를 두 번 클릭하여 편집할 수도 있습니다.

구성 요소 재정렬

경험에있는 구성 요소는 끌어서 놓기를 사용하여 이동할 수 있습니다. 구성 요소의 순서를 변경하려면 클릭하여 올바른 위치로 끕니다. 녹색 선은 구성 요소가있는 위치를 나타냅니다.

구성 요소를 경험의 다른 위치로 끌 수도 있습니다.

인 페이지 경험 미리보기

경험을 미리 보면 새 브라우저 탭에서 경험이 열리므로 갤러리 제작 환경 외부에서 미리 볼 수 있습니다. 미리보기를 사용하여 경험이 예상대로 렌더링되는지 확인할 수 있습니다. 미리보기는 반응 형이므로 브라우저 창을 좁히거나 넓혀 중단 점을 볼 수 있습니다. 경험을 미리 보려면 미리보기버튼을 클릭합니다.

인 페이지 경험 게시

경험을 게시하면 변경 사항이 적용되어 프로덕션 인 페이지 경험에 게시됩니다. 경험에 대한 모든 변경 사항은 표시되기 전에 게시되어야합니다. 인페이지 환경을 게시하려면게시 및 포함을 클릭합니다 . 인페이지 환경 미리 보기 및 게시에 대한 자세한 내용은 인페이지 경험미리 보기 및 게시를 참조하십시오 .
