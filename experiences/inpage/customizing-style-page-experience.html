인 페이지 경험의 스타일 사용자 지정

{{ page.description }}

스타일은 인페이지 환경에서사용되는 글꼴, 글꼴 크기 및 색상을 제어합니다 . 스타일을 사용자 지정하려면 경험을 편집한 다음스타일왼쪽 탐색에서

참고: 인페이지 환경을 변경한 후 경험을다시 게시하여 해당 경험을 표시하는 사이트에 변경 사항이나타나도록 해야 합니다.

장치 상태 전환

페이지 상단에있는 링크를 사용하여 데스크톱, 태블릿 또는 모바일을 포함한 다양한 장치 상태에서 스타일을 사용자 지정할 수 있습니다.

기본적으로 스타일을 사용자화할 때데스크톱모드가 표시됩니다. 장치 상태는 선택한 장치 유형에서 경험이 어떻게 보이는지 미리 보는 데 사용할 수 있습니다.

경험 상태 전환

인페이지경험에는 시청자가 경험과 상호 작용할 때 변경되는 시각적 상태가 다릅니다. 각 상태의 스타일은 페이지 상단의 링크를 클릭하여 사용자 정의 할 수 있습니다.

다음 상태를 사용자 정의 할 수 있습니다.

  • 플레이하기 전에 -비디오 재생이 시작되기 전에 처음 렌더링 될 때 경험이 표시되는 방식
  • 놀이 -동영상이 재생 될 때 경험이 표시되는 방식
  • 플레이 후 -동영상 재생 후 경험이 표시되는 방식 (다음 동영상 자동 재생이 비활성화 된 경우)

이러한 각 상태의 스타일을 지정하려면 상태를 클릭 한 다음 필요에 따라 스타일을 사용자 지정합니다.

테마 선택

테마는 인페이지 환경의 스타일을빠르게 지정하는 데 사용할 수 있는 미리 정의된 색상 및 글꼴 세트를 제공합니다 . 갤러리는 기본 테마 세트를 제공하지만나만의 맞춤 테마 만들기계정 전체에서 공유됩니다. 테마가 경험에 적용되면 스타일 옵션을 사용하여 모양을 추가로 사용자 지정할 수 있습니다. 현재 테마는 스타일링 메뉴 상단에 표시됩니다. 이 예에서는라이트테마가 선택됩니다.

테마를 적용하려면인페이지 경험 , 이 차례를 따라라:

  1. 왼쪽 탐색 메뉴에서 현재 테마를 클릭하여 테마 선택기를 엽니 다.
  2. 테마를 클릭하여 선택하십시오. 미리보기가 업데이트되어 테마 색상이 표시됩니다.
  3. 완료되면 페이지 상단의 스타일로 < 돌아가기링크를 클릭합니다.

기본 테마재설정링크를 클릭하면 기본 테마가 경험에 적용됩니다.

스타일링 할 개체 결정

스타일을 지정할 개체를 결정하려면 경험의 개체 위에 마우스를 올려보세요. 대부분의 경우 녹색 상자가 개체를 둘러 쌉니다. 해당 개체의 스타일이 활성화되고 적용되지 않는 다른 모든 스타일은 회색으로 표시됩니다. 예를 들어 오버레이 설명 위로 마우스를 가져가면위에 까는 것그리고오버레이 설명스타일. NS 위에 까는 것스타일은 오버레이 제목 및 오버레이 설명을 포함하여 오버레이의 모양을 제어합니다. 오버레이 설명은 비디오 설명 개체의 모양을제어합니다. 오버레이를 사용하면 전체 오버레이에 스타일을 지정하거나 제목 및 설명에 대한 스타일을 개별적으로 지정할 수 있습니다.

개체 위로 마우스를 가져갈 때 녹색 상자가 없으면 개체가 개체의 일부로 간주됨을 나타냅니다. 글로벌스타일을 사용하여 스타일을 지정할 수 있습니다. 글로벌스타일. 경험에 추가된 텍스트 객체는 텍스트가 경험에 배치될 때 텍스트 편집기를 사용하여 스타일을 변경하지 않는 한 GLOBAL스타일을 사용하여 스타일을 지정할 수 있습니다.

스타일 변경

스타일 패널은 경험에서 개체의 스타일을 사용자 지정하는 데 사용됩니다. 스타일 패널을 사용하여 다음을 사용자 지정할 수 있습니다.

  • 텍스트 색상 및 배경 색상
  • 폰트
  • 글꼴 크기

사용자 정의 CSS 파일도 지정할수 있습니다.

텍스트 및 배경색 값 변경

색상 값을 선택하려면 다음 단계를 따르십시오.

  1. 색상 원을 클릭하여 색상 선택기를 엽니 다.
  2. 색상 슬라이더를 사용하여 색상 범위를 설정합니다.
  3. 불투명도 슬라이더를 사용하여 색상의 불투명도를 설정합니다.
  4. 색상 사각형을 클릭하여 색상을 선택합니다.
  5. X를클릭하여 색상 피커를 닫습니다.

색상 코드를 입력하여 색상을 지정할 수도 있습니다.

  • 마녀 -HEX 색상 값은 #RRGGBB로 지정됩니다. RR (빨간색), GG (녹색) 및 BB (파란색) 는 색상 강도를 지정하는 00과 FF 사이의 16진수 정수입니다.
  • RGBA -RGBA 색상 값은 rgba (red, green, blue, alpha)로 지정됩니다. 알파 매개 변수는 0.0 (완전 투명)에서 1.0 (완전 불투명) 사이의 숫자입니다.
  • HSLA -HSLA 색상 값은 hsla (hue, saturation, lightness, alpha)로 지정됩니다.

위/아래 아이콘() 색상 모드 사이를 전환합니다.

글꼴 선택

스타일 패널에 현재 글꼴이 표시됩니다. 드롭 다운 목록을 사용하여 새 글꼴을 선택합니다. X를 클릭하여 글꼴 선택기를 닫습니다.

글꼴 크기 선택

현재 글꼴 크기가 스타일 패널에 표시됩니다. 드롭 다운 목록을 사용하여 새 글꼴 크기를 선택합니다. X를 클릭하여 글꼴 크기 선택기를 닫습니다.

변경된 모든 스타일에는 실행 취소 화살표가 표시됩니다.

실행 취소 화살표() 스타일을 원래 값으로 재설정합니다. 스타일 패널 아래쪽에 있는 모든 기본값재설정을 클릭하여 모든 스타일을 재설정할 수도 있습니다.

사용자 정의 글꼴 추가

경험에서 사용자 지정 글꼴을 사용할 수도 있습니다. Google Font 디렉토리는 웹 페이지에서 사용할 수 있는 오픈 소스 글꼴을제공합니다. 사용할 글꼴을 찾은 다음 (이 예에서는 Acme 글꼴을 사용함) 이 스타일선택을 클릭합니다 . 글꼴 정보가 표시됩니다. 선택한 글꼴에 대한 스타일 시트 링크가 필요합니다.

글꼴 선택기에 사용자 정의 글꼴을 추가하려면 다음과 같이 하십시오.

  1. 경험을 편집하고 클릭하십시오. 스타일왼쪽 탐색에서
  2. 왼쪽 탐색에서 글꼴 드롭 다운 목록을 클릭합니다.
  3. 사용자 지정 글꼴추가링크를 클릭합니다.
  4. 사용자 정의 글꼴 CSS 파일의 URL을 입력하십시오. 위의 예에 따라 글꼴 CSS 파일에 대한 URL은 다음과 같습니다. 
    https://fonts.googleapis.com/css2? 가족 = 애크미 &디스플레이 =스왑
  5. 라이브러리에 글꼴추가를 클릭합니다 .
  6. 글꼴이 글꼴 선택기 상단에 추가되었는지 확인합니다.
참고: 오픈 소스 글꼴 모음은 https://fonts.google.com 에서 사용할 수있습니다.
참고: CSS 파일이 CORs 뒤에 있거나 다른 유형의 파일 제한이있는 경우 해당 URL이 글꼴 사용을 허용하도록 허용되지 않았기 때문에 해당 URL이 저작 환경, 미리보기 페이지 또는 공유 가능한 링크에 올바르게 표시되지 않습니다. 사용자 지정 글꼴을 사용하는 경우 경험에서 글꼴에 액세스할 수 있는지 확인합니다.
참고: 인 페이지 환경은 현재 base64 인라인 글꼴을 지원하지 않습니다. 이에 대한 해결 방법은 대신 경험의 글꼴을 재정의하는 CSS URL을 통해 사용자 정의 CSS 파일을 추가하는 것입니다.

사용자 정의 CSS 파일 사용

사용자 정의 CSS 파일의 스타일을 사용하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리맞춤 CSS왼쪽 탐색에서
  2. CSS 파일의 URL을 입력하거나 CSS 코드를 붙여 넣습니다.
  3. 저장을 클릭합니다 .
