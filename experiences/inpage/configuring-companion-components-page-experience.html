인 페이지 경험을위한 컴패니언 구성 요소 구성

{{ page.description }}

모든 인 페이지 경험 템플릿은 컴패니언 구성 요소를 포함하는 기능을 제공합니다. 컴패니언 구성 요소는 다음과 같습니다.

  • 맞춤 HTML
  • 이미지
  • 본문

컴패니언 구성 요소를 사용하면 재생중인 동영상에 따라 플레이어 외부의 콘텐츠가 변경 될 수 있습니다. 예를 들어 제품 동영상이 재생 될 때 플레이어 오른쪽에 해당 제품에 대한 자세한 정보를 표시하는 클릭 가능한 이미지가있을 수 있습니다. 컴패니언 구성 요소는 카드 및 링크 상호 작용과 유사하지만 차이점은 카드 및 링크가 비디오 플레이어 자체에 표시된다는 점입니다. 동영상이재생상태일 때 컴패니언 컴포넌트가 비디오 플레이어의 오른쪽에 표시됩니다.

참고: 링크 및 카드 상호 작용 추가에 대한 자세한 내용은 페이지 내 환경에 상호 작용추가를 참조하십시오 .
참고: 경험을 변경한 후에는 경험을다시 게시하여 해당 경험을 표시하는 사이트에 변경 사항이나타나도록 해야 합니다.

컴패니언 구성 요소보기

구성된 컴패니언 구성 요소를 보려면 환경을 편집한 다음비디오왼쪽 탐색에서 상호 작용으로 구성된 비디오는상호 작용상호 작용 유형을 나타내는 열입니다.

컴패니언 구성 요소는 HTML ), 이미지 ( ) 및 텍스트 ( ).

참고: 링크 ( ) 및 카드 ( ) 비디오에 링크 및 카드가 구성되어 있음을 나타내는 아이콘이 나타날 수 있습니다. 링크 및 카드에 대한 자세한 내용은 페이지 내 환경에 상호 작용추가를 참조하십시오 .

컴패니언 HTML 구성 요소 구성

컴패니언 HTML 구성 요소를 구성하려면 다음 단계를 따르십시오.

  1. 경험을 편집한 다음비디오왼쪽 탐색에서
  2. 비디오를 클릭하여 선택하십시오.
  3. 대화식추가 > 컴패니언 HTML을클릭하거나 HTML을 클릭합니다 ()아이콘은작업열에 있습니다. 상호 작용 편집기가 열립니다.
  4. HTML 구성 요소의이름을입력합니다.
  5. HTML 코드를 붙여 넣습니다.
  6. 구성 요소의 시작 시간과 기간을 구성하려면구성링크를 클릭합니다.
  7. 시작 시간 및 기간을 선택합니다 .
    • 전체 동영상보기 -전체 동영상에 대한 구성 요소가 표시됩니다.
    • 시작 시간 및 기간 선택 -구성 요소의 시작 시간과 기간을 수동으로 설정합니다.
  8. 구성 요소의지속 시간을설정합니다.
    참고: 재생 시간은 플레이어 아래의 슬라이더를 사용하여 설정할 수도 있습니다. 컴포넌트가 나타날 때 왼쪽 화살표를 드래그한 다음 오른쪽 화살표를 컴포넌트가 사라질 시점까지 드래그합니다.
  9. 확인하다영상 끝날때까지시작 시간만 설정하고 동영상이 끝날 때까지 구성요소를 표시하려면
  10. 확인하다레이아웃의 모든 비디오에 이 상호작용을 적용합니다.경험의 모든 비디오에 이 구성요소를 표시하려는 경우.
  11. 저장을클릭하여 변경 사항을 저장합니다.
참고: 다른 비디오에 상호 작용을 할당하려면 플레이어 위의 비디오 선택기 드롭다운 목록을 사용하여 다른 비디오를 선택합니다.

컴패니언 이미지 구성 요소 구성

컴패니언 이미지 구성 요소를 구성하려면 다음 단계를 따르세요.

  1. 경험을 편집한 다음비디오왼쪽 탐색에서
  2. 비디오를 클릭하여 선택하십시오.
  3. 대화식추가 > 컴패니언 이미지 를클릭하거나이미지를 클릭합니다 ()아이콘은작업열에 있습니다. 상호 작용 편집기가 열립니다.
  4. 이미지 구성 요소의이름을입력합니다.
  5. 이미지를 지정하려면 다음을 수행 할 수 있습니다.
    • 파일 시스템에서 이미지를 끌어서 놓기 영역에 놓으십시오.
    • 찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다.
    • 원격 이미지 파일의 URL을 입력하십시오.
  6. 이미지크기를 설정합니다 .
    • 이미지가 컨테이너에 맞게 크기가 조정됩니다. -컨테이너보다 크기가 큰 이미지는 크기에 맞게 축소됩니다. 작은 이미지는 업 스케일되지 않습니다.
    • 너비와 높이 -이미지의 너비와 높이를 수동으로 입력
  7. 이미지정렬을 설정합니다 .
  8. 클릭스루 URL을 입력합니다 . 이미지를 클릭 할 때 이동할 URL입니다. 새 창에서 링크를열 수도있습니다.
  9. 구성 요소의 시작 시간과 기간을 구성하려면구성링크를 클릭합니다.
  10. 시작 시간 및 기간을 선택합니다 .
    • 전체 동영상보기 -전체 동영상에 대한 구성 요소가 표시됩니다.
    • 시작 시간 및 기간 선택 -구성 요소의 시작 시간과 기간을 수동으로 설정합니다.
  11. 구성 요소의지속 시간을설정합니다.
    참고: 재생 시간은 플레이어 아래의 슬라이더를 사용하여 설정할 수도 있습니다. 컴포넌트가 나타날 때 왼쪽 화살표를 드래그한 다음 오른쪽 화살표를 컴포넌트가 사라질 시점까지 드래그합니다.
  12. 확인하다영상 끝날때까지시작 시간만 설정하고 동영상이 끝날 때까지 구성요소를 표시하려면
  13. 확인하다레이아웃의 모든 비디오에 이 상호작용을 적용합니다.경험의 모든 비디오에 이 구성요소를 표시하려는 경우.
  14. 저장을클릭하여 변경 사항을 저장합니다.
참고: 다른 비디오에 상호 작용을 할당하려면 플레이어 위의 비디오 선택기 드롭다운 목록을 사용하여 다른 비디오를 선택합니다.

컴패니언 구성 요소 편집 / 삭제

상호 작용열의 아이콘 위에 마우스를 올려 놓으면 해당 유형의 모든 상호 작용이 표시됩니다. 이 예에서는 이미지 구성 요소가 구성되었습니다.

구성 요소를 편집하려면대화식열의 아이콘 위로 마우스를 가져간 다음 편집 () 구성 요소 t 옆에 있는 아이콘을클릭하거나 비디오 제목을 클릭합니다. 상호 작용 편집기가 열립니다. 구성 요소를 편집한 다음구하다변화들.

구성 요소를 삭제하려면대화식열의 아이콘 위로 마우스를 가져간 다음 삭제 () 구성 요소 옆에 있는 아이콘.

편집 아이콘() 에서작업열에 있습니다. 비디오에 대한 모든 상호 작용 목록이 표시됩니다.

편집 아이콘을 클릭하여 구성 요소를 편집하거나 삭제 아이콘을 클릭하여 상호 작용을 삭제합니다.

컴패니언 컴포넌트는레이아웃메뉴에서도 액세스할 수 있습니다. 재생상태를 선택하면 컴패니언 아이콘이 경험에 표시됩니다.
