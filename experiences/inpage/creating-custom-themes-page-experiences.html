인페이지 경험을 위한 맞춤형 테마 만들기

{{ page.description }}

테마는 사전 정의된 색상 및 글꼴 세트를 제공하여인페이지 환경의 스타일을 빠르게 지정하는 데 사용할 수있습니다. 갤러리는 기본 테마 세트를 제공하지만 사용자 지정 테마를 직접 만들 수도 있습니다. 생성 된 모든 사용자 지정 테마는 Video Cloud 계정에서 공유됩니다. 테마가 적용되면인페이지 경험 , 다음을 사용할 수 있습니다. 스타일링 옵션경험의 모양을 추가로 사용자 정의합니다.

사용자 지정 테마 만들기

사용자 지정 테마를 만들려면 다음 단계를 따르십시오.

참고: 기존 테마가 있는 경우 테마를복제한 다음 편집하는 대신 새 테마를만드는 것이 더 빠를 수 있습니다.
  1. 경험을 편집한 다음스타일왼쪽 탐색에서 현재 테마는 스타일링 메뉴 상단에 표시됩니다.
  2. 테마를 클릭하십시오. 모든 테마 목록이 표시됩니다.
  3. 사용자 지정 테마추가링크를 클릭합니다.
  4. 테마의이름을입력합니다.
  5. 적절하게 스타일을 편집하십시오. 스타일 설정에 대한 모든 변경 사항은 자동으로 저장됩니다.
  6. 완료되면 스타일로 < 돌아가기를 클릭합니다 .

인페이지 환경에 테마 적용

인페이지 환경에테마를 적용하려면 다음 단계를 따르십시오.

  1. 프로젝트를 편집한 다음스타일왼쪽 탐색에서 현재 테마는 스타일링 메뉴 상단에 표시됩니다.
  2. 테마를 클릭하십시오. 모든 테마 목록이 표시됩니다. 새 테마를 클릭하여 선택하십시오.
  3. 완료되면 스타일로 < 돌아가기를 클릭합니다 .

테마 편집

테마를 편집하면 경험에서 특정 스타일 설정이 변경되지않은 경우 테마가 적용된 기존 페이지 내 경험에 스타일 변경 사항이 적용됩니다 . 예를 들어 기업 테마, 기업테마가 만들어지고 오버레이 텍스트 색상이 빨간색으로 설정됩니다. 새 인 페이지 환경이 생성되고 오버레이 텍스트 색상이 파란색으로 변경됩니다. 누군가 신청하면기업 테마이 새로운 환경에서는 모든 스타일 변경 사항이 에 지정된 스타일로 대체됩니다. 기업 테마스타일이 이전에 변경되었으므로 오버레이 텍스트 색상을 제외하고.

테마를 편집하려면 다음 단계를 따르십시오.

참고: 기본 제공 테마는 편집할 수 없습니다.
  1. 편집경험클릭스타일왼쪽 탐색에서 현재 테마는 스타일링 메뉴 상단에 표시됩니다.
  2. 테마를 클릭하십시오. 모든 테마 목록이 표시됩니다.
  3. 수정 아이콘() 편집하려는 테마에 대해
  4. 적절하게 스타일을 편집하십시오. 스타일 설정에 대한 모든 변경 사항은 자동으로 저장됩니다.
  5. 완료되면 스타일로 < 돌아가기를 클릭합니다 .

테마 복제

테마를 복제하려면 다음 단계를 따르십시오.

  1. 편집경험클릭스타일왼쪽 탐색에서 현재 테마는 스타일링 메뉴 상단에 표시됩니다.
  2. 테마를 클릭하십시오. 모든 테마 목록이 표시됩니다.
  3. 복제 아이콘() 복제하려는 테마에 대해
  4. 테마의 새 이름을 입력하고저장을 클릭합니다 .
  5. 적절하게 스타일을 편집하십시오. 스타일 설정에 대한 모든 변경 사항은 자동으로 저장됩니다.
  6. 완료되면 스타일로 < 돌아가기를 클릭합니다 .

테마 삭제

테마를 삭제하려면 다음 단계를 따르십시오.

참고: 기본 제공 테마는 삭제할 수 없습니다.
  1. 편집경험클릭스타일왼쪽 탐색에서 현재 테마는 스타일링 메뉴 상단에 표시됩니다.
  2. 테마를 클릭하십시오. 모든 테마 목록이 표시됩니다.
  3. 삭제 아이콘() 삭제하려는 테마에 대해 삭제를 확인하는 메시지가 표시됩니다.
  4. 완료되면 스타일로 < 돌아가기를 클릭합니다 .
