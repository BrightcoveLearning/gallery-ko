페이지 내용

    시뮬레이션된 라이브 비디오 사용

    이 항목에서는 VOD 비디오가 라이브 상태인 것처럼 보이도록 구성하는 방법을 배웁니다.

    Simulive는시뮬레이트된 라이브에대해 짧으며 VOD 콘텐츠를 라이브 상태인 것처럼 최종 사용자에게 표시할 수 있는 방법입니다. Simulive 콘텐츠는 이벤트 주최자에게 매력적입니다. 콘텐츠를 사전 녹화하고 라이브에 앞서 광범위한 테스트를 수행할 수 있습니다. 인코더 문제 또는 기타 물류로 인해 발생하는 기존 라이브 이벤트의 잠재적인 고장 날로 인해 발생할 수 있는 위험을 줄입니다. Simulive 동영상은 비디오와 함께 카운트다운 타이머를 표시하여 실시간 방송을 표시합니다. 아래의 예는 주요 컬렉션의 simulive 비디오입니다.

    기본 및 보조 컬렉션에 있는 비디오의 경우 카운트다운 타이머가 비디오의 카드에 표시됩니다.

    가상 이벤트 환경에서 simulive 콘텐츠를 사용하려면 다음을 수행해야 합니다.

    • 시뮤라이브 플레이어 플러그인 설정
    • 플레이어를 경험과 연결
    • 시뮤라이브 사용자 정의 필드 만들기
    • 시뮤라이브 비디오 속성 구성
    참고: 카운트다운 타이머의 일부로 표시되는 텍스트는 구성 파일의labels개체에 있으며 쉽게 사용자 정의하거나 지역화할 수 있습니다.

    시뮤라이브 플레이어 플러그인 설정하기

    Simulive 플레이어 플러그인은 가상 이벤트 경험에서 사용하는 플레이어에 맞게 구성되어야 합니다.

    1. 탐색 머리글에서 플레이어를 클릭하여 플레이어 모듈을 엽니다.
    2. 브라이트코브는 가상 이벤트 경험을 위한 전용 플레이어를 만들 것을 권장합니다. 전용 플레이어를 만들지 않은 경우새 플레이어를 클릭하고 플레이어이름을 지정한 다음저장을클릭합니다.
    3. 구성하고자 하는 플레이어의 링크를 클릭합니다.
    4. 왼쪽 탐색 메뉴에서플러그인을클릭합니다.
    5. 플러그인 > 사용자 정의 플러그인 추가를클릭합니다.
    6. 플러그인 이름에을 입력합니다simulivePlugin .
    7. 자바 스크립트 URL의경우 다음을 입력합니다.
      https://virtual-events-experience.s3.amazonaws.com/v1/videojs-simulive-plugin.js
    8. CSS URL의경우 다음을 입력합니다.
      https://virtual-events-experience.s3.amazonaws.com/v1/videojs-simulive-plugin.css
    9. 저장을 클릭하여 변경 사항을 저장합니다.
    참고: 변경 내용이 플레이어에 게시되기까지 최대 5분이 소요될 수 있습니다.

    사용자 지정 플레이어 구성

    사용자 지정 플레이어를 사용하도록 가상 이벤트 환경을 구성하려면 다음 단계를 수행하십시오.

    1. 갤러리모듈을 엽니다.
    2. 경험을 편집합니다.
    3. 왼쪽 탐색에서비디오 및 재생 > 플레이어 및 리드 양식을클릭합니다.
    4. 사이트별 플레이어 사용 옵션을 선택하고 Simulive플러그인으로 구성된 플레이어를 선택합니다.
    5. 저장을 클릭합니다.

    시뮤라이브 사용자 정의 필드 만들기

    사용자 정의 필드는 비디오를 사용할 수 있는 날짜와 시간을 저장하는 데 사용됩니다. simulive 사용자 정의 필드를 만들려면 다음과 같이 하십시오.

    1. 탐색 헤더에서 ADMIN을 클릭한 다음 비디오 필드를클릭합니다. ADMIN 메뉴에 액세스하려면 계정 관리자여야 합니다.
    2. 사용자 정의 필드 추가를클릭합니다.
    3. 다음 값을 사용하여 새 사용자 정의 필드를 만듭니다.
      • 표시 이름 - 시뮤라이브 시작
      • 내부 이름 - 시뮤라이브_시작
      • 유형 - 텍스트
      • 설명 - 예상 형식은 YYYY-MM-DDH:MM:SS-HH:MM 입니다.
    4. 필드 저장을클릭합니다.
    사용자 정의 필드 생성에 대한 자세한 내용은사용자 정의 메타데이터 필드 생성을참조하십시오.

    시뮤라이브 비디오 속성 구성

    사용자 정의 필드Simulive Start는비디오를 사용할 수있는 날짜와 시간을 보유합니다. 시뮤라이브 시작 시간에 도달할 때까지 동영상을 볼 수 없습니다. Simulive 동영상도simulive에태그를 지정해야 합니다. 비디오를 simulive 비디오로 구성하려면 다음 단계를 수행하십시오.

    1. 미디어 모듈을 엽니다.
    2. 비디오 제목을 클릭하여 속성을 엽니 다.
    3. 비디오 속성섹션을 찾아편집을클릭합니다.
    4. 시뮬레이리브를사용하여 비디오에 태그를 지정합니다.
    5. 저장을 클릭합니다.
    6. 사용자 정의 필드섹션을 찾아편집을클릭합니다.
    7. 시뮤라이브 시작필드에 날짜 및 시간 값을 입력합니다. 필드의 올바른 형식은 다음과 같습니다. YYYY-MM-DDTHH:MM:SS-HH:MM어디YYYY-MM-DD연도, 월, 일이며 적절한 앞에 0이 붙습니다. HH:MM:SS 24 시간시, 분, 초이며 적절한 앞에 0이 붙습니다. -HH:MM시간대 오프셋입니다 (예 : UTC-4의 경우 -04 : 00, UTC + 8.5의 경우 +08 : 30 등).
      참고: Simulive는 로컬 컴퓨터 시간과 구성된Simulive 시작시간을 비교하여 이벤트가 시작될 시기를 결정하기 위해 지정된 사용자의 로컬 컴퓨터 시간에 의존합니다. 이러한 이유로 일부 사용자의 경우 이벤트가 예상보다 일찍 시작될 수 있습니다. 이는 일반적으로 이러한 사용자의 컴퓨터 시간이 정확하지 않기 때문입니다.
    8. 저장을 클릭합니다.

    정책 키 값 업데이트

    카운트다운 타이머가 시뮤라이브 비디오에 표시되려면 구성 파일의 정책 키 값을 검색 사용 정책 키로 업데이트해야 합니다. 이렇게 하지 않으면 비디오와 함께 카운트다운이 표시되지 않습니다. 검색 사용 정책 키를 가져오는 단계는코드 샘플을 참조하십시오. 정책 키를 생성합니다 . 새 정책 키를 만든 후 구성 파일의policyKey값을 업데이트하고 환경을 다시 게시합니다.
