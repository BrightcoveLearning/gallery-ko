사이트 편집기를 사용하여 가상 이벤트 경험 사용자 지정

{{ page.description }}

갤러리 사이트 편집기는 경험의 모양과 느낌을 사용자 지정하는 데 사용됩니다. 경험 제목을 클릭하여 사이트 편집기를 엽니 다. 사이트 편집기는 다음과 같은 기능을 제공합니다.

  • 페이지에 개체 추가
  • 사이트에 비디오 추가
  • 사이트 스타일 사용자 지정
  • 클릭 유도 문안 구성
  • 모양 및 동작 사용자 지정
  • 사이트 기능 사용자 지정
  • 비디오 및 재생 사용자 지정
  • 사이트 구성 사용자 지정
  • 사이트 세부 정보 구성

이러한 옵션은 왼쪽 탐색 메뉴를 클릭하여 액세스 할 수 있습니다.

페이지에 개체 추가

텍스트, 이미지 및 링크와 같은 개체를 페이지에 추가하여 비디오 경험을 사용자 지정하고 브랜딩 할 수 있습니다. 페이지에 개체를 추가하는 방법에 대한 자세한 내용은 가상 이벤트 경험 페이지의 콘텐츠사용자 지정을 참조하십시오 .

사이트에 비디오 추가

사이트에 비디오를 추가하려면비디오왼쪽 탐색에서 비디오 사이트에 비디오를 추가하는 방법에 대한 자세한 내용은 가상 이벤트 경험에 비디오추가를 참조하십시오 .

사이트 스타일 사용자 지정

사이트에서 사용되는 색상과 글꼴을 사용자 지정하거나 다른 테마를 선택하려면스타일왼쪽 탐색에서 스타일을 사용자 지정하는 방법에 대한 자세한 내용은 포털 경험의 스타일사용자 지정을 참조하십시오 .

클릭 유도 문안 구성

클릭 유도 문안은 페이지에 사용자 정의 HTML, 이미지 또는 광고를 포함하는 기능을 제공합니다. 모든 템플릿이 클릭 유도 문안을 지원하는 것은 아닙니다. 클릭 유도문안을 구성하려면클릭 유도문안왼쪽 탐색에서

클릭 유도 문안 구성에 대한 자세한 내용은 포털 환경에서 클릭 유도 문안구성을 참조하십시오 .

모양 및 동작 사용자 지정

모양 및 동작 설정은 사용자 지정 머리글 / 바닥 글을 구성하고, 사용자 지정 CSS 및 JavaScript 파일을 사용하고, 배경 이미지 또는 비디오를 구성하는 데 사용됩니다. 모양 및 동작 설정을 구성하려면외모와 행동왼쪽 탐색에서

모양 및 동작 설정에 대한 자세한 내용은 가상 이벤트 경험의 모양 및 동작사용자 지정을 참조하십시오 .

사이트 기능 사용자 지정

사이트 기능 설정은 사이트 검색을 활성화하고, 소셜 공유 옵션을 구성하고, 비디오에 대한 댓글을 허용하는 데 사용됩니다. 사이트 기능을 구성하려면사이트 기능왼쪽 탐색에서

사이트 기능을 사용자 지정하는 방법에 대한 자세한 내용은 포털 경험의 사이트 기능사용자 지정을 참조하십시오 .

비디오 및 재생 설정 사용자 지정

비디오 및 재생 설정은 경험에 대한 사용자 지정 플레이어를 선택하고, Audience 통합 및 리드 양식을 구성하고, 다운로드 및 관련 링크와 같은 비디오 옵션을 구성하는 기능을 제공합니다. 비디오 및 재생 설정을 사용자 지정하려면비디오 및 재생왼쪽 탐색에서

비디오 및 재생 설정 구성에 대한 자세한 내용은 포털 환경의 비디오 및 재생 설정사용자 지정을 참조하십시오 .

사이트 구성 사용자 지정

사이트 구성 설정은 사용자 지정 도메인에서 사이트를 호스팅하고 SSL을 사용하여 보안을 유지하거나 사이트에 대한 액세스를 제한하는 기능을 제공합니다. 사이트 구성을 구성하려면사이트 구성왼쪽 탐색에서

사이트 구성 설정에 대한 자세한 내용은 포털 환경의 사이트 구성사용자 지정을 참조하십시오 .

사이트 세부 정보 구성

개요 설정은 사이트의 이름, 설명 및 언어를 편집하는 기능을 제공합니다. 사이트 세부 정보를 구성하려면개요왼쪽 탐색에서

사이트 세부 정보 구성에 대한 자세한 내용은 포털 경험에 대한 세부 정보구성을 참조하십시오 .
