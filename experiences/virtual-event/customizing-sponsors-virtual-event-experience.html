이벤트 스폰서 커스터마이징

{{ page.description }}

스폰서 이미지는 가상 이벤트 홈페이지에 추가 할 수 있습니다.

스폰서 정보는 구성 파일에서sponsors객체를 편집하여 사용자 정의됩니다.

  • name -스폰서 이름
  • image_url -스폰서 로고, 이미지 크기가 100px 너비로 조정됩니다.
  • link_url -이미지의 도착 URL

구성 파일의sponsors개체에는 홈페이지에서 스폰서 정보가 표시되는 위치를 제어하는 매개 변수도 있습니다. 스폰서 섹션 포지셔닝에 대한 자세한 내용은 홈페이지에서 콘텐츠위치 지정을 참조하십시오 .

불투명도 제거 및 로고 이미지 크기 조정

기본적으로 로고 이미지는 80 %의 불투명도 수준으로 스타일이 지정됩니다. 다음 CSS 스타일을 사용하여 불투명도를 100 %로 설정할 수 있습니다. max-width속성은 이미지의 크기를 변경하는 데 사용할 수 있습니다.


<style>
.bc-sponsors-section .bc-sponsors-list img {
    -webkit-filter: none;
    opacity: 1;
    max-width: 200;
}
</style>
환경을 편집하고사용자 정의 헤더 HTML필드의 맨 위에 CSS를 추가합니다.
