이벤트 의제 관리

{{ page.description }}

의제 모듈은 외부.csv 아젠다 파일 업로드를 지원하므로 모든 의제 항목을 아젠다 모듈에 수동으로 입력할 필요가 없습니다. 일정 파일 업로드는 이벤트 일정이 처음 생성되는 방식입니다. 의제가 업로드된 후 의제 모듈을 사용하여 편집할 수 있습니다.

참고: 업로드할 아젠다 파일을 만드는 방법에 대한 자세한 내용은 이벤트 아젠다 파일만들기를 참조하십시오 .

일정 파일 업로드

기존 의제를 Agenda 모듈로 가져오기 전에 의제 파일을.csv 파일로 내보내야 합니다. 아젠다 모듈은.csv 파일만 가져올 수 있습니다.

갤러리 사이트 편집기를 사용하여 이벤트 의제를 업로드하고 구성할 수 있습니다. 가상 이벤트 경험에 의제 파일을 업로드하려면 다음 단계를 따르십시오.

  1. 경험 이름을 클릭하여 갤러리 사이트 편집기를 엽니다.
  2. 왼쪽 탐색에서사이트 기능 > 의제를 클릭합니다 .
  3. 일정업로드 를 클릭합니다 .
  4. 업로드할 의제 파일을 찾아 선택합니다.
  5. 내 아젠다페이지에 의제가 나열되어 있는지 확인합니다.

일정 가져오기 중 오류가 발생하면 메시지가 표시됩니다. 가져오기 후상태가비활성화됩니다 . DEACTIVATED 상태의 의제는 가상 이벤트 경험에 표시되지 않습니다.

의제 제목 편집

일정 제목은 일정 상단의 체험 홈페이지에 표시됩니다. 기본적으로 의제 제목은 가져온 의제 파일 이름으로 설정됩니다.

의제 제목을 편집하려면 편집 아이콘 을 클릭하고 제목을 입력한 다음 Enter 키를 누릅니다 .

의제 보기

의제 파일이 업로드되면내 의제페이지에 표시됩니다.의제 세부 정보를 보려면이벤트 이름을 클릭하십시오 . 세부 정보를 표시하는 의제가 열립니다.

페이지 왼쪽의 날짜를 클릭하면 그날의 일정을 볼 수 있습니다. 트랙은 페이지 상단의 열 헤더에 표시됩니다.

세션 추가

새 세션을 추가하려면 다음 단계를 따르세요.

  1. 의제를 클릭하여 의제 편집기를 엽니다.
  2. 열려 있는 시간대를 클릭합니다.
  3. 세션 정보를 입력합니다.
    • 세션 이름
    • 스피커 - 클릭스피커 선택그리고하나 이상의 스피커 선택세션을 위해
    • 시작일
    • 시작 시간
    • 지속
    • 기술
    • 트랙 - 세션을 할당할 트랙을 선택합니다.
    • 세션 유형
      • 세션
      • 기조
      • 회의
      • 다른
    • 비디오 ID - 세션, 기조 연설 및 기타 세션 유형에서 사용 가능
    • URL - 회의 세션 유형에서 사용 가능
    • 표지 이미지
    • 관련 콘텐츠 - 세션 콘텐츠(유인물 또는 슬라이드)에 대한 링크를 제공하는 데 사용할 수 있습니다.
      • 레이블 - 콘텐츠 레이블
      • URL - 콘텐츠 링크
  4. 저장을 클릭합니다 .

세션이 의제에 추가되었는지 확인합니다. 같은 시간에 예약된 동일한 트랙에 여러 세션이 있는 경우 강조 표시되어 서로 옆에 나타납니다.

세션 편집 및 삭제

세션을 편집하거나 삭제하려면 세션을 클릭하여 세션 편집 패널을 엽니다. 세션을 편집하려면 원하는 대로 변경하고저장을 클릭합니다 . 세션을 삭제하려면 [세션삭제] 를 클릭합니다 .

세션에 발표자 추가

세션에 발표자를 추가하려면 다음 단계를 따르세요.

  1. 세션을 클릭하여 세션 편집기를 엽니다.
  2. 스피커선택을 클릭합니다 .
  3. 하나 이상의 스피커를 선택하거나 스피커만들기를 클릭하여 새스피커를 만듭니다.
    참고: 발언자 생성에 대한 자세한 내용은 이벤트 발표자관리를 참조하십시오 .
  4. 저장을 클릭합니다 .
검색필드는 스피커를 검색하는 데 사용할 수 있습니다.

새 트랙 추가

트랙은 관련 세션을 그룹화하는 데 사용됩니다. 예를 들어 마케팅 트랙에는 모든 마케팅 관련 세션이 있습니다. 의제는 일반적으로 여러 트랙을 포함합니다. 새 트랙을 추가하려면 다음 단계를 따르세요.

  1. 의제를 클릭하여 의제 편집기를 엽니다.
  2. 아젠다 상단에 있는 새 트랙추가링크를 클릭합니다. 링크를 표시하려면 의제를 오른쪽으로 스크롤해야 할 수 있습니다.
  3. 트랙 이름을입력하고 Enter 키를 누릅니다 .

트랙 편집

트랙을 편집하려면 트랙 이름을 클릭하고 변경한 다음 Enter키를 눌러 변경 사항을 저장합니다.

트랙 재정렬

의제에서 트랙 순서를 변경하려면 다음 단계를 따르세요.

  1. 의제를 클릭하여 의제 편집기를 엽니다.
  2. 이동할 트랙 이름 위로 마우스를 가져간 다음 잡기 아이콘을 클릭한 다음 트랙을 새 위치로 끕니다.

의제 활성화 및 비활성화

의제 상태는 의제가 연결된 가상 이벤트 경험에 표시되는지 여부를 결정합니다. 상태는 의제 이름 아래 페이지 머리글에 표시됩니다.

  • 활성화됨 - 의제는 연결된 가상 이벤트 경험에 표시됩니다.
  • 비활성화됨 - 의제는 연결된 가상 이벤트 경험에 표시되지 않습니다.

의제를 활성화하려면활성화페이지 상단에 있습니다. 의제를 비활성화하려면 해당 의제를 클릭하고비활성화를 클릭합니다 . 의제를 활성화/비활성화한 후에는 경험을 다시 게시할 필요가 없습니다.
