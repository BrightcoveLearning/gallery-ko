가상 이벤트 경험 페이지의 콘텐츠 사용자 정의

{{ page.description }}

텍스트, 이미지 및 링크와 같은 개체를 가상 이벤트 경험 페이지에 추가하여 비디오 경험을 사용자 지정하고 브랜딩 할 수 있습니다.

페이지에 개체 추가

사이트 편집기를 사용하여 이미지, 텍스트, 링크 및 환영 메시지와 같은 개체를 페이지에 추가 할 수 있습니다. 경험을 편집하여 시작한 다음페이지왼쪽 탐색에서 페이지 템플릿에 플러스 아이콘() 페이지에 추가할 수 있는 개체를 나타냅니다.

더하기 아이콘 위로 마우스를 가져간 다음 클릭하여 적절한 개체를 추가합니다.

페이지 템플릿에는 템플릿의 일부인 텍스트 개체가 포함될 수도 있습니다. 마우스 포인터를 가리키면편집버튼이 표시됩니다. 개체를 편집하려면 개체 또는편집버튼을 클릭합니다.

페이지 헤더에 로고 추가

로고는 텍스트 또는 이미지 파일 일 수 있습니다. 이미지의 경우 JPEG 및 PNG 이미지가 지원됩니다. 페이지 헤더에 로고를 추가하려면 페이지헤더에서+ 로고 추가를 클릭합니다.

텍스트 로고 추가

  1. 에서 텍스트 편집기를 사용하십시오. 텍스트탭을 클릭하여 헤더의 텍스트를 입력하고 서식을 지정합니다.
  2. 저장을 클릭합니다 .

이미지 로고 추가

  1. 이미지탭을 클릭합니다. 이미지 브라우저 대화 상자가 열립니다.
  2. 이미지를 지정하려면 다음을 수행 할 수 있습니다.
    • 파일 시스템에서 이미지를 끌어서 놓기 영역에 놓으십시오.
    • 찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다.
    • 원격 이미지 파일의 URL을 입력하십시오.
  3. 딸깍 하는 소리구하다완료되면.

메뉴 링크 추가

페이지 헤더에 메뉴 링크를 추가하여 사용자가 사이트의 다른 부분으로 이동할 수있는 방법을 제공 할 수 있습니다. 페이지 머리글에 메뉴 링크를 추가하려면 다음과 같이 하십시오.

  1. 딸깍 하는 소리 +메뉴 추가헤더 또는 클릭메뉴헤더에서 더하기 아이콘(). 디스플레이는 브라우저 창의 너비에 따라 변경됩니다.
  2. 텍스트및링크를 입력합니다 . 새 브라우저 창에서링크를 열려면 새 창을 선택합니다.
  3. 저장을 클릭합니다 .

추가 탐색 링크는 다음을 클릭하여 입력할 수 있습니다. 헤더에. 바닥글 링크를 추가하려면 + 메뉴 링크 추가를 클릭합니다 . 링크를 편집하려면 클릭하십시오.

환영 메시지 추가

사이트 방문자를 환영하기 위해 환영 메시지를 추가 할 수 있으며 텍스트와 이미지를 포함 할 수 있습니다. 고급 환영 메시지에 사용자 정의 HTML을 사용할 수 있습니다. 환영 메시지를 추가하려면 + 환영 메시지를 클릭합니다 .

문자 메시지 추가

텍스트탭은 텍스트 및/또는 이미지를 추가하는 데 사용할 수 있습니다. 페이지에서 이미지는 텍스트 아래에 나타납니다.

  1. 텍스트를 입력하십시오. 서식 지정 막대를 사용하여 텍스트 서식을 지정할 수 있습니다.
  2. 이미지를 추가하려면 다음을 수행 할 수 있습니다.
    • 파일 시스템에서 이미지를 끌어서 놓기 영역에 놓으십시오.
    • 찾아보기를클릭하고 파일 시스템에서 이미지를 선택합니다.
    • 원격 이미지 파일의 URL을 입력하십시오.
  3. 저장을 클릭합니다 .

사용자 정의 HTML 사용

사용자 정의 HTML은텍스트탭의 WYSIWYG 편집기를 사용하여 만들 수 없는 고급 레이아웃을 원할 때 유용합니다.

  1. 사용자 지정 HTML탭을 클릭합니다.
  2. 환영 메시지에 대한 사용자 정의 HTML을 붙여 넣으십시오.
  3. 저장을 클릭합니다 .

페이지 바닥 글에 탐색 링크 추가

페이지 바닥 글에 탐색 링크를 추가 할 수 있습니다. 이것은 제목과 아래의 일련의 링크를 원할 때 유용합니다. 예를 들면 다음과 같습니다.

페이지 바닥글 탐색 객체는 제목의자리 표시자와 연결된링크에 대한 +메뉴 링크추가의 두 부분으로 구성됩니다.

페이지 바닥글에 탐색 링크를 추가하려면 다음과 같이 하십시오.

  1. 자리 표시자를클릭하고 머리글 값을 입력한 다음저장을 클릭합니다 .
  2. +메뉴 링크추가를 클릭하고텍스트 , 링크 URL 및목록 순서를 입력합니다 . 링크가 작성될 때목록 순서값을 설정하여 링크의 순서를 지정할 수도 있습니다.

+메뉴 링크 추가를 클릭하여 추가 탐색 링크를 입력할 수있습니다.

페이지에서 개체 삭제

텍스트, 이미지 및 링크는 몇 가지 다른 방법으로 페이지에서 제거 할 수 있습니다.

페이지 머리글에서 텍스트 또는 이미지를 제거하려면

  1. 개체 위로 마우스를 이동하고 삭제 아이콘을 클릭합니다.

페이지 머리글에서 탐색 링크를 제거하려면

  1. 탐색 헤더에서 링크를 클릭하십시오.
  2. 삭제 를 클릭합니다 .

페이지 바닥 글에서 탐색 링크를 제거하려면

  1. 페이지 바닥 글에있는 링크를 클릭하십시오.
  2. 삭제 를 클릭합니다 .

환영 메시지 또는 텍스트를 제거하려면

  1. 텍스트 개체를 클릭합니다.
  2. 제목 텍스트를 지우고저장을 클릭합니다 .
