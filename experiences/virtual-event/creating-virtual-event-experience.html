가상 이벤트 경험 만들기

{{ page.description }}

갤러리는 가상 이벤트를 호스팅하고 전달하기위한 가상 이벤트 경험을 만드는 데 사용할 수있는 가상 이벤트 템플릿을 제공합니다. 새 가상 이벤트 경험을 만든 후 경험을 미리 채우라는 메시지가 표시됩니다. 브라이트코브는감사하지 않기를 선택하는 것이 좋습니다 .

경험에 비디오를 추가하는 프로세스는 가상 이벤트 템플릿이 컬렉션이 특정 방식으로 구성 될 것으로 예상하지만 다른 갤러리 포털 경험과 유사합니다. 자세한 내용은 가상 이벤트 환경에 비디오추가를 참조하십시오 .

템플릿에 구성 파일 추가

Virtual Event Portal Experience를 사용하려면 사이트에 구성 파일이 포함되어 있어야합니다. 구성 파일은 경험 헤더의 일부로 템플릿에 추가되고 일부 일반 템플릿 옵션을 제어하며 사이트의 여러 섹션을 구성 및 사용자 지정하는 데 사용됩니다. Brightcove는 새 가상 이벤트 포털 경험을 생성 한 직후 구성 파일을 추가 할 것을 권장합니다. 자세한 내용은 환경에 구성 파일추가를 참조하십시오 .

레이아웃 사용자 지정

갤러리 사이트 편집기는 페이지 레이아웃을 사용자 지정하는 데 사용됩니다. 로고, 메뉴 링크, 환영 메시지 및 페이지 바닥 글은 모두 갤러리 사이트 편집기를 사용하여 사용자 지정할 수 있습니다. 레이아웃 사용자 지정에 대한 자세한 내용은 가상 이벤트 경험 페이지의 콘텐츠사용자 지정을 참조하십시오 .

이벤트 의제 구성

이벤트 의제는 라이브 이벤트 및 기타 세션의 일정을 나열합니다. 의제가 생성되어 게시되면 이벤트 홈페이지에 표시됩니다. 이벤트 의제 생성에 대한 자세한 내용은 이벤트 아젠다관리를 참조하십시오 .

행사 연사 관리

발표자 섹션은 모든 이벤트 발표자로 자동으로 채워집니다. 모든 세션 발언자가 발언자 섹션에 나타납니다. 이벤트 발표자 관리에 대한 자세한 내용은 이벤트 발표자관리를 참조하십시오 .

이벤트 스폰서 커스터마이징

스폰서 섹션은 이벤트 스폰서를 강조하는 데 사용됩니다. 이벤트 스폰서를 사용자 지정하는 방법에 대한 자세한 내용은 이벤트 스폰서사용자 지정을 참조하십시오 .
