이벤트 의제 파일 만들기

{{ page.description }}

일반적으로 이벤트 의제는 처음에 Microsoft Excel 또는 Google 스프레드시트와 같은 타사 도구에서 생성됩니다. Agenda 모듈에서는 처음에 이벤트 의제를 만들기 위해 외부.csv 의제 파일을 업로드해야 합니다. 의제 파일이 업로드되면 의제 모듈을 사용하여 편집할 수 있습니다.

참고: 샘플 아젠다 템플릿 파일은 여기에서 다운로드할 수 있습니다.
참고: 의제 모듈에서 의제를 생성할 수 있는 방법은 없습니다. 이벤트 의제를 초기에 채우려면.csv 파일을 업로드해야 합니다.

열 이름 지정 요구 사항

초기 일정 파일을 생성할 때 샘플 일정 템플릿을 다운로드하여 편집하거나 기존 일정 파일을 사용할 수 있습니다. Agenda 모듈은 특정 열 이름의 존재를 찾습니다. 이러한 열은 의제 모듈의 적절한 필드에 매핑됩니다. 의제 파일에 있을 수 있는 다른 열은 의제 모듈에서 무시됩니다. 의제 파일을 업로드하려면 다음 열이 필요합니다.

  • name -세션 이름
  • start_date - 세션 시작 날짜(MM/DD/YYYY 형식)
  • start_time -세션 시작 시간
  • timezone - start_time의 시간대 ( IANA 시간대의 영역 ID 사용 )
  • duration - 세션 시간(분)
  • track - 트랙을 식별하는 값 (예: 기술, 영업, 마케팅). ALL의 트랙 이름은 세션이모든 트랙 (즉, 점심, 휴식 시간) 에 걸쳐있음을 나타냅니다.

다음 열은 선택 사항이지만 의제 파일에 있는 경우 의제 모듈로도 가져옵니다.

  • description -세션 설명
  • speaker1_first -발표자 이름
  • speaker1_last -화자 성
  • speaker1_email - 발표자 이메일 주소(발표자를 고유하게 식별하는 데 사용)
  • speaker2_first -발표자 2 이름
  • speaker2_last -화자 2 성
  • speaker2_email - 화자 2 이메일 주소(화자를 고유하게 식별하는 데 사용)
  • speaker3_first - 화자 3의 이름
  • speaker3_last - 스피커 3 성
  • speaker3_email - 화자 3 이메일 주소(화자를 고유하게 식별하는 데 사용)
  • session_type - 세션 유형(세션, 기조연설, 회의, 기타)
  • video _id - 비디오 ID(세션, 기조 연설 및 기타 세션 유형용)
  • url - 회의 URL(회의 세션 유형용)
  • related _label - 의제 모듈에 표시될 관련 링크의 레이블
  • related_link - 세션 유인물, 슬라이드 등의 URL 링크
참고: 의제 업로드는 최대 3개의 스피커를 지원하며, 아젠다 모듈을 사용하여 세션에 추가 스피커를 추가할 수 있습니다.

아젠다를 아젠다 모듈로 가져올 준비가 되면.csv 파일로 내보냅니다. 아젠다 모듈은.csv 파일만 가져올 수 있습니다. 아젠다 파일을 Agenda 모듈에 업로드하는 방법에 대한 자세한 내용은 이벤트 아젠다업로드 및 사용자 지정을 참조하십시오 .
