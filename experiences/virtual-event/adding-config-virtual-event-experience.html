환경에 구성 파일 추가

{{ page.description }}

Virtual Event Experience를 사용하려면 사이트에 구성 파일이 포함되어 있어야합니다. 구성 파일에는 사이트 설정, 레이아웃 옵션, 이벤트 일정, 스피커 및 표시 될 기타 콘텐츠가 포함됩니다. 새 가상 이벤트 경험을 만든 후 구성 파일을 추가하는 것이 가장 먼저 수행되어야합니다.

Virtual Event Experience에 구성 파일을 추가하려면 다음 단계를 따르십시오.

  1. 다음 링크로 이동합니다.
    https://virtual-events-experience.s3.amazonaws.com/v1/config.txt
  2. 콘텐츠를 클립 보드에 복사합니다.
  3. 갤러리 모듈을 엽니 다.
  4. 경험을 편집하십시오.
  5. 딸깍 하는 소리외모와 행동> 머리글과 바닥 글왼쪽 탐색에서
  6. 구성 파일을사용자 정의 헤더 HTML필드에 붙여넣습니다.
  7. 저장을 클릭합니다 .

당신이경험을 다시 게시변경 사항이 나타납니다.

정책 키및에셋베이스URL값 업데이트

카운트다운 타이머가 simulive 비디오에 표시되려면 구성 파일의 PolicyKey값을 검색 가능 정책 키로 업데이트해야 합니다. 제공된 값은 귀하의 계정에서 작동하지 않습니다. 이렇게하지 않으면 동영상에 카운트 다운이 표시되지 않습니다. 검색 가능 정책 키를 얻는 단계는 개발자 문서코드 샘플을 참조하십시오. 정책 키를 생성합니다 . 새 정책 키를 만든 후 구성 파일의policyKey값을 업데이트합니다.

AssetBaseURL값 경로는 템플릿에서 사용하는 파일을 로드하는 데 사용됩니다. 높은 수준의 사용자 지정이 수행되지 않는 한 대부분의 경우이 작업은 그대로 둘 수 있습니다. 거기있는 경로 값을 제거해야합니다.

구성 파일에서정책 키및 AssetBaseURL값이 업데이트된 후 환경을 다시 게시합니다.
