정적 콘텐츠 작업

{{ page.description }}

정적 콘텐츠 개체는 사용자 지정 HTML을 템플릿에 삽입하는 데 사용됩니다. 이 샘플 히어로 배너는 텍스트와 이미지의 조합을 사용하여 구성되었으며 페이지 상단에 삽입되었습니다.

정적 콘텐츠는 모든 HTML 일 수 있으며 템플릿의 어느 곳에 나 삽입 할 수 있습니다. 정적 컨텐트 객체는 구성 파일의static_content객체에 추가됩니다. 페이지에 정적 콘텐츠를 배치하는 방법에 대한 자세한 내용은 홈 페이지에서 콘텐츠위치 지정을 참조하십시오 .
