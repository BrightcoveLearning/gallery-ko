이벤트 아젠다 구성

이 항목에서는 가상 이벤트 경험의 이벤트 일정을 사용자 지정하는 방법에 대해 알아 봅니다.

이벤트 일정은 일반적으로 추천 컬렉션에 표시되는 예약 된 라이브 이벤트를 표시하는 데 사용됩니다. 각 세션과 함께 표시되는 버튼은 세션 상태에 따라 다릅니다.

  • 이벤트 전 -버튼이 표시되지 않고 카운트 다운 타이머가 표시됩니다.
  • 동안 -이벤트가 진행 중일 때 지금 라이브
  • 이벤트 후 -버튼은 과거 이벤트에 대한 VIEW SESSION을 표시합니다.

의제의 일수에 따라보기드롭다운이 나타나서 시청자가 확인할 수 있는 날짜의 일정을 선택할 수 있습니다.

의제 정보는 구성 파일에서schedule객체를 편집하여 사용자 정의됩니다.

  • name -세션 이름
  • description -세션 설명-현재 페이지에 표시되지 않지만 사용자 정의 JavaScript를 사용하여 표시 할 수 있습니다.
  • starts_at -세션 시작 날짜 및 시간
  • ends_at -세션 종료 날짜 및 시간
  • live_id -미사용
  • vod_id -비디오 ID
  • speakers -세션 발표자 정보
  • image_url -세션 이미지. 기본 이미지 크기는 320px x 180px입니다. 이미지는 이 크기로 잘립니다.

구성 파일의schedule객체에는 일정 위에title나타나는 에 대한 매개변수와 스케줄이 표시되는지 여부를 설정하는 매개변수도 있습니다.한expanded모드에 있습니다.
