가상 이벤트 경험의 바닥글 사용자 지정

이 항목에서는 Virtual Event Experience 페이지의 바닥 글을 사용자 지정하는 방법에 대해 알아 봅니다.

바닥 글은 모든 페이지의 맨 아래에 나타납니다. 바닥 글은 구성 파일의 값으로 구성됩니다. 구성 파일의 각 바닥 글 항목에는 다음 항목이 연결됩니다.

  • 링크 URL
  • 라벨
  • 그룹

그룹은 바닥 글에서 레이블 / 링크가 표시되는 위치를 제어하는 데 사용됩니다. 사용 가능한 그룹은 네 가지가 있습니다.

  • 그룹 0 -HTML 콜 아웃에 사용
  • 그룹 1 -법적 텍스트 및 저작권 정보에 사용
  • 그룹 2 및 3 -링크 목록에 사용 (최대 5 개의 링크 사용 권장)

바닥글에 나타나는 소셜 아이콘은 구성 파일에서social객체를 편집하여 편집/제거할 수 있습니다. 자세한 내용은 가상 이벤트 경험에 대한 소셜 아이콘구성을 참조하십시오 .

바닥 글에 링크 추가

바닥글 링크는 구성 파일에서footer_links객체를 편집하여 구성됩니다.

  • link_url -도착 URL
  • label -바닥 글에 표시 할 라벨
  • group -링크가 표시되는 그룹
