가상 이벤트 경험의 헤더 사용자 정의

이 주제에서는 가상 이벤트 경험 페이지의 헤더를 사용자 정의하는 방법을 학습합니다.

헤더는 모든 페이지의 상단에 나타납니다. 헤더는 구성 파일의 값으로 구성됩니다. 헤더는 다음 구성 요소로 구성됩니다.

  • 헤더 로고
  • 연결
  • 사회 아이콘
  • 검색

헤더 로고

헤더 이미지는 구성 파일의images객체에 의해 지정됩니다.

  • header_logo -헤더에 표시 할 이미지 URL

.Png 및.jpg 파일이 지원됩니다. 헤더 로고는 너비가 178px로 비례하여 축소됩니다.

연결

기본적으로홈 , 라이브 스트림및온디맨드 세션링크는 템플릿에 하드코딩되어 있습니다.

  • 집 -홈, 페이지 상단으로 이동
  • 라이브 스트림 -추천 컬렉션으로 이동 (최고 비디오 플레이어)
  • 온 디맨드 세션 -트랙 섹션으로 이동

구성 파일에서header_links객체를 편집하여 추가 링크를 추가할 수 있습니다.

  • link_url -도착 URL
  • label -헤더에 표시 할 라벨

사회 아이콘

기본적으로 헤더에는 소셜 아이콘이 표시됩니다.

  • 페이스 북
  • 인스 타 그램
  • 트위터
  • LinkedIn
  • 유튜브
머리글에 나타나는 소셜 아이콘은 구성 파일에서social객체를 편집하여 편집/제거할 수 있습니다. 자세한 내용은 가상 이벤트 경험에 대한 소셜 아이콘구성을 참조하십시오 .

검색

검색을 활성화하고 검색 필드를 표시하려면 경험에 대한 검색을활성화합니다. 기본적으로 헤더의 검색 필드에는 텍스트 입력을위한 한 줄이 표시됩니다.

예제의 검색 상자는 검색 필드 주위에 상자를 표시하도록 스타일이 지정되었습니다.

헤더 별 스타일링

모바일 장치의 링크

기본적으로 헤더에 추가 된 링크는 모바일 장치를 사용할 때 표시되지 않습니다. 다음 JavaScript는이 문제를 해결합니다.


for (let i = 0, l = ciabConfig.header_links.length; i < l; i++) {
  let link = ciabConfig.header_links[i];
  let li = document.createElement('li');
  let ul = document.getElementById('main-nav-cat');
  li.className = 'main-nav-category main-nav-category-home';
  li.innerHTML = ('' + link.label + ''); ul.appendChild(li);
}
