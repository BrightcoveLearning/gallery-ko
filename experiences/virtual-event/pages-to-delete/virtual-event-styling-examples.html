가상 이벤트 스타일링 예

이 항목에서는 가상 이벤트 템플릿을 사용자 지정하는 데 사용할 수있는 몇 가지 샘플 코드를 제공합니다.

사회 공유 아이콘

기본적으로 경험에 대해비디오 공유설정이 활성화되어 있으면 비디오 세부 정보 페이지의 비디오 옆에 표시되지 않습니다. 다음 CSS는 비디오 세부 사항 페이지에 소셜 아이콘을 표시합니다.


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}
