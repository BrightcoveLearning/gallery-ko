이벤트 스피커 사용자 지정

이 항목에서는 Virtual Event Experience 홈페이지에 표시되는 이벤트 스피커를 사용자 지정하는 방법에 대해 알아 봅니다.

가상 이벤트 홈페이지에 발표자 이미지와 짧은 약력을 추가 할 수 있습니다.

스피커 정보는 구성 파일에서speakers객체를 편집하여 사용자 정의됩니다.

  • name -발표자 이름
  • description -이미지 아래에 표시되는 스피커 정보. 이는 15 단어의 약 3 줄로 제한됩니다.
  • image_url -스피커 이미지. 최대 너비는 200px를 권장합니다. 이미지 크기가 조정되지 않습니다.
  • link_url - (선택 사항) 이미지의 대상 URL (예: 스피커 LinkedIn 페이지에 링크될 수 있음)

기본적으로 템플릿 CSS는 스피커 이미지를 50% 로border-radius구성하여 이미지를 원형으로 만듭니다.

구성 파일의speakers개체에는 홈페이지에서 발표자 정보가 표시되는 위치를 제어하는 매개 변수도 있습니다. 스피커 섹션 배치에 대한 자세한 내용은 홈 페이지에서 콘텐츠위치 지정을 참조하십시오 .
