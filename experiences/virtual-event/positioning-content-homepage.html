홈 페이지에 콘텐츠 배치

{{ page.description }}

구성 파일의 여러 개체에는 페이지에서 개체가 표시 될 위치를 지정할 수있는 매개 변수가 있습니다. 예를 들어 정적 콘텐츠, 발표자, 트랙 및 스폰서는 모두 페이지에서 재배치 할 수 있습니다. 위치를 지정할 수있는 개체는 구성 파일에서 다음 속성을 갖습니다.

  • insert_target -주어진 객체가 렌더링되는 시작점으로 사용될 CSS 선택기 (간단한 클래스 또는 ID)
  • insert_mode -콘텐츠 삽입 방법
    • after
    • before
    • append
    • prepend

목표 결정

이insert_target매개 변수는 CSS 선택기 (간단한 클래스 또는 ID 형식 사용) 를 기대합니다. 이 선택기와 일치하는 요소는 주어진 객체가 렌더링되는 시작점으로 사용됩니다. 사용할 CSS 선택기 또는 ID를 결정하는 가장 좋은 방법은 가상 이벤트 홈페이지로 이동 한 다음 브라우저 개발자 도구를 사용하여 페이지의 요소를 검사하는 것입니다.

홈페이지의 주요 섹션은<div>태그를 사용하여 구성되어 있음을 알 수 있습니다.

삽입 모드 선택

계층에서 객체를 삽입할 위치를 결정한 후insert_mode속성은 해당 객체가 계층에 삽입되는 방식을 결정합니다.

  • after - 지정된div수준 뒤에 객체를 넣습니다.
  • before - 동일한div레벨에서 지정된 객체 앞에 객체를 넣습니다.
  • append - 마지막div자식으로 지정된 객체 안에 객체를 넣습니다.
  • prepend - 첫 번째div자식으로 지정된 객체 안에 객체를 넣습니다.

아래 다이어그램은 다양한insert_mode값의 효과를 보여줍니다.

개체 위치 지정 예

다음 코드 샘플은 정적 콘텐츠 개체를 사용하여 보조 컬렉션 위에 제목을 삽입하는 방법을 보여줍니다.


{
    "insert_target": ".bc-on-demand-section",
    "insert_mode": "prepend",
    "content": "<h2 class='bc-on-demand-section-title'>ON-DEMAND SESIONS</h2>",
    "require_roles": [],
    "omit_roles": []
}

브라우저 도구를 사용하면 가 의 내부에 첫 번째 요소로h2삽입되었음을 확인할 수bc-on-demand-section div있습니다.
